标准美国黄卡叫牌法

总 则：

强牌细叫，差牌高叫

任何位置都使用5张高花开叫

5-5或6-6等长套优先开叫较高花色

4-4低套一般开叫1D

3-3低套一般开叫1C

无将开叫表示平均牌性但可有5张高花或低花套

1NT=15-17

2NT=20-21

3NT=25-27

强2C开叫

2D以上花色开叫为弱阻击叫

1NT开叫后的应叫：

2C是非逼叫性斯台曼Stayman，意即可在二阶停叫，开叫人必须应叫2D/2H/2S。通常应叫人有8大牌点以上，但也可能应叫人很少大牌点但试图对任何回应停叫，例如应叫人4-4-4-1牌型草花单张。

如果开叫人是4-4高花时应叫2H,如果应叫人在3阶再叫任一低花，显示有满贯兴趣并至少为5张套。如果应叫人再叫另一高花，表示此高花为5张套（暗示另一高花4张〕，但仅在3阶再叫时才是进局逼叫。

杰科贝Jacoby转移叫显示有5张以上高花，2D转换到红桃，2H转换到黑桃。开叫人必须接受转换，仅当开叫人有17点并有四张支持时才在3阶跳应叫。（应用杰科贝转移叫时另一高花不能有4张，否则应用斯台曼Stayman〕。例如：

1NT -- 2D
2H=正常接受转换
3H=17大牌点和4张红桃支持

如果接受转换叫后，应叫人叫了新套，表示是自然叫并且逼叫进局（1NT-2D-2H-2S不逼叫进局），可能的叫牌进程如下：

1NT -- 2H -- 2S ……

PASS =满足于打2S。
2NT,3S =邀叫。对2NT开叫人低限时PASS或叫回3S,高限时叫3NT/4S。
3C/3D/3H = 自然套和进局逼叫。另一高花5张套。如果是低花，通常有满贯意图。
3NT = 要求在3NT/4S 中作选择。
4S = 进局止叫，六张以上套。

对1NT的2S应叫要求回叫3C,然后应叫人如果是C套就PASS,是D套就叫3D.例如：

1NT -- 2S ……

3C -- Pass = C套
3C -- 3D = D 套

其它对1NT的应叫：

1NT -- 

3C/3D = 6张以上套，邀叫3NT.
3H/3S = 6张以上套，满贯兴趣（否则应使用转换叫）
4C = 戈伯Gerber Ace问叫。对开叫和应叫1NT/2NT后的4C都是戈伯Ace问叫。答叫A的数量。答法同黑木4NT问叫…。（接着叫5C是问K）

1NT -- 4C

4D = 0/4 A
4H = 1 A
4S = 2 A
4NT= 3 A

1NT - 4C - 4X - 5C -

5D = 0/4 K
5H = 1 K
5S = 2 K
5NT= 3 K

如果在使用Gerber 4C 问叫后选择了除5C外的其它任何再叫，都表示要打此定约。

开叫1NT后应叫人直接加叫4NT是邀叫6NT，这时4NT仅是满贯邀叫因为此时可用GERBER 4C来作ACE问叫。

如果对手加倍，所有约定应叫仍然有效。例如：

1NT -- （Double） --

2C = Stayman
2D = 转换叫

如果对手对你方的1NT开叫进行花色或无将争叫，就不再使用斯台曼和转换叫。除扣叫外都是自然叫，此时扣叫作为斯台曼使用并且具备进局实力。

如果杰科贝转换叫被加倍，开叫人可以PASS，再加倍表示被加倍花色是强套，完成转换表示3张以上将牌，高限强配合时在3阶跳应叫。如果开叫人PASS或再加倍，应叫人可选择自然叫、再加倍、PASS.或叫2阶高花止叫。

如果转换叫被盖叫，开叫人仅在有强支持时在3阶出套，加倍是惩罚性的。如果开叫人PASS,应叫人可作自然叫品、加倍以显示实力，或PASS.

如果斯台曼被加倍，开叫人可作自然再叫，4张C时PASS,4张以上强C套时再加倍，如果斯台曼被盖叫，开叫人可在2阶出4张高花套，在对手的套中持有4张以上时惩罚性加倍，或PASS.任何情况下应叫人都可继续自然再叫或作惩罚性加倍。

对2NT/3NT开叫的应叫：

使用斯台曼和杰科贝高花转换叫

2NT ……

3C = 斯台曼
3D/3H = 杰科贝转换叫
4C = 戈伯A问叫
4NT = 6NT邀叫
3NT…4C = 斯台曼
4D/4H =杰科贝转换叫

在1H/1S开叫后的应叫及后续叫牌：

1H/1S开叫表示5张以上套。应叫有：

1H…

1S = 至少6大牌点、4张黑桃以上，暗示无红桃配合。
1NT = 6-10大牌点，否认有四张黑桃或三张红桃，不比叫
2C/2D = 11大牌点以上，四张以上套
2H = 三张以上红桃支持，6-10点（加牌型点）
2NT = 逼叫进局，为杰科贝2NT应叫，13点（加牌型点）以上，要求开叫人显示短套以便
    应叫人进行满贯估计。
2S/3C/3D = 强跳叫，满贯邀请
3H = 限制性加叫〔10-12点，三张以上红桃〕，建议在10点、弱三张红桃、平均牌型时
    应叫1NT.
3NT = 15-17大牌点，平均牌型，红桃二张。〔由于此叫品太高，不利于本方进一步试探
    满贯，因此仅在只想3NT进局时适用，例如在红桃中无大牌且手中JQ多于AK.〕
4H = 通常五张以上红桃，另有一门花色单缺，少于10大牌点。

之后开叫人的再叫是自然和标准的。

低限牌13-16点的再叫：

在最便宜的级别叫无将；

对应叫人的花色加叫一级

再叫一新花色〔注意不可逆叫〕

在最低级别再叫开叫人的花色

中等牌力的再叫：

跳加叫或跳再叫开叫人的花色

新套逆叫

非逆叫的新花色〔13-18点〕

高限牌力的再叫

跳叫无将

双跳加叫应叫人花色或双跳再叫开叫花色

跳叫新花色

如果应叫人对1H/1S开叫应叫2NT,即杰科贝2NT,要求开叫人显示单缺，如果开叫人无短套，就表示手里的实力如下；

1H – 2NT

3C/3D/3S = 盖花色单缺，其它叫品否定有短套
4H = 低限牌
3NT = 中等牌力15-17点
3H = 高限牌力18点以上

应叫人接着可进局止叫，在有满贯兴趣时叫黑木4NT或进行扣叫

应叫人随后的叫牌：

如果应叫人在一阶叫过一花色，其后他有停在部分定约、邀局、进局和逼叫进局等各种选择，见如下示例：

停于部分定约的有效叫品有：PASS/1NT/在2阶再叫先前的花色。

1H –- 1S

2C—Pass/2H/2S = 6-10点，建议停于部分定约，邀局的叫品为2NT或3阶加叫
2D—2NT/3D/3H/3S = 11-12点，邀局。

新花色应叫〔除开叫人在1NT应叫后的再叫〕是逼叫一轮〔不逼叫进局〕，如果是第四花色则不是自然叫品〔1C-1D-1H-1S除外〕如下例：

1H –- 1S -- 2C –- 2D = 逼叫一轮，非自然叫品

但是

1H –- 1S -- 1NT –- 2C/2D = 非逼叫，应叫人如想逼叫进局必须跳叫3C/3D.

在第四花色逼叫后，开叫人应该：

1)  在有3张支持时叫应叫人的第一花色，非低限时可跳加叫。

2)  再叫第二花色表示弱牌〔跳再叫表示有额外价值，极好的牌型〕

3)  叫NT表示第四花色有止张〔跳叫进局有额外牌力〕

4)  加叫第四花色表示有四张

5)  再叫第一花色表示有额外的实力和长度

开叫人1NT再叫后的第二轮逼叫，逆叫或跳叫新花为进局逼叫

1C -- 1H -- 1NT—2S/3D = 进局逼叫，以下为进局止叫：3NT/4H/4S/5C/5D

如果应叫人第一次是在2阶应叫，除在3阶跳加叫开叫人的第一花色是进局逼叫外〔应叫人在10-12点和3张以上支持时应在开叫后直接作限制性加叫〕，其余规则与应叫人第一次在一阶应叫时一样：

1S –- 2C

2D = 逼叫一轮，应叫人低限时应在2S/2NT/3C/3D中选叫，不能PASS,因为开叫人可能
    持有多达18大牌点〔实力仅弱于跳再叫新花〕。
2H—2NT/3C/3H = 邀局，11-12点
2S = 低限不逼叫，应叫人持11-12大牌点和双张黑桃时可邀叫2NT
3D =进局逼叫，可能是人为叫品
3S =进局逼叫

注：应叫人在2阶应叫新花表示他保证会再叫，除非开叫人的叫品已进局

对1C/1D开叫的应叫：

1D一般表示4张以上，因为3张低花时应优先开叫1C,仅在4432牌型时〔D3,C2〕例外

应叫及后续叫牌都与前面所述的思想相同，应叫人需要更多的将牌来加叫〔四张以上加叫1D,五张以上才能加叫1C,仅在竞争叫牌时可少一张〕，2NT/3NT应叫是标准的。

1C –-

2NT = 13-15大牌点，进局逼叫
3NT = 16-17大牌点。

低花的加叫无逼叫。

对2C开叫的应叫及后续叫牌：

2C开叫表示至少22点以上，或9赢墩以上

应叫：2D=人为叫品示弱，也可能是等待叫，好牌但不适于做庄

      2H/2S/3C/3D=自然叫，进局逼叫，至少5张、8大牌点以上。

      2NT = 8大牌点，平均牌型。

如果开叫人在2D应叫后再叫2NT〔表示22-24点〕，则应叫与2NT开叫时相同：

2C –- 2D ……

2NT – 3C = 斯台曼
3D/3H = 杰科贝转换叫
4C = 戈伯A问叫
4NT = 邀叫6NT

如果开叫人在2D应叫后再叫一个花色，则逼叫到此花色的3阶〔高花〕或4阶〔低花〕，开叫人此套一般5张以上，当开叫人持4441牌型时选最强套叫牌。

2C—2D—2H  逼叫到3H.

弱2开叫2D/2H/2S的后续叫牌：

弱2叫品表示5-11大牌点，6张较好套。特殊情况下可能是好的5张套，也可能是差的7张套〔不足以在3阶开叫〕，在另一高花4张以上时一般不应使用弱2叫品。应叫：

2NT应叫是逼叫，显示进局兴趣〔当对手用加倍或争叫进行干扰时仍保持原意〕，开叫人再叫原套表示低限5-8大牌点，高限时叫再叫另一花色表示在此套中有A或K，如果开叫人高限且在一低花有4或5张套，他应在4阶叫此低花。任何对开叫花色的加叫可能是阻击性的，并准备打此定约，3NT应叫也是准备打此定约

新花色应叫表示5张以上套、逼叫一轮，开叫人在有3张甚至好的2张支持时加叫。

当对应叫人花色无配合时，开叫人应叫规则如下：

低限时再叫最低级别原套。
高限时出新花色或叫无将

对3阶以上阻击性开叫的应叫：

任何新花应叫〔除成局叫外〕都是逼叫

满贯叫牌：

黑木4NT问A,应叫人按级答叫A的数目，然后5NT问K,此时应保证联手握有全部4个A

4NT —

5C = 0/4A
5D = 1A
5H = 2A
5S = 3A

4NT — 5X — 5NT

6C = 0/4K
6D = 1K
6H = 2K
6S = 3K

跳叫5NT为大满贯逼叫，要求同伴在持有将牌的AKQ中的二张时叫大满贯，否则叫小满贯。

5NT —

6阶将牌花色=未持有将牌的AKQ中的二张以上。
7阶将牌花色=持有将牌的AKQ中的二张以上。

如果使用黑木者想将定约停在5NT上〔发现A不够〕，可以叫一个未叫过的花色，应叫人必须叫5NT。
例如：1H-2D-3D-4NT-5C-5S.

这里是一些在同意将牌后使用扣叫的规则：

1.  （己方）未叫花色先叫，A优先于缺门。

2.  计划好扣叫顺序，最低的扣叫并不一定是最好的，它可能导致你的下一轮扣叫比你掉换扣叫顺序时的扣叫更高，例如：4C-4X-4S比3S-4X-5C要低一级。

3.  先扣叫A,再扣叫K,

4.  成局以下时，仅在有满贯希望时才扣叫，

5.  在超过将牌花色的5阶进行扣叫是试探大满贯。

6.  如果同伴之一已经判断出最佳定约，他应尽可能及早叫出以避免不必要的失误。

7.  如果扣叫被加倍，再加倍表示此套有第二轮控制，PASS表示仍有兴趣。

8.  避免在同伴已叫过的花色上扣叫单缺，除非能保证同伴不会误会是有支持。

防守叫牌：

超叫显示8-16大牌点〔加倍后再出套表示更强的牌力〕，应叫中唯一逼叫是扣叫敌方花色，要求超叫人答复超叫的质量：

（1D）-- 1S --（Pass）--2D --（Pass） --

2S=低限超叫
其它=11大牌点以上

对1阶超叫的应叫如下：

加叫：3张以上支持，6-11点
加叫2级：4张以上支持，10-12点
进局：4张以上支持,弱的牌型牌
1NT：9-12点；  2NT：平叫为11-13大牌点，跳叫为13-15大牌点 3NT：15-16大牌点
新花色：不逼叫，通常表示对超叫人的花色无配合。
扣叫：进局逼叫

注：在加叫超叫花色后的竞争性叫牌中，不要再次加叫该花色

1NT超叫表示15-18大牌点的平均牌型〔对敌方开叫花色有止张〕，除2C斯台曼外其它叫品均非人工叫品。

跳争叫2NT是不寻常NT,显示较低的两套花色至少5-5以上，它也可用于盖叫强2C开叫，〔但不用于盖叫弱2阶开叫--此时的2NT表示16-19大牌点〕，应叫为：选择叫，跳选择叫（阻击性），扣叫为进局或满贯试探，新花色（不逼叫），3NT,黑木4NT.

跳盖叫为阻击性，与此级别的开叫价值相同。

（1D）--

2S=与开叫2S同等价值
3C=类似于开叫3C

在平衡位置、非逼叫性加叫（如1H-P-2H-）和对1NT开叫、阻击性开叫后不使用跳盖叫

当敌方只叫出一套时扣叫表示麦克尔Michaels扣叫，显示5-5以上两套，当敌方开叫低花时，扣叫显示两高花，当敌方开叫高花时，扣叫表示另一高花及一未知的低花。

（1D）--2D=至少5-5以上高花，8大牌点以上。

（1S）--2S=至少5-5以上红桃及一未知低花，10大牌点以上

应叫人对高花的扣叫可叫2NT来询问其低花。

（1H）-- 2H--（Pass）-- 2NT —

（Pass）-- 3C = 梅花套
（Pass）-- 3D = 方块套

在竞争叫牌时，如果应叫人无法叫2NT,他可叫4C（不逼叫）或4NT（逼叫）来找到低花套。

3NT永远是进局止叫

平衡位置重开叫与直接位置开叫相似，但可能其点力更低。在第四家位置重开叫1NT显示10-15大牌点，注意在本方有局时不使用此叫品。当有好套及开叫实力时应使用跳争叫，加倍后叫2NT表示16-17大牌点，直接叫2NT表示18-20大牌点。扣叫表示技术性加倍的实力并在扣叫花色单缺。

对敌方开叫在4D以下的未成局叫品的加倍是技术性加倍，对成局叫品的加倍都是惩罚性的，对技术性加倍的未成局的跳应叫是邀请性的，如果想逼叫，应叫人应扣叫敌方的开叫花色。

对付阻击性开叫，盖叫花色或无将是自然叫品，扣叫是麦克尔扣叫。

竞争性叫牌:

这里的可能的叫牌序列是无穷尽的，因此应制定简单的规则以防止同伴间误会

叫品与无干扰时一样，并不保证额外的点力，但有时在竞争情况下需要选择一个非常规的叫品，

如：1D --（Pass）-- 1S --（2C）

2S；持S.J43  H.A875  D.AQJ4  C.J3 （此牌在RHO《右手敌方》未争叫2C时应叫1NT）

扣叫RHO的套表示本方有进局实力但还未清楚什么定约最佳。通常显示为进局逼叫性加叫牌力。

如：1S --（2C）-- 3C = 进局逼叫，一般S有支持。

在一阶水平的应叫与没有争叫时一样，应叫人优先支持开叫人花色，在2阶的新花应叫一般显示至少5张套和10大牌点以上，如果是在3阶，就是进局逼叫。

负加倍使用到2S为止，保证在另一未叫高花4张以上。注意：在1C-1D后是例外，此时保证两个未叫高花4-4以上。在2阶或更高水平上叫高花表示11点以上和5张以上套。

1C --（1D）-- Double = 4-4 以上高花

1C --（1H）-- Double = 正好4张黑桃（叫1S保证5张以上）

1D --（1S）-- Double = 4张红桃和6大牌点以上或5张红桃和5-10大牌点。

对负加倍的应叫：

成局线下的平叫：不逼叫，少于16大牌点
跳叫：不逼叫，16-18大牌点
扣叫敌方花色：19大牌点以上，逼叫进局
Pass（极少）：惩罚性

负加倍者成局以下的再叫除扣叫敌方花色外均不逼叫

由于采用负加倍，同伴不能在直接位置进行惩罚性加倍，因此，当敌方的争叫后接着两家PASS时，开叫人在敌方花色少于2张时应尝试重开叫性加倍以便同伴在敌方花色持有强牌时能惩罚性PASS.

如果RHO（右手敌家）作技术性加倍：

1D --（Double）--

1H/1S = 逼叫，点力无限制
2C = 不逼叫（6-10大牌点，通常为6张套）
2NT = 限制性加叫（至少10大牌点）或更好
再加倍 = 10大牌点以上，不适于叫1H/1S/2NT
3D = 阻击性，好的将牌支持但少于10大牌点。


其它花色的跳应叫是准备打此定约，如下例：

1D --（Double）-- 2H/2S/3C = 六张以上套，实力同阻击叫。

在RHO（右手敌家）PASS后对同伴技术性加倍的应叫：

（1D）-- Double --（Pass）--

平叫花色 = 0-9点
1NT = 6-10大牌点，平均牌型，在敌方花色有止张
2NT = 10-12大牌点，在敌方花色有止张,无4张高花
跳叫 = 10-12点，不逼叫
扣叫 = 13点以上或10-12点的4-4以上高花
3NT = 13-16大牌点，
双跳叫 = 少于10点，6张以上，阻击性。
Pass（罕有）= 在敌方花色有5张以上（价值3墩）

在弱应叫（即平叫花色）后技术性加倍者的再叫：

Pass = 少于15大牌点
加叫 = 16-18大牌点，4张支持；在非竞争叫牌下主动加叫一级到3阶为18-20大牌点
跳加叫 = 18-20大牌点，4张支持
新花 = 18-20大牌点，5张以上套（如果在另一未叫高花有4张可以稍少点力）
跳叫新花 = 6张以上坚强套，强牌但不逼叫。
1NT = 18-20大牌点
2NT = 平叫时是19-21大牌点，跳叫则是21-22大牌点
3NT = 有9赢墩
扣叫 = 21大牌点以上，有满贯兴趣

对1NT开叫的直接加倍是惩罚性的，持有与1NT开叫人同样牌力，例如：对强无将开叫需17大牌点，对弱无将开叫可少1-2点。在第三家PASS后，应叫人持平均牌型应该PASS,持弱的牌型牌就在2阶出套，跳应叫表示8大牌点以上的牌型牌（对弱无将为6大牌点以上），邀请进局。

以下是判断一个加倍是否是惩罚性的一些原则，应注意在同伴正常争叫后对敌方的加倍是惩罚性的

1. 当加倍发生在叫牌后期时。例如：1S-P-2S-P-4S-X

2. 但加倍者的同伴已精确地描述出他的牌型和牌力后

3. 当同伴间曾对敌方进行过再加倍时

4. 当同伴间曾作过惩罚性加倍或对技术性加倍作过惩罚性PASS时。

5. 当加倍一人为叫牌时

6. 当对阻击叫的高花应叫加倍时

一个再加倍可能是以下三种情况之一：

1. 如果敌方加倍你方在4阶以上定约时就是准备打此定约：

4S--（Double）--Redouble = 惩罚性

1NT--（Pass）--2D--（Double）--Redouble = 惩罚性,好的D套

2. 如果敌方是技术性加倍则表示持强牌

1S--（Double）-- RedoubleE = 10大牌点以上

3. SOS，如果你方在3阶以下的叫牌被惩罚性加倍时要求同伴另叫一套花色

1D--（Pass）--Pass--（Double）--Pass-（Pass）--Redouble = SOS,应叫人在未叫花色中至少支持其中两套花色。

除非有特别说明，否则在敌方争叫或加倍后将取消在非竞争性叫牌时的约定叫。

1H —（1S）-- 2NT = 自然叫，12-14大牌点

2C --（Double）-- 2D = 自然叫品

如果敌方使用约定叫（例如麦克尔扣叫或不正常无将），你可以加倍表示10大牌点以上，或者扣叫敌方花色之一来逼叫进局。
