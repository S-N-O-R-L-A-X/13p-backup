竞争性加倍（Competitive Double）

　

对对方低阶水平的花色定约作加倍，目的并非惩罚，而是为了表示持一定的牌力，要求同伴协作，选择最佳定约和争叫花色。方法如下：

（1）对对方已配合的争叫花色作加倍，必须符合下列条件：

a．同伴开叫，应叫者作过花色应叫；
b．同伴双方均没有叫过无将;
c．同伴双方未曾表示过将牌配合； 
d．对方已配合的争叫花色不超过2S水平。

例如西持：S95 HA108 DKJ973 CQ85。叫牌过程

北      东      南      西
        1C      pass    1D
1S      pass    pass    X

西的加倍即为竞争性加倍，表示持牌力10点左右，要求同伴选择打2 C或2 D定约。

又如北持：S A8 HJ105 D763 CAKQ102。叫牌过程：

北      东      南      西
1C      1D      1H      2D
X

北的加倍为竞争性加倍，表示持牌力至少为14点，要求同伴在C或H中选择定约。

再如西持：S95 HQ8 DAK9754 CJ103。叫牌过程：

北      东      南      西
1C      1H      1S      2D
2S      pass    pass    X


西的加倍为竞争性加倍，要求同伴在3D或3H中选择。

（2）对对方未获其同伴加叫的争叫花色加倍（不超过2S水平），一般属于惩罚性，但是，若先前的叫牌过程已证明这个加倍作为惩罚手段是不可能和不合理的时候，这个加倍即成为竟争性加倍。

例如。西持:SAJ8 H853 D K103 CQ954。叫牌过程：

北      东      南      西
        1D      pass    1NT
2H      pass    pass    X

西的加倍为竞争性加倍，因为西已应叫过1NT，排除持四张红心，所以加倍作为惩罚是不合理的，仅表示持高限牌力，要求同伴继续叫牌。

（3）在对方叫牌水平已达到三阶时，一般不用竞争性加倍，或仅限制在极小的范围内运用，且必须符合下列条件：

a．同伴已叫过牌；
b．加倍者已作出过否定性加倍、排除性加倍或应叫性加倍；
c．再一次加倍的花色，仍是对方原来叫过的花色，且不超过3S水平。

例如，西持：SKQ95 H54 DA76 CQ1072。叫牌过程：

北      东      南      西
        1D      1H      X
3H      pass    pass    X

西第一次加倍是否定性加倍，第二次加倍即为竞争性加倍，要求同伴继续参与叫牌。
