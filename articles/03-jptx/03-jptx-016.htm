技术性加倍

　

技术性加倍是防守叫牌的重要武器，是对开叫方进行反击的一把利剑，但却是一把双刃剑，既能伤人，也会伤己。所以用好技术性加倍是牌手的一个基本技巧。

1、低实力技术性加倍：

点力11-15点，未叫花色至少三张支持。因为贴点制竞争未进局定约很重要，所以应根据局况及牌型适当加减。因在贴点制中，阻击叫更多依靠牌型，我认为在对方未成局的阻击叫都是技术性，至少偏重技术性。因这是最常用的叫品，在此不再累述。

2、高实力技术性加倍：

16点以上，对未叫花色都有支持的好牌或自己有一花色长套。这时对花色长套要求较高，应至少AKQXX以上。
比直接争叫1NT还强或一手接近直接进局的牌也先做技术性加倍。

3、对1NT开叫的技术性加倍:

我认为对弱NT的加倍，表示自己应开叫强无将开叫的牌。对强无将开叫，应是一手至少有6个以上快速赢墩的牌。

4、对方两人都叫过牌的技术性加倍。

   N      E      S      W
   1H     /      2H     X
或 1H     /      2D     X

这样的加倍在未叫过的花色要求比直接加倍高一些，也要根据局况判断。

5、延迟的技术性加倍：

N      E      S      W
1D     /      1S     /
1NT    X

未叫过两门至少4-4的延迟的技术性加倍

N      E      S      W
1H     /      2C     /
2H     X

这时的加倍也是延迟的技术性加倍，因对方点力比自己强，这应在无局的情况下，其他两门至少5-4，并且点力集中在另两门。

6、平衡位置的技术性加倍：

凡是能够在直接位置上进行技术性加倍的牌，均可在平衡位置做技术性加倍。但是因为在加倍之前已有两家不叫，平衡位置牌手是为了使叫牌能延续，在大牌点略少，或牌型不完全合适时，都可做技术性加倍。

7、同伴开叫后，对方争叫的否定性加倍

N      E      S      W
1H     2C     X

你这时的加倍表示如对方不争叫，你叫1S的低限牌，6-10点，4张S。

8、同伴争叫过的技术性加倍:

N      E      S      W
1S     2D     2S     X

西的加倍表示希望同伴叫出3C或3H，但不反对同伴再叫6张D，是2524或1525牌型。

9、对特殊叫品的加倍：

对逼叫性1C的加倍，依然是技术性加倍，叫2C是一般争叫。
对逼叫性2C、2D的加倍，表示开叫牌力，C 或D 是实套

10、对技术性加倍的应叫:

对技术性加倍的应叫只有在认为打宕对方得分更多时才能PASS，在你一手白牌时，千万不能PASS。各种叫品，书上已写了很多，不在累述。
