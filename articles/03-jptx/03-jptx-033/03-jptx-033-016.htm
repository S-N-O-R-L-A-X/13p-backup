§9、敌方插叫后之处理方法

　　以上我们所讨论的各种情况，都是假定对方是完全平静无声，让你俩能自由的交换消息，写意的叫成最佳合约，但在牌桌上的真实情形不可能会让你这么舒服，对着你的强牌，对方一定的乐意用些捣乱性的叫品来困扰你，甚至用欺诈的手段来愚弄你。这些当然会增加你很多困难：叫牌空间减少使你无法精确的将牌叫得很清楚，孰真孰假的叫品一定也会令你困惑。因此，我们必须设计一套对付对方这些捣乱性叫品的方法，只要你能妥善的运用、处理对方这些叫牌，未曾不会让你在叫牌甚或打牌时多得到一点线索了。
　　虽然对方这些叫牌并不一定是真实的，但我们无法设计出一套可以那么弹性的运用在各种真假不明的情况，只能原则上假定对方这些叫牌是真实的。
　　首先我们假定开叫1C的左手敌方（Left hand 0pponent,L.H.O）加入叫牌；回牌人所答叫之意义则视敌方所加入的叫牌不同而完全不一样。
1、L.H.O赌倍时：
　　这个赌倍有些人仍表示是迫伴赌倍（take-out double）。有此人则用作是表示有梅花牌组的赌倍，我们都同们处理。
　　由于这个赌倍并不损失掉任何空间，因此回牌人之回答可丝毫不受其影响，只是有些细节地方不同。回牌人答叫：
派司：0-3点的任何牌型。
1D：4-7点。
再赌倍：8点以上的平均牌型。
1NT：4-7点，两门低花5-5以上。
其他叫品：与敌方未赌倍时之意义完全相同。
　　如回牌人之答叫并未受到对方赌倍的影响时，开叫人之再叫及其以后的发展仍与敌方未赌倍时完全相同。
回牌人再赌倍表示是8点以上的平均牌型时，开叫人如亦持平均牌型，通常都应派司，而视身价情况准备处罚敌方。
　　回牌人叫1NT表示有5-5的两门低花，4-7点后，开叫人应马上决定要打什么合约，此时无任何叫品是迫叫；甚至跳叫三线低花都不一定是表示好牌，可能仅是阻塞性的叫品，跳到四线则是邀请。
2、L.H.O一阶插叫叫后：
回牌人回答：
派司：0-3点或4点而无A或K。
赌倍：4-7点，无5张高花或无5张好的低花。
一阶花色：4-7点，5张以上的牌组。
1NT：敌叫组有挡；如是平均牌型一定是8-10点，但不一定非平均牌型不可。
简单叫出二阶花色：5-7点，5张以上的牌组。
跳叫一门花色：8点以上，5张以上的牌组；通常敌叫组没挡，否则应优先考虑叫1NT。
示叫敌叫组：8点以上，敌叫组无挡且无5张的牌组。
2NT：平均牌型，11-13点，敌叫组有挡。
3NT：平均牌型，14-15点，敌叫组有良好挡张。
跳示叫敌叫组：8点以上的三门牌组，敌叫组短门。
　　在回牌人示弱（7点以下）后，开叫人要跳叫或是示叫敌叫组才是迫叫；简单叫出一门花色，回牌人是可以派司的。而经过敌方插叫后，取消掉所有的询问叫，所有的叫牌都表示其自然的意义而非有特别的意义。
　　有时，敌方所叫正好是你欲回答的牌组（比方，你有8点以上，5张红心，正欲回答1H；而上家正好盖叫1H），可能敌方诈叫，然亦可能他只是不幸而已；但你此时的任何叫品都已无法告诉同伴你有这样的牌了。因此，在身价有利的情况（敌有而已无）时，你就可以派司过去，等同伴赌倍（通常同伴此门会甚短，多半会作迫伴赌倍）时予以罚放严重的处罚敌方，此谓之陷井性的派司（trap pass）。身价不利时，处罚对方并不见得占便宜，宁愿自己打算了，不采此种叫法。
　　当然，你会碰到这种情形，你正在作陷井性的派司，而同伴所叫却非赌倍而使你希望落空，但你必须马上采取补救方法，使你同伴知道你是这样的牌，这时你就马上叫出这个牌组，同伴绝不会当你是示叫（第一次连以赌倍表示有4个点都无法人的，那够资格示叫什么），一定就是这样的牌。由于回牌人可能是陷井性的派司，开叫人在敌有而已无身价时，在2D以下通常都要作保护叫。
　　回牌人叫1NT是迫叫成局，只保证敌叫组有挡，而不保证一定是平均牌型，可能有5张甚或以上的牌组，这时，开叫人及回牌人以后第一次叫出者都表示是其真正的牌组。以下我们用几个叫法来说明：

例： 西 　　　东

　　在西家开叫1C后，下家超叫1H，东家当然是很标准的跳叫3D，西家此时如不小心随手叫出3S表示有5张黑桃，东家怎么办，只好勉强加成4S（幸好黑桃还有2张，如仅持单张就更尴尬了），但是这是个摇摇欲坠的合约，而却错失了稳稳当当的3NT（在任何首攻下，十墩通常是没问题的）。因此西家在3D后的最佳叫品当然是3NT。这种叫法当然并非一成不变的，完全要根据你对牌张的判断和实际的情况而定。比方这牌的黑桃和红心互换（北家超叫1S而西家持有5张红心），西家在3D之后当然叫3H，东家有红心支持就加成4H，无法支持红心而欲打3NT黑桃却无挡时便可以叫3S，希望能由可能有挡的西家主打3NT。

解 答：
（1）派司：4大点本可作示弱赌倍，但两个J加上一个Q，这种4点实在不好，仍以派司为佳。
（2）赌倍：告诉同伴有4-7点，无5张的牌组。
（3）2H：有5张红心，5个点，正好可简单在二者叫出。
（4）1NT：10个大点，平均牌型，黑桃挡张不错，符合回答1NT的条件。
（5）3D：10个点，5张不错的方块，必须跳叫3D才能表示出来。
（6）2S：有9个点，可以作积极性答叫，但又无5张牌组可叫，黑桃没挡不能叫1NT，故示叫2S。
（7）1NT：有5张方块本可以跳叫3D，但黑桃有挡，仍以叫1NT为佳，下圈再叫方块亦可表示是5张牌组。
（8）1NT：虽11点，6张很好的红心，但黑桃有挡，先叫1NT，下圈再叫红心，更可将牌叫得清楚。
（9）2NT：12个点，黑桃挡张甚好，牌型平均，最好一次叫足2NT。
（10）赌倍：6个点，5张方块本应该叫2D，但尚有4张红心，如叫2D容易错失成局，先叫赌倍后，不管同伴叫什么，再叫2D便可告诉同伴尚有4张红心。
（11）2S：道理和上牌一样，如径叫3C，同伴如黑桃稍为有挡张，因只有4张红心只好叫3NT，那就可能失掉较稳当的4H而却打个不好的3NT。

3、L.H.O二阶插叫后：
　　敌方在二阶插叫，使叫牌空间更减少，因此，回牌人之答叫必须更加精确，回牌人答叫：
派司：0-4点，任何牌型，或5-6点，但无可叫牌组。
赌倍：6-7点或8-12点，无可叫牌组。
二者花色：4-6点，5张以上牌组。
三者花色：8点心上，5张以上牌组。
2NT：敌叫组有挡，如为平均牌型是8-10点，但不一定是平均牌型。
3NT：11-13点的平均牌型，敌叫组有良好挡张。
示叫敌叫组：8点以上，如敌组是高花，则保证有另外一门高花4张，如敌组是低花，则保证两门高花最少4-3以上。
　　回牌人之答叫可能同时适用于几个情况，回牌人必须根据自己的牌选择一最能将牌情叫清楚的叫品回答。
开叫人之再叫以及以后的发展与前一节大致相同，唯叫牌空间更少，所选择的叫品必须更加慎重。

（9）黑档有第一控制--SA。
（10）同伴既然那么强烈的要上满贯而方块又没有什么，那一定有SA、CK的，如没有HK总该有HQ及SQ的，故满贯的机会仍是很浓的，而同伴最怕的是方块挡不住，你方块既然有第二控制当然要叫6C。
4、L.H.O在三阶或高阶窜叫时：
　　对方这种叫品毫无疑问是纯捣乱性的叫法，而我们最头痛的就是这个；对着只表示过有16点以上的牌组、牌型未明的同伴，而对方一蹦就已到三阶甚或四阶，你一定很以处理。叫，或是不叫？叫什么？真的是很难！事实上，所有的制度对这种情况都没有什么妥善的办法，我们只能建议读者一些有点实用的叫品，多数仍要靠本身对牌张的判断作最好的决定。
　　回牌人赌倍：
　　对窜叫到三阶的赌倍可能只是随意性的赌倍，并不保证这门花色上一定有什么力量，只表示是8点以上的平均牌型而无其他合适的叫品；开叫人如有牌组可叫便叫出，是平均牌型时则可考虑派司。而对窜叫到四阶的赌倍通常几乎一定是强制性的赌倍，开叫人除非有很好的理由，否则都应派司，因此回牌必须保证在这门王牌有一些赢墩，不得因为旁门有些大牌点而来赌倍敌方（对着畸型的怪牌，旁门的大牌是靠不住的）。所以，有时虽有8、9个大牌点，但真没有适合叫品也得忍痛派司过去。
　　回牌人叫出比敌叫组高一阶的花色：
　　我们规定这等于是费希平特约（Fishbein）特约叫；就是迫伴的意思；通常对于高花较为兴趣，如敌叫组叫高花，则保证另外一门高花4张。注意的是，如这个叫品已到成局，则不再当作是费希平特约，而是自然的意思。比方1C-4D-4H并非费希平特约而表示是自然的最少5张的红心牌组。当然，这时回牌人也必一定要有8点以上的牌力。
　　回牌人示叫敌叫组：
　　一定是很好的牌，通常是三门都可以配合，开叫人并不需要特别着眼于高花，只要将自己的牌组叫出即可。敌叫组如是高花，即要作类似1C-3S-4S的叫法，回牌人的牌需非常好才行。
　　回牌人作其他的叫品：
　　其他的叫品都是自然的意义，在成局以下都是迫叫，如已至成局则不迫叫，需注意如同1C-4S-4NT的叫品，我们不当作是自然叫，而是迫伴，可能有三门牌组，便也很可能只有二门牌组，开叫人如有所选择时需要最低的花色叫起。
5、R.H.O.插入叫牌后：
　　有时，在回牌人答叫后，R.H.O.（开叫人的右手对方，Righrt Hand Opponent）会加入叫牌，这时,由于回牌人已答叫过一圈，情况较为明了，而且，开叫人并不需要有任何行动，回牌人即有机会再叫牌，因此，在这种情况时，开叫人如持最低限的力量且无可以很经济叫出来的牌组,这时,通常都应派司;所以,要是开叫人能即采取行动,都可以当他有较好的力量。
例 开叫人 L.H.O. 回牌人 R.H.O.
（一） 1C　 --　　1D　　 1S
　　　 1NT
　　这通常应有18-20点，因为回牌人是可能0点的，若只有16点，被一刀杀下来是很惨的。
（二） 1C 　--　　1D　　 1S
　　　 --
　　开叫人这时的派司，一定就是最低限，而且没有5张牌组。
（三） 1C　 --　　1D　　 2S
　　　 --
　　这时的派司，开叫人的力量可能还是低限，但不一定是没有牌组可叫（因为要在三阶叫出来，通常要有较好的力量）。有时点力虽够，但牌组不佳也不一定要叫的。
（四） 1C 　--　 1D　　 1H
　　　　X
　　这个赌倍乃是迫伴赌倍，通常在同伴未表明点力、牌型时，独自一个人是不够资格处罚对方的低阶合约的。因此，类似这样的赌倍，通常都是迫伴赌倍。
（五） 1C　 --　 1S　　 2D
　　　　X
　　这个赌倍则是处罚赌倍；答叫人已表示有8点以上，5张的黑桃牌组，你已经没什么好迫伴的。
（六） 1C 　--　 1H　　 1S
　　2H（1NT）
　　这时的2H（1NT）不再是询问叫，只表示其自然的意义；即在对方插叫后，不再有任何的询问叫。

§10、结论

　　1C乃精准制最重要也是最历害的武器，而且也是本制度与自然叫牌制差距最大的地方。因此，我们必须要用较多的篇幅来解释这些比较复杂的叫品，希望读者能够全盘了解，使用得非常纯熟，这就是你胜利的最大本钱。
