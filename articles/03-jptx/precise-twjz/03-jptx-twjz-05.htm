中華精準制 ( Modified Precision in Taiwan )


1NT 開 叫 ……

1NT : 13 - 15 點平均牌型, 無五張堅強低花
│
├─ PASS : 凡是無成局可能, 且認為 1NT 比其他合約都好時
│
├─ 2C : 詢問高花,通常帶有點邀請成局的味道, 持 4 張高花的 10 - 11 點或 5 張?
│        花的9-10 點是很典型的叫 2C 例子, 有低花牌組而預邀請成局的牌也可先
│        2C 再將牌組叫去
│        │
│        ├─ 2D : 沒有 4 張高花
│        │   │
│        │   └─ 答叫人再叫出新花應都是 5 張以上的牌組, 開叫人應視配合與否
│        │        及高低限而決定是否加叫一局
│        │
│        ├─ 2H : 紅心 4 張, 可能還有 4 張黑桃
│        └─ 2S : 黑桃 4 張, 絕無 4 張紅心
│
├─ 2D :迫叫成局,力量足夠成局而欲找尋最佳合約或是不只成局時都應先叫 2D 迫叫
│        通常最少應有 13 點以上
│        │
│        ├─ 2H, 2S : 4 張牌組, 兩門 4-4 時先叫 2H
│        │   │
│        │   └─ 2NT : 迫叫, 且有滿貫興趣否則應跳叫 3NT
│        │        │
│        │        ├─ 3C, 3D, 3H(3S) : 再叫新花, 低限開叫, 4 張牌組
│        │        ├─ 3H(3S) : 再叫原花色, 4-3-3-3 牌型, 低限開叫
│        │        ├─ 3NT : 4-3-3-3 牌型, 高限開叫
│        │        ├─ 4C, 4D : 高限開叫, 4 張牌組
│        │        ├─ 4H : 高限開叫, 4 張黑桃, 3 張梅花
│        │        └─ 4S : 高限開叫, 4 張黑桃, 3 張方塊
│        │
│        ├─ 2NT : 無 4 張高花, 低限開叫
│        │   │
│        │   └─ 3C : 詢問開叫人有怎樣的低花牌組
│        │        │
│        │        ├─ 3D  : 4 張梅花, 4-3-3-3 的牌型
│        │        ├─ 3H  : 4 張方塊, 4-3-3-3 的牌型
│        │        ├─ 3S  : 5 張梅花
│        │        ├─ 3NT : 5 張方塊
│        │        ├─ 4H  : 兩門低花 4-4, 3 張紅心
│        │        └─ 4S  : 兩門低花 4-4, 3 張黑桃
│        │
│        ├─ 3C, 3D 高限開叫, 5 張牌組
│        │
│        ├─ 3H, 3S 高限開叫, 兩門低花4-4,所叫是高花 3 張牌組, 即 4-4-3-2.
│        │   │
│        │   └─ 答叫人任何叫至成局的合約都是束叫
│        │
│        └─ 3NT : 4-3-3-3 的牌型, 一門低花 4 張, 高限開叫
│             │
│             └─ 4C : 詢問哪門是低花 4 張
│                  │
│                  ├─ 4D : 梅花 4 張
│                  └─ 4H : 方塊 4 張
│                       │
│                       └─ 再叫幾乎都是束叫, 除了 4NT 是問 A
│
├─ 2H, 2S, 3C, 3D :絕對束叫,無法成局的牌但有長門的花色, 通常 2H, 2S 應有 5
│                   張以上, 3C, 3D 要 6 張以上
│
├─ 2NT : 邀請 3NT,10 - 11 點的平均牌型, 通常無 4 張高花, 有 5 張低花時 9 點
│         也可以邀請
│
├─ 3H, 3S : 14 - 15 點以上, 4-4-4-1 的牌型, 所叫是單張的花色
│   │
│   ├─ 3NT : 束叫, 單張花色有良好擋張, 或認為 3NT 為較佳合約
│   │   │
│   │   └─ 4C : 17 - 18 點, 6 個控制以上
│   │        │
│   │        ├─ 4D:如有一門可以配合, 一定要先叫 4D, 然後由答叫人報出自己
│   │        │      的控制數, 由 4H(6 個控制) 開始
│   │        │        │
│   │        │        └─ 之後的下一個叫品都是束叫
│   │        │
│   │        └─ 如果沒有一門花色配合, 則由 4H(13 點) 開始報自己的點力
│   │             │
│   │             └─ 之後的下一個叫品都是束叫
│   │
│   └─ 加高一級詢問控制(跳過 3NT) : 詢問答叫人全手控制數
│        │
│        ├─ 第一級 : 0 - 4 個控制
│        ├─ 第二級 : 5 個控制 
│        ├─ 第三級 : 6 個控制
│        └─ 依次類推 
│        （詢問完後, 開叫人之任何再叫都是束叫）
│
├─ 3NT : 束叫, 認為可以完成 3NT 或是認為不可能有其他更好的合約
│
│
├─ 4D, 4H : 轉換叫, 轉換 4H, 4S, 願意打高花成局的合約而欲讓開叫人做莊, 通常
│            是 6 張以上牌組, 8 - 9 點以上
│
├─ 4S : 兩門 5-5 以上的低花, 應有成局把握的力量, 至少 10 點以上
│
├─ 4NT, 5NT:分別邀請小滿貫及大滿貫, 18 - 19 時邀請小滿貫, 22 - 23 時大滿貫
│
├─ 4C : 哥柏特約, 問 A, 當僅知道 A 的數目即可決定合約時便可以用此叫品
│
└─ 遭敵賭倍後之處理
     │
     ├─ PASS : 通常表示可打 1NT, 7 - 8 點以上的牌力, 但也有可能是 6 點以下
     │          但無路可逃
     │
     ├─ 再賭倍 : 求救信號, 命令開叫人叫 2C, 可能是有 5 張以上的梅花牌組, 或
     │            是逃跑的準備
     │
     ├─ 2C, 2D, 2H : 分別轉換 2D, 2H, 2S, 通常應有 5 張以上的牌組
     │
     ├─ 2S : 迫叫成局, 但如認為一定可成局時, 可跳叫 3NT
     │
     └─ 2NT : 兩門低花 5-5 以上, 力量不多
