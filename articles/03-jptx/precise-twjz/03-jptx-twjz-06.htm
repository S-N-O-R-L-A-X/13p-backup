2C : 11 - 15 點, 6 張以上牌組 , 或 5 張以上牌組帶一門 4 張高花
│
├─ PASS : 凡是無成局可能, 且認為 2C 比其他合約都好時
│
├─ 2D : 迫叫, 詢問力量及牌型; 開叫人答叫後,  答叫人再叫新花仍是迫叫.
│        │
│        ├─ 2H, 2S : 4 張牌組.
│        │   │
│        │   └─ 2NT : 迫叫, 詢問力量及牌型
│        │        │
│        │        ├─ 3C : 低限開叫, 5-4-2-2.
│        │        ├─ 3D : 低限開叫, 5-4-3-1.
│        │        ├─ 3H : 高限開叫, 5-4-3-1 牌型, D 單張.
│        │        ├─ 3S : 高限開叫, 5-4-3-1 牌型, 較高花色單張.
│        │        └─ 3NT : 高限開叫, 5-4-2-2
│        │
│        ├─ 2NT : 無 4 張高花, 低限開叫, 6-3-2-2.
│        │   │
│        │   └─ 3C : 詢問開叫人有怎樣的 3 張牌組
│        │        │
│        │        ├─ 3D  : 3 張方塊.
│        │        ├─ 3H  : 3 張紅心.
│        │        └─ 3S  : 3 張黑桃.
│        │       
│        │
│        ├─ 3C : 無 4 張高花, 低限開叫, 6-3-3-1 or 6-4-3-0.
│        │   │
│        │   └─ 3D : 詢問開叫人有怎樣的牌組
│        │        │
│        │        ├─ 3H  : 6-3-3-1.
│        │        ├─ 3S  : 6-4-3-0, 紅心缺門.
│        │        └─ 3NT : 6-4-3-0, 黑桃缺門.
│        │ 
│        ├─ 3D, 3H, 3S : 高限開叫, 6-3-3-1 or 6-4-3-0.
│        │   │
│        │   └─ 開叫人所叫是短門.
│        │
│        └─ 3NT : 高限開叫, 6-3-2-2
│            
│
├─ 2H, 2S : 應有 5 張以上, 8 - 11 點, 不迫叫.
│
├─ 2NT : 邀請 3NT, 通常無 4 張高花.
│
├─ 3C : 8 - 10 點, 3 張以上 C 支援.
│        
├─ 3D, 3H, 3S : 11 - 12 點, 6 張以上牌組.
│
├─ 3NT : 束叫, 認為可以完成 3NT 或是認為不可能有其他更好的合約.
│
└─ 4C : 竄叫, C 良好支援, 通常 8 點以下.
