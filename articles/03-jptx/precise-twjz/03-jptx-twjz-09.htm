中華精準制 ( Modified Precision in Taiwan )

2NT : 22 - 24 點, 平均牌型, 或如有 5 張不太好的高花平均牌型(5-3-3-2), 或是
│    4-4-4-1 牌型, 單張是大牌(A, K, 或Q)
│
├─ PASS : 0 - 2 點, 認為無法成局
│
├─ 3C : Stayman, 詢問高花
│   │
│   ├─ 3D  : 沒有高花        
│   ├─ 3H  : 4 張紅心, 絕無 4 張黑桃
│   ├ 再叫 4C
│   ├─ 3S  : 4 張黑桃, 絕無 4 張紅心是詢問低.
│   └─ 3NT : 兩門高花 4-4, 回牌人欲叫回四階高花, 必須用轉換叫法花
│
├─ 3D : 有一門 6 張的低花, 探試 3NT, 或是低花牌組的好牌, 約 9 - 10 點, 試探
│   │   配合與否而要上滿貫的牌
│   │
│   ├─ 3H  : 表示回牌人如持梅花牌組, 就可以打 3NT
│   │   │
│   │   ├─ 3NT : 梅花牌組的爛牌
│   │   ├─ 4D  : 方塊牌組的爛牌
│   │   ├─ 4C  : 方塊牌組的好牌
│   │   ├─ 4H, 4S : 梅花牌組, 更好的牌
│   │   └─ 4NT : 黑木詢問 A
│   │
│   ├─ 3S  : 表示回牌人如持方塊牌組, 就可以打 3NT
│   │   │
│   │   └─ 以下同 3H
│   │
│   ├─ 3NT : 不論回牌人是持哪門低花, 都可以打 3NT
│   ├─ 4C  : 不論哪門低花都無法打3NT,希望同伴是梅花時 PASS, 是方塊時叫 4D.
│   ├─ 4D  : 無法打 3NT, 但梅花牌組可打 5C, 方塊時只能打 4D
│   └─ 5C  : 無法打 3NT,不論哪門低花都可成局,回牌人是梅花PASS, 是方塊 5D.
│
├─ 3H, 3S : 5 張以上的牌組, 迫叫一圈, 由同伴決定要高花成局或 3NT
│   │
│   ├─ 3NT : 王牌不支援
│   ├─ 4H, 4S : 王牌可支援
│   └─ 叫出一門新花 : 示叫, 表示此門有第一控制, 高限且王牌極度配合
│
├─ 3NT : 束叫
├─ 4C : 哥柏特約, 詢問 A
├─ 4D, 4H : 轉換 4H, 4S
├─ 4S : 5-5 的兩門低花
├─ 4NT : 邀請 6NT, 9 - 10 點平均牌型
└─ 5NT : 邀請 7NT, 13 - 14 點平均牌型

3C, 3D : 8 - 11 點, 7 張牌組

3H, 3S : 8 - 11 點, 7 張牌組

3NT : 不到 16 點, AKQ 帶頭 7 張高花牌組, 無缺門, 旁門至多 1 張 K

4C : 8 - 11 點, 帶兩張大牌 ( A or K or Q ) 8 張 H 牌組

4D : 8 - 11 點, 帶兩張大牌 ( A or K or Q ) 8 張 S 牌組

4H : 8 - 11 點, 帶一張大牌 ( A or K or Q ) 8 張 H 牌組

4S : 8 - 11 點, 帶一張大牌 ( A or K or Q ) 8 張 S 牌組

新花三級詢問
       ├─ 第一級 : 無控制.
       ├─ 第二級 : 第二 控制 ( 單張或有 K ).
       └─ 第三級 : 第一 控制 ( 缺門或有 A ).
