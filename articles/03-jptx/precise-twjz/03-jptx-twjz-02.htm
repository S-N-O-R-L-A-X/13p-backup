1C : 16 點以上任何牌型, 迫叫
│
├─ 1D : 0 - 7 點
│   
├─ 1H, 1S, 2C, 2D : 16 - 21 點, 通常 5 張, 不迫叫
│   │   │               5 張低花, 4 張高花時, 低限先叫高花, 高限先叫低花
│   │   │               如持高花 4-4-4-1 時, 也可叫
│   │   ├─ 3 張王牌, 4 - 7 點, 簡單支援
│   │   ├─ 4 張王牌, 6 - 7 點, 跳叫支援
│   │   ├─ 4 張王牌, 牌型甚佳, 跳叫成局
│   │   └─ 0 - 3 點, PASS
 │   │
│   ├─ 1NT : 16 - 18 點, 平均牌型
│   │   ├─ 2C  :  Stayman, 詢問高花
│   │   ├─ 2D  : 轉換叫, 迫開叫人叫 2H
 │   │   ├─ 2H  : 轉換叫, 迫開叫人叫 2S
 │   │   ├─ 2S  : 最少 5 - 5 的兩門低花
│   │   ├─ 2NT : 邀請成局
│   │   ├─ 3C  : 束叫, 6 張以上的梅花, 最多一張大牌
│   │   └─ 3D  : 束叫, 同 3C
 │   │
│   ├─ 2NT : 19 - 21 點, 平均牌型
│   │   │
│   │   ├─ 3C   : Stayman, 詢問高花
│   │   │   ├─ 3D : 無高花
│   │   │   └─ 3H, 3S : 高花牌組, 迫叫
│   │   │        └─ 4C : 詢問低花
│   │   │             ├─ 4D : 4 張梅花
│   │   │             ├─ 4H : 4 張方塊
│   │   │             └─ 4S : 兩門低花各 4 張, 或無低花
│   │
│   │   ├─ 3D  : Flint, 迫開叫人叫3H,爛牌,高花長門,希望停在 3 線高花上.
 │   │   ├─ 3H  : 5 張以上的牌組, 迫叫
│   │   ├─ 3S  : 5 張以上的黑桃, 迫叫
│   │   ├─ 3NT : 束叫
│   │   ├─ 4C  : 0 - 4 點, 兩門低花 5-5 以上
│   │   ├─ 4D  : 轉換 4H
 │   │   ├─ 4H  : 轉換 4S
 │   │   └─ 4S  : 5 - 7 點, 兩門低花 5-5 以上, 力量集中在低花, 迫叫
│   │
│   ├─ 3NT : 25 - 27 點, 平均牌型
│   │   ├─ 4C  : Baron, 詢問 4 張牌組, 不限定高花
│   │   │   └─ 從最低有 4 張的花色叫起, 雙方未找到配合之王牌不得 PASS
 │   │   ├─ 4D  : 轉換 4H
 │   │   ├─ 4H  : 轉換 4S
 │   │   └─ 4NT : 邀請上小滿貫(非黑木詢問)
 │   │
│   ├─ 2H, 2S, 3C, 3D : 22 點以上, 所叫牌組至少為半堅強牌組, 迫叫
│   │   ├─ 二度消極答叫 : 0 - 3 點, 任何牌型, (2S, 3C, 3D, 3H)
 │   │   │   └─ 開叫人再叫新花, 迫叫, 答叫人擇其較佳一門支援
│   │   └─ 積極性答叫 : 4- 7點, 有三張王牌優先支援, 否則叫出 5 張以上的.
 │   │                     花色 ,或無王.                         
 │   └─ 3H, 3S, 4C, 4D : 22 點以上, 所叫是單張花色, 迫叫           
 │                  └─回牌人必須馬上決定要打什麼合約,如4-4王牌,4-3王牌或無王
│   
├─ 1H, 1S, 2C, 2D : 8 點以上, 至少 5 張的牌組
│   │
│   ├─ 簡單換叫一門花色(優先考慮, 非跳叫) : 5 張以上的牌組
│   │   ├─ 叫出第二門有 4 張以上的牌組           
│   │   ├─ 再叫其第一次的牌組 : 6 張以上         
 │   │   ├─ 同伴的牌組有 3 張以上支援時可同意     
│   │   └─ 以上皆不合, 若未叫花色有擋, 可叫無王  
│   │    若11點以上,且有4?控制, 要跳叫
│   │   
│   ├─ 同意王牌, 加一支援 : 3 張以上, 為六級詢問
│   │   ├─ 第一級 : 5 張以上, 無大牌(無 A, K, Q)
 │   │   ├─ 第二級 : 5 張牌組, 帶 1 張大牌
│   │   ├─ 第三級 : 5 張牌組, 帶 2 張大牌
│   │   ├─ 第四級 : 6 張牌組, 帶 1 張大牌
│   │   ├─ 第五級 : 6 張牌組, 帶 2 張大牌
│   │   └─ 第六級 : 5 張以上牌組, 3 張大牌
│   │        │
│   │        └─ 再叫一門新花(非跳叫), 為五級詢問
│   │             │
│   │             ├─ 第一級 : 無控制, 3 張或以上小牌
│   │             ├─ 第二級 : 第三控制, 2 張小牌或有 Q
 │   │             ├─ 第三級 : 短門, 單張或缺門
│   │             ├─ 第四級 : 有 K
 │   │             └─ 第五級 : 有 A
 │   │                  │
│   │                  └─ 再叫一門新花, 為三級詢問
│   │                       │
│   │                       ├─ 第一級 : 無控制或第三控制
│   │                       ├─ 第二級 : 第二控制, 單張或有 K
 │   │                       └─ 第三級 : 第一控制, 缺門或有 A
 │   │
│   ├─高花加二支援:19-21 點(包括分配點), 王牌 4 張以上的良好支援, 通常.
 │   │                  是不太平均的牌型
│   │
│   ├─ 高花直接加叫成局 : 16 - 18 點的平均牌型, 4 張以上的王牌支援, 通常王
│   │                      牌上應有 2 個大牌
│   │
│   ├─ 二級跳叫新花 : Splinter, 王牌極度配合, 所叫花色是單張或缺門
│   │
│   ├─ 1NT(1H, 1S 後) : 16 - 18 點, 無 5 張以上牌組, 且無法支援同伴牌組
│   │   │               詢問答叫者手上的控制
│   │   ├─ 2C : 少於 2 個控制
│   │   ├─ 2D : 3 個控制
│   │   ├─ 2H : 4 個控制
│   │   └─ .. : 依次類推
│   │        │
│   │        ├─ 簡單叫回原王牌 : 為六級詢問, 以下同直接六級詢問
│   │        ├─ 跳叫支援王牌 : 低限 16 - 18 點, 3 張王牌, 通常是平均牌型,
 │   │        │                  答叫人可選擇無王或王牌合約
│   │        ├─ 簡單叫出一門非王牌的新花 : 詢問答叫人此花色有幾張大牌
│   │        │   ├─ 第一級 : 沒有大牌
│   │        │   ├─ 第二級 : 1 張大牌
│   │        │   ├─ 第三級 : 2 張大牌
│   │        │   └─ 第四級 : 3 張大牌
│   │        │        │
│   │        │        └─ 再叫一門新花, 同上詢問大牌
│   │        │
│   │        └─ 2NT : 16 - 18 點的平均牌型, 通常對答叫人的牌組無 3 張支援
│   │             ├─ 再叫第二門有 4 張以上的牌組
│   │             ├─ 原王牌有 6 張以上應再叫一次
│   │             └─ 以上皆無可叫 3NT
 │   │
│   ├─ 2NT(1H, 1S 後) : 19 - 21 點, 平均牌型
│   │   │
│   │   └─ 答叫人再叫與 1NT(1H, 1S 後) 大略相同, 有 4 個控制, 11 點以上必
│   │        須僅量跳叫表示
│   │
│   ├─ 2NT(2C, 2D 後) : 16 - 18 點的平均牌型
│   │   │
│   │   ├─ 直接叫出高花 : 好牌, 且有 4 張高花
│   │   └─ 再叫原低花 : 詢問高花, 不太好的牌, 有 4 張高花, 不保證有 6 張
│   │                     低花
│   ├─ 3NT(2C, 2D 後) : 19 - 21 點的平均牌型, 通常不會是太好的牌, 且多數無
│   │                    4 張高花, 否則 19 - 21 點寧願叫 2NT
 │   │
│   └─ 跳叫一門新花 : 一門獨立之堅強牌組, 並不保證有很多力量
│        │
│        ├─ 有 3 個控制以上, 叫出有控制的花色
│        ├─ 少於 3 個控制, 有 3 張王牌配合則支援
│        ├─ 少於 3 個控制, 本身王牌有 2 張大牌則再叫之
│        └─ 以上皆不符, 叫最簡單的無王
│
├─ 1NT : 8 - 13 點或 16 點以上的平均牌型
│   │
│   ├─ 2C : 詢問高花
│   │   │
│   │   ├─ 2D  : 8 - 10 點, 無 4 張的高花
│   │   ├─ 2H  : 8 - 10 點, 4 張紅心, 可能還有 4 張黑桃
│   │   ├─ 2S  : 8 - 10 點, 4 張黑桃, 絕無 4 張紅心
│   │   ├─ 2NT : 11 - 13 點, 4-3-3-3 的牌型
│   │   │   │
│   │   │   └─ 3C : 詢問哪門是 4 張
│   │   │        │
│   │   │        ├─ 3D  : 方塊 4 張
│   │   │        ├─ 3H  : 紅心 4 張
│   │   │        ├─ 3S  : 黑桃 4 張
│   │   │        └─ 3NT : 梅花 4 張
│   │   │
│   │   ├─ 3C  : 11 - 13 點, 梅花和另一門 4-4
 │   │   │   │
│   │   │   └─ 3D : 詢問哪門花色 4 張
│   │   │        │
│   │   │        ├─ 3H  : 紅心 4 張
│   │   │        ├─ 3S  : 黑桃 4 張
│   │   │        └─ 3NT : 方塊 4 張
│   │   │
│   │   ├─ 3D  : 11 - 13 點, 方塊和紅心 4-4
 │   │   ├─ 3H  : 11 - 13 點, 紅心和黑桃 4-4
 │   │   ├─ 3S  : 11 - 13 點, 黑桃和方塊 4-4
 │   │   └─ 16 點以上, 按 11 - 13 點加高一階回答
│   │
│   └─ 2D, 2H, 2S : 5 張以上牌組, 不欲詢問高花, 答叫人按高低限及支援與否
│        │           分四級回答
│        │
│        ├─ 第一級 : 低擋且無支援 ┐
│        ├─ 第二級 : 低擋且有支援 ├ 這裡所謂有支援是 3 張帶 1 張大牌以上
│        ├─ 第三級 : 高擋且無支援 │ 3 張小牌在此不算有支援
│        └─ 第四級 : 高擋且有支援 ┘
│
├─ 2NT : 14 - 15 點的平均牌型
│   │
│   ├─ 3C : Stayman, 詢問高花
│   │   │
│   │   └─ 3D, 3H, 3S : 表無高花或有紅心, 黑桃高花
│   │        │
│   │        └─ 4C : 詢問低花, 若再問 4NT 是自然的意義而非問 A
 │   │             │
│   │             ├─ 4D, 4H : 分別表示有 4 張梅花或 4 張方塊
│   │             └─ 4S : 兩門低花 4-4, 或一門低花也沒有
│   │
│   ├─ 4C : 哥柏特約, 詢問 A
 │   ├─ 4NT : 邀請上小滿貫
│   └─ 5NT : 邀請上大滿貫
│
├─ 3NT : 一門獨立堅強的 7 張牌組, 旁門最多只能有一個 K, 絕不可有 A, Q 則可
│   │    能有幾個, 而且沒有缺門
│   │
│   ├─ 4C : 已知哪門是堅強牌組, 希望回牌人叫出旁門的 K
 │   │   │
│   │   ├─ 4D, 4H, 4S : 有 K 的花色
│   │   │   │
│   │   │   └─ 4NT : 詢問回牌人旁門是否有 Q
 │   │   │        │
│   │   │        ├─ 無 Q 時在五階叫回原王牌
│   │   │        ├─ 有一個 Q 時便在五階叫出
│   │   │        ├─ 有兩個 Q 時則在六階叫出沒有 Q 的那門花色
│   │   │        └─ 旁門三個 Q 都有則在六階叫回原王牌
│   │   │
│   │   ├─ 4NT : 有梅花 K, 或是沒有 K 的梅花牌組
│   │   └─ 叫回王牌 : 沒有 K
 │   │        │
│   │        └─ 再叫一門新花 : 同下面的三級詢問
│   │
│   ├─ 4D : 不知堅強牌組在哪裡, 要回牌人叫出來
│   │   │
│   │   └─ 4H, 4S, 4NT, 5C : 分別表示紅心, 黑桃, 方塊, 梅花是堅強牌組
│   │        │
│   │        └─ 再叫一門新花 : 詢問回牌人對此門之控制(三級詢問)
 │   │             │
│   │             ├─ 第一級 : 無控制
│   │             ├─ 第二級 : 有第二控制, 單張或有 K
 │   │             └─ 第三級 : 按規定, 應該不可能會答出第三級 !!!
 │   └─ 4NT : 同 1C - 3NT - 4C - 4X - 4NT, 詢問旁門是否有 Q
 │   
├─ 2H, 2S, 3C, 3D : 8 點以上, 4-4-4-1 的牌型, 分別表示黑桃, 梅花, 方塊, 紅
│   │               心單張
│   ├─ 2S, 3C, 3D, 3H : 再叫回牌人單張之花色(非跳叫), 詢問回牌人的點力範圍
│   │   │
│   │   ├─ 第一級 :  8 - 10 點 ┐
│   │   ├─ 第二級 : 11 - 13 點 ├ 如單張是大牌, 應酌予扣點
│   │   ├─ 第三級 : 14 - 15 點 │
│   │   └─ 第四級 : 16 點以上  ┘
│   │        │
│   │        └─再叫此單張花色(非跳叫), 詢問回牌人全手的控制數目
│   │            │
│   │            ├─ 第一級 : 0 - 2 個控制
│   │            ├─ 第二級 : 3 個控制
│   │            ├─ 第三級 : 4 個控制
│   │            └─ 以下類推
│   │
│   ├─開叫人直接叫出一門非單張的花色(或經由上面詢問點力後,叫出一門花色)
 │   │   詢問回牌人此門有幾個大牌, 通常就有決定以此門為王牌的意思
│   │   │
│   │   ├─ 第一級 : 此門無大牌 ┐
│   │   ├─ 第二級 : 有一張大牌 ├凡是開叫人再叫出一門花色,都是在詢問這門.
 │   │   ├─ 第三級 : 有二張大牌 │ 花色上有幾個大牌
│   │   └─ 第四級 : 有三張大牌 ┘
│   │
│   └─ 無王 : 無法找到 8 張王牌, 對回牌人單張花色有良好的擋張
│
├─ 3H, 3S : 4 - 7 點, 7 張牌組                               
│                                                            
├─ 4H, 4S : 4 - 7 點, 8 張無 A 或 K 的牌組                   
│                                                             
 ├─ 4C, 4D : 4 - 7 點, 8 張有 A 或 k 的紅心(4C), 黑桃(4D) 牌組
│
│           （開叫人再叫非王牌花色是三級詢問除 4H, 4S是絕對束叫）
│
├─ L.H.O. 賭倍時 :
 │   │
│   ├─ PASS : 0 - 3 點
│   ├─ 1D : 4 - 7 點
│   ├─ 再賭倍 : 8 點以上的平均牌型
│   │   │
│   │   └─ 開叫人如持平均牌型應 PASS, 並視身價情況準備處罰敵方
│   ├─ 1NT : 4 - 7 點, 兩門低花 5-5 以上
│   │   │
│   │   └─ 開叫人應馬上決定打什麼合約
│   │
│   └─ 其他叫品 : 與敵方未賭倍時意義完全相同
│
├─ L.H.O. 一階插叫後 :
 │   │
│   ├─ PASS : 0 - 3 點或 4 點而無 A 或 K
 │   ├─ 賭倍 : 4 - 7 點, 無 5 張高花或無 5 張好的低花
│   ├─ 一階花色 : 4 - 7 點, 5 張以上的牌組
│   ├─ 1NT : 敵叫組有擋, 如是平均牌型一定是 8 - 10 點, 但不一定要平均牌型
│   ├─ 簡單叫出二階高花 : 5 - 7 點, 5 張以上的牌組
│   ├─ 跳叫一門花色 : 8 點以上, 5 張以上的牌組, 通常敵叫組沒有擋, 否則應優
│   │                  先叫出 1NT
 │   ├─ 示叫敵叫組 : 8 點以上, 敵叫組無擋, 且無 5 張牌組
│   ├─ 2NT : 平均牌型, 11 - 13 點, 敵叫組有擋
│   ├─ 3NT : 平均牌型, 14 - 15 點, 敵叫組有良好擋張
│   └─ 跳示叫敵叫組 : 8 點以上的三門牌組, 敵叫組短門
│
└─ L.H.O. 二階插叫後 :
     │
      ├─ PASS : 0 - 4 點, 任何牌型, 或 5 - 6 點, 但無可叫牌組
      ├─ 賭倍 : 6 - 7 點或 8 - 12 點, 無可叫牌組
      ├─ 二階花色 : 4 - 6 點, 5 張以上牌組
      ├─ 三階花色 : 8 點以上, 5 張以上牌組
      ├─ 2NT : 敵叫組有擋, 如為平均牌型是 8 - 10 點, 但不一定是平均牌型
      ├─ 3NT : 11 - 13 點的平均牌型, 敵叫組有良好擋張
      └─ 示叫敵叫組 : 8 點以上, 如敵叫組是高花, 則保證有另一門是高花 4 張
                        如敵叫組是低花, 則保證兩門高花至少是 4-3 以上
