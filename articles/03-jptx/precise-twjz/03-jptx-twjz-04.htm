中華精準制 ( Modified Precision in Taiwan )


1H/1S 開 叫 ……

1H, 1S : 11 - 15 個大牌點, 5 張以上的牌組, 通常開叫的是手上最長的一門牌組, 兩門一樣長時,
│       選擇開叫較高的花色(例外:梅花和高花是6-5, 還是叫高花
│
├─ 加一支援 : 8 - 10 點, 3 張以上王牌支援
│   │
│   ├─短門邀請:直接叫出一門花色,1S-2S後的 3C, 3D, 3H 表示梅花, 方塊, ?
│   │              心的短門邀請,1H-2H後的 2NT, 3C, 3D 表示黑桃, 梅花, 方塊
│   │              的短門邀請
│   │                   
│   ├─長門邀請:1S-2S 後再叫 2NT-3C, 開叫人再叫 3D, 3H, 3S 表示方塊, 紅心.
│   │            梅花的長門邀請, 1H-2H 後再叫 2S-2NT, 開叫人再叫 3C, 3D, 3.
│   │              表示梅花, 方塊, 黑桃的長門邀請, 通常要高限即 14 - 15 點
│   │                   
│   └─直接邀請:開叫人直接叫出 3H, 3S, 通常要高限, 而答叫人若也是高限即 1.
│                點的話就叫成局
│
├─ 加二支援 : 11 - 13 點, 4 張以上或 3 張帶一張大牌的王牌支援
│   │
│   ├─開叫人除非是拿到最低限(11 - 12 點)的 5-3-3-2 牌型, 否則都應叫成局, ?
│   │   非開叫者有特殊牌型, 否則不應試探滿貫
│   │                   
│   ├─直接叫出一們新花:1S-3S 後再叫 4C, 4D, 4H 表示 5 張以上的梅花, 方塊.
│   │   紅心,1H-3H 後再叫 3NT, 4C, 4D 表示 5 張以上的黑桃、梅花、 方塊
│   │                   
│   └─ 轉換叫:1S-3S 後經 3NT-4C 再叫 4D, 4H, 4S 表示方塊, 紅心, 梅花缺門,
│               1H-3H 後經 3S-3NT 再叫 4C, 4D, 4H 表示梅花, 方塊, 黑桃缺門
│
├─ 直接跳叫成局 : 王牌極度配合, 至少 4 張以上
│   │
│   └─開叫人機乎沒有再叫的權力, 除非遇到缺門而且沒有浪廢的力量時才有滿貫的
│       可能, 因此開叫人再叫出即表示缺門的花色
│
├─ 跳叫3NT:13 - 15 點的平均牌型, 王牌有 4 張以上支援, 至少有一張大牌,迫叫.
│   │
│   └─開叫人如果認為只有一局時應該束叫, 如認為有滿貫的希望, 便從最低的花色
│       示叫有控制的花色
│
├─ 二級跳叫新花 : Splinter, 13 - 15 點, 王牌極配合, 所叫是單張或缺門
│   │
│   └─開叫人如果認為只有一局時應該束叫, 如認為有滿貫的希望, 便從最低的花色
│       示叫有控制的花色
│
├─ 一蓋一回答(1H - 1S) : 8 點以上, 4 張以上黑桃, 迫叫一圈
│   │
│   ├─ 支援同伴王牌 : 如有 4 張黑桃, 應該優先支援
│   ├─ 2C, 2D : 應該是 4 張以上的牌組              
│   ├─ 2H : 6 張以上的紅心                         
│   └─ 1NT : 若持 5-3-3-2 的牌型只能叫無王        
│　 （11 - 13 點低限時示弱嘛，14 - 15 點高限時示強，示強不外是跳叫或倒叫）
│
├─ 二蓋一回答:11 點以上, 所叫通常是 5 張牌組, 低花可有 4 張, 1S-2H 要 5 張
│   │
│   └ 與一蓋一相同, 但開叫人凡是超過原來王牌之再叫都應有高限的力量, 即
│      1S-2D-3C 或 1H-2D-3D 等都表示高限, 唯一例外是 1S-2H-3H 一定是低限
│      開叫人若持低限且無可叫的 4 張花色, 都只能在二階叫回原王牌, 因此,
│      1S-2D-2S 並不保證有 6 張黑桃
│
├─ 跳叫 2NT : 16 點以上平均牌型, 通常 16 - 18 點, 不可有 5 張牌組
│   │
│   ├─ 3NT : 11 - 13 點, 平均牌型(5-3-3-2)
│   ├─ 3C  : 11 - 13 點, 不平均牌型
│   ├─ 4NT : 14 - 15 點, 平均牌型(5-3-3-2)
│   ├─ 直接叫出牌組 : 14 - 15 點, 不平均牌
│   └─ 3S  : 6 張黑桃,其餘的在三階叫出是4張的牌組, 跳至四階應有 5 張以上
│
├─ 1NT :有足夠回答的點力,不適合以上叫品, 原則 8 - 10 點,亦可多到 15 點迫叫.
│   │
│   ├─ 第二門 4 張以上的花色         
│   ├─ 再叫本門須有 6 張以上         
│   ├─ 再叫 2NT 必須是高限的平均牌型
│   └─ 一門三張的低花               
│  （依順序優先選擇, 如高限可選擇是否要跳叫或倒叫, 此後答叫人必須馬上決定要打什麼合約）
│
└─ 弱性跳叫新花 : 7 張以上牌組, 不會超過 8 點, 力量集中在所叫的花色
