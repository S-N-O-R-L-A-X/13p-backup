第十一章  最后的改动

正如我在序言里写的那样，这个体系并不是一个一成不变的僵化的体系。由于写这本书花了我数年的时间，这期间自然会有新的思想加入其中。当然，绝大多数试验性的新叫品最后都被淘汰了。我把一些重要的改动列举如下：

改动1：1 - 1 - 1 = 接力

在和Geir Helgemo开始搭档前，我对这个体系进行了一次重新评估，觉得应该将强开叫后的1应叫赋予双重含义：5张以上，8个以上大牌点；或15点以上的均型牌。这样做使得我们可以在1后用1作为接力叫，而1NT则显示套！

1 - 1
1 - ？

1 = 5张以上，8个以上大牌点；
      或15点以上均型牌

1NT = 15点以上，均型牌。后续接力同1 - 1NT，只是点力范围改成了15-16点（低限）或17点以上（高限）。
2 = 表示持有套，后续叫牌原则上同1 - 1 - 1NT。

这一改动为1开叫后的积极性应叫腾出了更多的空间：

1 - 2 = 4-4-4-1牌型，有一个黑花色单张，12个以上大牌点
      2NT = 4-4-4-1牌型，有一个红花色单张，12个以上大牌点
      3 = 4-4-4-1牌型，有一个黑花色单张，8-12大牌点（和前面设计一样）
      3 = 4-4-4-1牌型，有一个红花色单张，8-12大牌点（和前面设计一样）
      3 = 一个坚固的7张套花色，边花没有牌力
      3 = 随你定！（例如可以是一个坚固的7张套花色，边花还有牌力）

改动2：选择性的2开叫

在1997年百慕大杯赛之前，我曾经设计了另一套2开叫方案。这个方案使得我们能够开叫某些很弱的持牌类型。它表示4-8个大牌点，5张高花+4张低花，具体花色不明。这个叫品是非建设性的，说明白一点，就是即使同伴有强开叫的低限实力，也看不到进局的希望。下面是对2开叫的答叫：

2/2 = 如果同伴是这门高花，就打此定约

2NT = 接力叫（至少是强开叫实力，或双高花的好牌）

3 = 希望打同伴的低花

3 = 6张以上，邀叫

3/4 = 阻击叫，同伴如果是套就改正为定约

3 = 6张以上，邀叫

3 = 套（3是新的接力：3 = 4张，3NT = 4张）

3 = 5张+5张低花（3接力：3NT = 套，4 = 套）

3 = 5张+5张

3NT = 5张+5张
