阿斯特罗(Astro) ( ! )

一种当对方开叫1NT, 自己是双套牌的争叫手段. 2 争叫表示至少是 5/4 或 4/5 的和一低花套的牌型, 2 表示至少是 5/4 或 4/5 和任一其它花色套牌型. HCP 的范围大概在 9-14. 其它任何叫品都是自然叫品.

阿斯特罗 可经过同伴间同意还可用于平衡位置, 对方的2NT开叫 (见下) 和对方的 赌博性3NT开叫

伯根加叫(Bergen) ( ! )

双向反德汝利(Two Way Rdrury) ( ! ) (推荐)

当同伴第3/4席开叫一阶高花, 应叫人用 2! 来表示有三张同伴所叫高花支持的限制性加叫, 用 2! 来表示有四张支持的限制性加叫. 后续应叫和上述的反德汝利是一样的, 仅除了当 2! 应叫后, 就失去了 2! 虚叫来表示正常开叫牌力了. 双向反德汝利的一个明显好处就是开叫人能明确自己方的将牌总长度, 额外的将牌将会对开叫人决定是否叫进局至关重要. 双向反德汝利能让一已经PASS了的牌手告诉同伴他有3张或4张同伴所叫高花的限制性加叫牌力. 这个版本的德汝利是我们强烈推荐的, 通过这一约定叫, 开叫人能明确知道己方的将牌总长度, 这和"总墩数定律"也是相一致的.

  2. 跳叫某一高花到三阶邀请进局

  3. 2NT 是逼叫性问叫

赌博性3NT(GNT) ( ! )

1. 10-12HCP 的 7 张坚实低花套, 无缺门或小单张, 开叫 3NT!. (不能有边张A或K). 如果应叫人在其它三门上有止张, 低花无缺门(并且无满贯兴趣), 他就可以PASS. 如果他没有所需要的止张和进手, 他必须叫 4, 开叫人如有必要可以改叫到 4.

2. Acol 3NT 开叫 (推荐)
16-21HCP, 至少6张(一般是7张)的坚实或半坚实的低花套, 旁门至少有两门(一般是三门)有止张, 无缺门或小单张.
注: 应叫人必须不能救开叫人, 如果满贯不太可能, 应叫人就应该PASS.

雅可比转移到低花(Jacoby Transfer to minors) ( ! )

当同伴开叫1NT, 应叫人有一长低花套, 但少于 8HCP, 有限的进手使得NT定约很困难, 可采用雅可比转移到低花约定叫, 将最终定约定位于部分分的低花定约.

1. 简单的SAYC方法:
1NT 开叫后,   2! 应叫表示转移到 3, 应叫人的低花套是就PASS, 是就改叫到3.

2. 更好的方法是让1NT开叫人成为最后的定约主打人:
1NT 开叫后,   2! 应叫表示转移到 3, 2NT! 应叫表示转移到 3.

注 1: 如果1NT开叫后对方争叫(不包括技术性加倍), 雅可比转移到低花就不再采用.
 注 2: 虽然这是SAYC的处理办法, 但更好的办法还是用2! 应叫表示转移到 3, 2NT! 应叫表示转移到 3. 这样一来, 能让持强牌的人做庄.


郎迪和修正的郎迪( Landy and Revised Landy)( ! )

一种简单的对付对方1NT开叫的约定叫品. 2! 争叫是虚叫, 其真实意思是要同伴叫他较好的高花. 其它的叫品都是自然. 修正的郎迪除了和上述的约定一样外, 再将 2! 用作表示双低花套.


拉文萨垫牌法(Laventhal Discards)

拉文萨垫牌法常用于作第一次垫牌. 垫某一花色的小牌 (除了将牌), 是要剩下两门花色中较低级别的花色.
例如: 如果是将牌, 此轮打, 垫小或是要同伴攻.

类似地, 垫某一花色的大牌 (除了将牌), 是要剩下两门花色中较高级别的花色.
例如: 如果是将牌, 此轮打, 垫大或是要同伴攻.

同样道理, 对于无将定约, 此垫牌法也适用.
例如: 此轮打, 垫大是要同伴攻, 垫小是要同伴攻.

莱本索尔(Lebensohl) ( ! )

一种对抗对方在我方开叫1NT后争叫的手段. 从根本上将, 这一手段引入2NT虚应叫. 要求1NT开叫人接力叫 3!. 应叫人然后PASS (如果他有套) 或止叫于他真正的花色套. 在对方的2阶争叫后, 3阶的新花应叫是自然应叫并且是逼叫的, 例如, 1NT-(2)-3! (如果应叫人的叫品是在对方争叫后跳叫, 也一样是自然逼叫的: 例如 1NT-(2)-3!).

如果应叫人在莱本索尔 2NT 后, 叫出的花色比对方争叫的花色级别要高, 那是邀叫进局. 例如, 1NT-(2)-2NT!-(P)-3!-(P)-3!). 这背后的逻辑是:如果应叫人想止叫的话, 他可以在对方的争叫后立即叫2阶的那门花色, 表示是弱牌, 这和标准的叫法是一致的(例如, 1NT-(2)-2).

新低花逼叫(New Minor Forcing) ( ! )

同伴开叫一阶低花或高花, 你应叫一阶高花, 同伴再叫1NT, 你再叫另一门低花表示11+HCP, 逼叫一轮, 虚叫. 并不承诺那门低花的任何牌力. 很象 "第四花色逼叫". 当应叫人有5张高花时, 新低花逼叫常可用来询问同伴是否有三张支持.

当应叫人有5, 4时(同伴开叫一阶低花), 新低花逼叫也很有效, 这时同时问叫同伴是否有4张或3张支持.

当应叫人想在后来的叫牌中邀请同伴叫自己的花色或同伴的花色进局, 新低花逼叫可用作暂时性的叫品. 如果应叫人的第二轮跳叫是逼叫的话, 新低花逼叫是唯一的描述这种邀叫牌力的叫品.

新低花逼叫后, 开叫人要进一步描述他的牌型, 以便应叫人决定最后的定约. 绝大多数情况下, 开叫人还有必要表示自己的1NT再叫是低限还是高限.

注: 竞叫中, 新低花逼叫不再被采用: 应叫人在同伴的1NT后的低花再叫是实叫, 不逼叫.

  1. 奇数牌张表示欢迎那门花色.

    　

  2. 小偶数牌张表示不欢迎那门花色, 而欢迎另外两门中级别低的边花 (不算将牌)

傀儡斯台曼(Puppet Stayman) ( ! )

斯台曼的一种变例. 在同伴的1NT开叫后, 2!应叫是要同伴在没有5张高花时义务性的再叫2!.

  2. 2! 表示有4张.

  4. 3! 表示有4张和4张或5张, 进局逼叫.

  5. 3! 表示有4张和4张或5张, 进局逼叫.

  6. 3NT! 无4张高花, 进局牌力.

  3. 对4NT问叫, 6阶统一的将牌应叫表示1或3个关键张, 比将牌花色级别高的一门花色是缺门. 

注 2:

  1. 罗马关键张黑木问叫人的5NT再叫, 如果不是第一或第二便宜的接力叫, 问叫最低级别的K. 例如, 如果有K, 应叫人就叫6.

  2. 许多牌手认为问叫将牌Q或K是尝试大满贯, 同时表示所有的关键张都不缺. 在此前提下, 同伴间任何一人如果还有未表明的有利牌力都该叫进大满贯.

  3. 罗马关键张黑木是基于将牌已经统一. 如果将牌还没有统一(例如, 1-P-1-P-4NT), 你应该假设最后叫的花色是统一的将牌套, 也就是.

  4. 接力叫不包括统一的将牌花色套, 叫会将牌花色是止叫.

  5. 如果联手还少一张关键张, 继续问叫K可能最后的定约不能停在6阶. 对此要有准备.

  6. 有些牌手叫回6阶统一的将牌来表示没有K, 而直接最便宜的扣叫自己有K的那门花色. 有些牌手叫5NT来表示在比将牌花色级别高的某一花色上有K. 如果你不知道对方应叫的意思, 就问罗马关键张黑木4NT的叫牌人.

不寻常对不寻常(Unusual Over Unusual) ( ! )

一种当同伴开叫一阶高花, 对方争叫不寻常NT的对抗手段.

在对方的不寻常2NT之后, 3和3应叫是虚叫, 分别表示有 和. 例如当同伴开叫1:

  1. 3! 表示有套, 并有足够的牌力逼叫

  2. 3! 表示限制性加叫(或更好)

  3. 3! 表示有套, 但不逼叫

  4. 3! 表示支持, 但低于限制性加叫, 也就是低于10HCP.


五阶高花自愿叫(Voluntary Bid of Five in a major Suit)

自愿的五阶高花叫品是满贯邀请, 同时也是一种问叫. 是问同伴是否有对方所叫花色的第一或第二轮控制, 如果有, 就叫进满贯. 如果你方叫过了三门花色, 对方没有加入叫牌, 那么则是问同伴是否有未叫过的花色的第一或第二轮控制.
