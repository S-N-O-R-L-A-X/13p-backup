我们这个时代的宋词

　

唐诗宋词虽好，离我们总有十万八千里之远。于是，在不经心时，留意起流行歌词。那是我们这个时代的宋词：

郑华绢的《升F大调行板》，记得的是它里面的那句“如果月亮随风摇曳，我会想着你 ”。想起的是在《金锁记》里，长安看过的那轮月亮。

刘若英唱的歌里，记住的是：“桅子花 白花瓣 落在我蓝色百褶裙上 爱你，你轻声说，我低下头　闻见一阵芬芳 那个永恒的夜晚　十七岁仲夏　你吻我的那个夜晚 让我往后的时光　每当有感叹 总想起　当天的星光”。记住她轻轻的说：“爱情总是叫人心中充满疑惑，它是斜雨不断忧愁的晚春”。

王菲的《暖昧》里，“从未等你的眼睛， 从梦中看到苏醒 ，从未跟你畅游逛，怎么知道高兴会忘形，从未给你饮过冰 零度天气看风景 ，从未攀过雪山 ，所以以为 天会继续晴”。听她唱：“等到风景都看透 也许你会陪我看细水长流”。林夕的好词，原该是给她唱的―――“越美丽的东西我越不可碰”。

苏慧伦的歌里，记得她勇敢的说，“我一个人住”。记得她那首《七岁的影子》：“我一直这样相信 / 在这一个城市里 / 有个和我一样的灵魂 / 我每天问着每一座公园和路灯 / 终于有那么一天 / 我听见有人敲门／原来是我七岁的影子／为什么你的快乐真模糊 / 她这样问 / 来吧，我来带你一起去远足。只过一条马路把心房裂缝补一补 ／ 温柔的风儿它将我们抱住／七岁的影子和我在草原上又唱又闹不怕老 / 来吧，我来带你一起去远足 / 只过一条马路 / 解救你缩小的幸福 / 当你的灵魂已变黑的时候 / 当梦想疲惫时／我带你去远足/过一条马路”。

梁咏琪的歌里，记得是她自己写的《花火》：“长路若太短　花火生命更短　双手可触及你　有眼泪仍是暖”。花火，是北野武一部电影吧？指的是烟花，一种最脆弱美丽的东西。

张艾嘉的歌里，记得住的是《最爱》。是《那夜我们谈了一夜的生活》。是《爱情有什么道理》。太多太多。《最爱》干净的钢琴声里，她缓缓的唱“一生只爱一个人，一世只怀一种愁”。这是钟晓阳的词，那个十九岁就写了《停车暂借问》的女子。

赵学而有首歌叫《细水长流》。直觉的认识这是林夕的词，在广东时在“海琪的天空”里听到了，顿时惊为天人，很辛苦的从网上找来MP3，如一江春水的钢琴里，赵学而的歌声喃喃流过。 而且是那么美丽美丽的歌词啊！爱极了那一句“若你双眼是深海 / 便已经浸没我”，还有“若有一天要被分开 / 我远山也踏破”。林夕好像很喜欢用“暗涌”这个词。好一句“你那暗涌如在叫唤我，唤我入内，怎可不奔向你”。配器都是钢琴与弦乐，背景做得很丰富。赵学而的唱功也非常好。里面好多歌都非常喜欢。象“谈一场不后悔的恋爱”，“如若今夜不下雨”，“明天是晴天吧”。那样的名字，给了亦舒，又是一个好故事。这样的词，好象充满了模糊的憧憬。可是那内里澎湃的勇气，“天空晴时，雷霆来时，它都长流”。什么也挡不住那隐隐的暗涌。

丁薇的歌，在校园民谣风行的时候喜欢那首《断翅的蝴蝶》。三宝的曲，黄小茂的词。词写的好。 “多年以前一晃如昨天 / 能再次牵你的手已是有缘 / 褪色的红颜黯然的双眼 / 美丽仍存心中你能否看见 //桃花开呀 杏花落呀 / 种下的情呀根深绵绵 / 前生无缘今生又欠欠 / 断翅的蝴蝶飞不上天 // 等到今天心碎又难免 / 忧伤化白云飘荡在蓝天 / 淡淡的来去浓浓的别离 / 以后日子将有此情可依 // 桃花开呀 杏花落呀 / 好好地待她 好好地过呀 / 情肠如线 三世里系缘 / 恋恋一生此情难定”。还有那爵士味浓浓的钢琴和曲风，让她的歌在记忆不褪色。

郁冬，一个也不太出名的歌手，记住的是他一首《时光流转》：“时间原来就是这么简单 / 轻易的改变我们的笑脸 /春去秋来飘落下的花瓣 /重复在我们的身边”；是那首《北京的冬天》：“北京的冬天嘴唇变得干裂的时候 / 有人开始忧愁想念着过去的朋友 /北风吹进来的那一天，候鸟已经飞了很远 /我们的爱变成无休止的期待”。还有那首《欧洲特快车》，他写给毛宁的歌，歌是好歌，只是给了一个讨厌的人：“我会很快的爱上别的姑娘 ，但是忘不了 忘不了你给我的悲伤， 我会很快的爱上别的姑娘， 但是忘不了 忘不了你给我的悲伤。”

那英的歌里，记得的是那首《最爱这一天》里：“人群中擦肩过有多少动人的容颜，我心里不能忘只有你微笑的双眼”。

忆莲的歌里，好词总是被她的声线盖过。总是不由自主被她绮丽的声音吸引，可是又怎么不被《至少还有你》里“直到感觉你的发线，有了白雪的痕迹”的轰然深情击中！

许美静的歌里，记住的是她的《风的叹息》：“如果、如果、如果， 你不能爱我，就请原谅我的寂寞，如果我还继续爱你，也请原谅我的忧郁，如果、如果、如果，我还能爱你我的忧郁却已打扰了你，就当它是、就当它是，就当它是风的叹息”。如风偶过的呢喃，无名的惆怅。

孙燕姿的歌里，喜欢她的《天黑黑》。初听时叹道被她那偶像的外表骗了，这首歌好听得不得了：“我爱上让我奋不顾身的一个人，我以为这就是我所追求的世界，然而横冲直撞被误解被骗，是否成人的世界背后总有残缺，我走在每天必须面对的分岔路，我怀念过去单纯美好的小幸福，爱总是让人哭让人觉得不满足，天空很大却看不大清楚，好孤独，天黑的时候，我又想起那首歌突然期待下起安静的雨，原来外婆的道理早就唱给我听，下起雨也要勇敢前进……我相信一切都会平息，我现在好想回家去，天黑黑欲落雨。”

蔡健雅的歌里，记得的是她在《纪念》里，哽咽的唱道，“想念变成一条线，在时间里面漫延，长得可以把世界切成了两个面。他在春天那一边 你的秋天刚落叶、 刚落叶。他的样子已改变，有新伴侣的气味、的气味，那一瞬间你终于发现，那曾深爱过的人 （嗯），早在告别的那天已消失在这个世界”。

彭羚的歌，有很多林夕作的词。有很多是用亦舒的书名作歌名的歌。象《要多美丽有多美丽》，象《小玩意》，喜欢《给我一段仁爱路》：“如果花开了就感激，如果分开了就放弃，陪你在路上满心欢喜，是为了风景美丽， 不是为了你。如果心开了就回忆，如果心灰了 就忘记，陪你完成这一段距离，要哭泣都来不及 。只是要求你：给我一段仁爱路的时间，给我一枝花的怀念”。
