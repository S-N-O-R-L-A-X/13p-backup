两年之后的欧锦赛杰作


时间：意大利式周末宵夜两年之后的1959年，

地点：意大利港口城市巴勒莫举行的欧洲锦标赛。

还是齐亚瑞迪的杰作，他主打的这个4H定约，可设计成一个双明手问题：

北 S.K72 ............ 双方有局
.. H.Q54 ............ 叫牌过程如下：
.. D.A54 ............ 南 ...... 西 ...... 北 ...... 东
.. C.J654 ........... .. ...... 福盖特 .. .. ...... 齐亚瑞迪
..................... .. ...... Pass .... Pass .... 1H
南 S.43 ............. Pass .... 1NT ..... 2S ...... 3H
.. S.AK109876 ....... Pass .... 4H ...... Pass .... Pass
.. D.KJ ............. Pass
.. C.K7

西首攻SJ并赢得此墩，续出S5，东以Q赢得，回引SA。齐亚瑞迪第一个英明决定是用HA将吃，西垫C2。鉴于已知东的6张黑心，齐亚瑞迪续出HK，以防西握有JXX，这是他的第二个英明决定，因为东在这一墩上垫了一张黑心。继续打H10西跟小牌，于是他出第三轮将牌到明手的Q，清除了西最后的将牌。形势如下：

北 S.—
.. H.—
.. D.A54
.. C.J654

南 S.--
.. S.987
.. D.KJ
.. C.K7

全付牌如下，庄家应如何打下去？

.......... 北 S.K72
............. H.Q54
............. D.A54
............. C.J654
西 S.J5 ............. 东 S.AQ10986
.. H.J32 ............... H.--
.. D.10963 ............. D.Q872
.. C.A932 .............. C.Q108
.......... 南 S.43
............. S.AK109876
............. D.KJ
............. C.K7

如果CA处于有利位置，定约就能完成。但齐亚瑞迪不考虑这种可能性，因为东若有这样一手好黑心，再加上CA，他无疑是会开叫的。

如果西持有CA，怎样才能完成定约呢？要是明手另外还有一个进张，庄家就能以DJ飞过去，再用DA垫一张梅花。

尽管缺少另外的进张，齐亚瑞迪仍出一张方块到J，再打两轮红心，残局如下：

.......... 北 S.--
............. H.--
............. D.A5
............. C.J6
西 S.-- ............. 东 S.--
.. H.-- ................ H.--
.. D.109 ............... D.Q8
.. C.A9 ................ C.Q10
.......... 南 S.--
............. S.7
............. D.K
............. C.K7

当齐亚瑞迪打最后一张将牌时，西只得垫一张方块，以保住梅花的AX双张（否则，庄家只要出一张小梅花就能树立K），明手丢一张梅花，而东为了保持方块的控制，垫了C10。然后齐亚瑞迪兑现DK，并以C7脱手。如果东以Q赢得，他就只能出一张方块给明手；如果西扑出A，他剩下的一张梅花就只能交给庄家的K。

变化：如果齐亚瑞迪不进行方块飞张，残局将是

.......... 北 S.--
............. H.--
............. D.A5
............. C.J6
西 S.-- ............. 东 S.--
.. H.-- ................ H.--
.. D.109 ............... D.Q8
.. C.A9 ................ C.Q10
.......... 南 S.--
............. S.7
............. D.J
............. C.K7

现在，出H7时西将垫一张方块，而北和东都垫一张梅花，不论庄家怎样打，都无法避免再丢失两墩了。

桥牌真是奥妙无穷啊，这次介绍两副意大利队的牌例只是沧海一粟而已，如此牌技，令人钦佩和仰慕，夜色已深，我该休息了，但不知能不能睡得着……
