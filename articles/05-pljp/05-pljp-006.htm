满贯定乾坤


在副数不是很多的队式赛上，尽管每副牌都不能掉以轻心，但最终对比赛结果产生影响的，往往是大牌。在二队打的难分难解时，1-2副满惯。尤其是大满惯的输赢，则决定了胜败的命运。最近T赛中，上演的戏剧性一幕，就是一个很好的例证。

比赛是在AS和TNT队之间进行的，双方队员如下：

开室：

东 bluelcd

南 jinlongwei

西 gosse

北 55996088

闭室：

东 niniok

南 snoopy2k

西 _lwh

北 mianbao

开始的8副中，TNT队的snoopy做成了一副精彩的6C。而AS队下了。TNT队以8IMP 略微领先。真正决定胜负的，出现在第9副。

东西有局，首先，受到考验的是北。先来个小测验。无局对有局，你的牌是

1.

S xxx

H x

D KQJ98xxx 

C x

2.

S xxx

H xx

D KQJ98xx

C --

你开叫社么？4D/5D？ 似乎没太大差别。5D好象给对手的压力更大些？实战中，两队的北，一个选择了4D，另一个选择了5D开叫。

接着受考验的是北。再来2个小测验。你拿了

1.

S AQJxxx

H A9

D x

C Q9xx

2.

S AQJxx

H A9x

D --

C Q9xxx

有局对无局，对方4D/5D开叫， 你分别叫社么？第1手牌，相对比较简单，无非是叫4S/5S或不叫。第2手牌，则又多了个选择，对了4D/5D，你还可选择加倍。你会如何选择呢？

接着，难题轮到了南。南拿了

S x   H Jxx   D Axxx   C J10xxx

在如下2个叫牌中，你如何叫：

1）开室
　

北       东       南       西
5D       加倍     ？   

2）闭室
　

北       东       南       西
4D       4S       ？

5D/6D/7D？ 还是不叫，或来个心理叫？ 2队的南，分别都采用了Pass/6D。

皮球最终踢到了西。我坐闭室的西，面对如下叫牌 （有局对无局）

北       东       南       西
4D       4S       6D       ？

我的牌是： S K109x   H KQ107x   D x   C AKx

牌实在是很好。6是一定的，7也很有可能。可已经到了6线。几乎完全没有试探的叫牌空间。或许，有牌手认为此时Pass是逼叫的。或6H是S带配合的大满惯试探。Pass，认为是逼叫确有合理处，对方很明显是牺牲叫。可和同伴完全没讨论过这类情况，很难确保同伴会理解这Pass的含义。而6H，则完全有可能问自然的含义。

因此，就只有2个选择，6S/7S。这可棘手的很。面对上面东的牌例1。只有6S，而对牌例2，则7S不错。这是不会有肯定答案的。只有靠猜。

实战时考虑：

1. 7S可能遇到极坏分配。

2. 那边对手也不一定叫到7/同伴也许会加叫7S。

3. 希望snoopy或面包会再以7D牺牲。他们是好手，会判断出这6S是铁的。

最后的一点，特别有吸引力。于是，我赌了6S。叫牌是否按我的计划呢？

6S轮到面包，他想了一会， 不叫。到同伴，没多长迟疑，也不叫。我开始暗暗担心了。只希望没7S。论到了snoopy。他开始考虑了。这是个好兆。亲爱的snoopy你快叫7D吧，心里祈祷着（通常不灵验。那副6C,snoopy就没按我祈祷的，去紧逼我同伴SK+假想的HQ）。

嘿嘿！snoopy真叫了7D！ 把我高兴坏了。可随即又开始有积分担心。要是我们才6S，加倍7D肯定是亏了。那桌同伴未必叫的出7D。可事到如今，也没办法了。Pass ，逼叫，告诉同伴有余力。或许，会有人担心这不叫表示D有第1论控制。但个人以为，在没有叫牌空间的情况下，这不该是个必须条件。

同伴，痛苦地思考了半天后叫了7S。我则心很定， 同伴能叫7S，这7S多半有得打。

首攻以后， 看到了同伴的牌是：S AQJxx H A9x D --- C Q9xxx

叫的很精彩。不是吗？ 我的牌是 S K109x H KQ107x D x C AKx

只有在极端恶劣的分布下，且正确首引，才有下牌。单边5-0分配的概率才2%。我的同伴表现了极好的判断力。

到目前为止，一切顺利。可故事还没结束。7S pass , pass, pass。面包笑嘻嘻的加倍。snoopy忙问是技术性加倍吗？ 哈哈！

snoopy思考了一会，平静的打出了C 4。我倒吸了口凉气。强做镇定，偷偷妙了一下，面包打出的牌。God! 是张S5。 刹那间，满眼金星，一篇黑暗。Lightner加倍啊！
