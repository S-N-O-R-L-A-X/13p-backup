好一个歪打正着！


即使是很高超很熟练的桥牌手有时也会令同伴发生误会，所以，桥艺并不是一种很绝对的玩意儿，歪打正着，因祸得福的牌例常常屡见不鲜，有时候的叫牌和打法可能使同伴误会，也有时候却可能使敌方误会，这就造成了桥牌桌上的一些戏剧性的结果。

.......... 北 KQ10
............. A10983
............. A43
............. 84
西 86542 .......... 东 J93
.. KQJ5 .............. 742
.. J ................. 96
.. K105 .............. A9763
.......... 南 A7
............. 6
............. KQ108752
............. QJ2

双方有局，南家开叫，叫牌过程如下：

南 .... 西 .... 北 .... 东
1 ... X ..... XX .... 2
3 ... Pass .. 3 ... pass
4 ... Pass .. 4NT ... Pass
5 ... Pass .. 6 ... All pass

西家首攻：K

南家第二轮叫3原意是表示梅花有挡张，可以尝试3NT，谁知北家误会南家持有第一轮控制力（指A或缺门），便立即采取行动，硬生生的把合约送上了小满贯。

以西家的牌而言，纵使南家不叫3，他也极可能是首攻K的，南家既然这样叫了之后，他更是不会想到首攻梅花了。

打牌的过程相当细密：明手的A赢第一墩，马上用手中的5将吃一墩红心，接着K，和7给明手的A肃清敌方的将牌，手中再以8将吃一墩红心，随后以2给4到明手，再将吃第四张红心，最后拔A，再出小黑桃，明手K取，以Q和3各垫掉手中的一张梅花，全手只送一墩梅花，大功告成。

这付牌的叫牌固然可谓失误，结果却歪打正着，打牌时庄家把明手所绝对需要的三个桥引计算得清清楚楚，实在是功力不凡的表现，最主要的是能够将手中的2作为第二次到明手的桥梁，倘若失此一桥，必然全功尽弃，这种“过河搭桥法”，高级牌手通常会具备保留这种机会的意识，初学者则必须引起警觉的了，不知你有没有遇到过这样的情况？
