似 非 而 是


以下的牌例选自中游实战。搭档的牌风稳健。双无局。我第一家开叫。持有：

S：10 5 2
H：A K 8 7 3
D：K 9 8 4
C：Q

我们叫精确，持有十二点大牌和五张不错的红心套，我开叫1H，下家争叫1S，同伴盖叫2C，上家不叫，我顺路应叫2D，下家不叫，同伴这时扣叫2S，上家又不叫，轮到我，叫牌过程如下：

南     西     北     东
1H     1S     2C     Pass
2D     Pass   2S     Pass
??

初步讨论：我们目前可能选择的叫品，从2NT至3S。抛开离谱的PASS和明显过高的3NT。这里值得讨论的叫品有：2NT、3C、3D、3H和3S。根据目前的叫牌进程来看，同伴显然是在试探成局，甚至可能是试探满贯定约的第一步。可是从我这个方位来看，成局定约会有多种可能。既有可能是无将成局定约，也有可能是高花成局定约，甚至是低花成局定约。我的回答首先就是要配合同伴合理地寻找成局定约，其次就是针对同伴的进一步可能的满贯定约试探，做出必要的铺垫。下面是同伴可能持有的部分牌例：

同伴可能持有的牌例一：
S：x x
H：x
D：A Q x x
C：A K x x x x

同伴可能持有的牌例二：
S：J x x
H：Q x
D：A Q x
C：A K J x x

同伴可能持有的牌例三：
S：K x
H：x x
D：A x x
C：A K J x x x

同伴可能持有的牌例四：
S：x x
H：x x
D：A Q x
C：A K J 10 x x

下面我们进一步探讨每个叫品的利弊。

2NT：首先我在黑桃上没有挡张，甚至半个挡张如Qxx或Jxx，也没有。同伴有可能持有S：Ax，希望我持有S：Qxx，并由我这个方位做无将成局定约，但是目前我没有确定的理由相信同伴就是持有这样的黑桃而进行扣叫。如果同伴也没有黑桃挡张，而将我的无将再叫加叫到无将成局，他一定会对我的黑桃持牌表示失望。从整个叫牌情况看，防守我们的无将成局定约，西没有理由不首攻黑桃，我也没有理由假设西一定只持有四张黑桃而争叫。同时这个叫品也是最容易忽略掉同伴满贯欲望的叫品。如果同伴持有比牌例三多一个方块Q而少一张小方块的牌，他很可能会错误判断我的点力分布而不试探满贯，采取直接封局的选择。

3C：它可以表示出草花Q的价值，同时否认了持有任何黑桃大牌的可能性。但是这个叫品对目前的牌型反映有一些扭曲，同伴会以为你持有1-5-4-3的牌型。当同伴持有上述可能的牌例一时，他会产生满贯欲望。

3D：此时再叫方块套，同伴可能会误认为我方块套具有额外的长度，即红心和方块套都是5张。这样，当他持有上述牌例三和牌例四时，他会叫5D。但是这个叫品也否认了黑桃的挡张和草花的长度支持。

3H：此时再叫红心套，暗示红心套一般有6张，或强的5张套如A K J 9 x，当同伴持有上述牌例二时，他会考虑红心的成局定约。如果对方首攻黑桃而红心分布是2-4时，四红心定约就有危险。

3S：在同伴扣叫之后的扣叫，往往表示某种该花色的控制。就目前的情况看，它表示高限1H开叫，草花配合，黑桃有控制。典型的牌例为：

S：x
H：A K x x x
D：K Q x x
C：Q x x

显然就目前我持牌的情况看，这个叫品未免夸大了草花的支持和扭曲了黑桃的持牌情况。

从目前的叫牌进程来看，最容易排除的是同伴对我的红心套持有标准支持或强支持的情况。同伴是好牌手，我相信他如果持有红心套的强支持，想打红心成局或满贯定约，他可以采用更加明确的叫品，如跳再叫3H，而不是采用比较模糊的扣叫2S，来向我传达信息并配合我进一步评估联手的牌力与配合情况。所以在剩下的几种情况下，同伴无论试探成局或想进一步试探满贯，所依据的共同基础就是那个强的草花套。所以同伴的草花套一定不劣于AKxxxx。在这个前提下，同伴最关心的就是草花套的配合情况。我的单张Q就是同伴最关心的牌张之一。如果同伴的草花套中还有J或10，这个单张Q的支持就和Qx或xxx的支持几乎相同。通过以上几个牌例可以看出这一点。那么我叫3C，同伴会不会误认为我持有1-5-4-3的牌型，而在持有类似于牌例一的牌时，叫上满贯呢？我认为他不会轻易那么去做的。他会考虑到我可能持有2-5-4-2的牌型，草花为Qx时，也会这么答叫。总之，当同伴持有上述所有牌例时，答叫3C都不会导致大的损失。所以如果需要把以上叫品进行排序的话，我的建议是：3C：10分；3D：7分；3H：3分；2NT：1分；3S：1分。

在实战中，我就是叫的3D，而同伴的持牌为：

S：K x
H：J x
D：Q 10 x
C：A K J 10 9 8

同伴在我答叫3D之后，将定约最后推进到5D。虽然由于种种原因定约做成了，但是我事后经过反复思考，认为3C的答叫带来的好处最大，可能导致的损失最小，尽管用单张Q表示对同伴的支持从表面上看显得不太合乎常规。
