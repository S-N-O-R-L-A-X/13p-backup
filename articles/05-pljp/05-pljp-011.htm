有得有失 （关于牌题五的探讨）
　

双人赛制，东西有局，北开叫
同伴坐北第一家开叫1S，东争叫2H，你坐南，假设你不叫，西也不叫，同伴再叫2S，东又叫出3C，你持有以下的牌，该如何应叫？
S: Q 8 3
H: A J 10 9 5
D: Q J 6
C: J 10

在这一叫牌过程中，由于你前面没有表示出对同伴的黑桃支持，所以你可以再叫3S，或者基于你的牌力，跳叫4S；你也可以考虑对东家的叫牌做一个惩罚性的加倍；还可以在3NT的水平之下扣叫你的红心A；甚至再叫一个3NT。你前一轮已经PASS过了，现在再PASS显得过于消极，而同伴在你第二轮又PASS的情况下几乎不可能做进一步的叫牌，所以不用花时间讨论。其他的叫品似乎与你前一轮的PASS有些离谱，所以这里就不做讨论了。

在分析上述各种叫品的利弊之前，看看我们能够从上述的叫牌过程中获得什么样的信息，能够做出什么样的推论。

首先东家在不利的局况下做了2阶红心争叫，从南家的红心持牌来看，东家的红心长套十分空虚，因此几乎可以肯定，东家至少持有KQ领头的6或7张红心长套；同样，东家在不利的局况下独立地再叫3C，说明他至少持有AQ领头的5张梅花长套（他没有梅花J10）。在双人赛中，有局方的争叫一旦被对方加倍并被打宕1墩，对方的得分就多于自身去做一个部分局定约，所以对方一旦发现这样的机会，几乎立即会进行惩罚。东家的3阶梅花再叫可以说是：“我已经做好了充分的准备，你们随时都可以来惩罚我，但是我不怕！”所以说，南北方要想通过惩罚性加倍从东西方身上获得500分就是天方夜谈。

你在东家第一轮争叫2H时PASS，表示你的直接加倍不是惩罚性的；你希望同伴能够用加倍进行平衡叫牌，然后你再罚放。但是同伴并没有用加倍进行平衡，而采用2S的再叫叫品，说明他的牌型不可能是5-1-4-3，而可能是6-1-4-2、6-0-5-2、6-0-4-3甚至是7-0-4-2。由于你们联手采用平衡性叫品来处理对方争叫之后的第三家叫牌问题，因此同伴的2S再叫并不预示着他有多少额外的牌力。他可能持有的牌大致有以下几种（牌例一至牌例八）：

牌例一：
S：A K 10 x x x
H：x
D：A 10 x x
C：x x

牌例二：
S：A K 10 x x x x
H：----
D：A 10 x x
C：x x

牌例三：
S：A K 10 x x x
H：----
D：A 10 x x x
C：x x

牌例四：
S：A K 10 x x x
H：x
D：A K x x
C：x x

牌例五：
S：A K 10 x x x
H：x
D：A 10 x x
C：K x

牌例六：
S：A K 10 x x x x
H：----
D：A 10 x x
C：K x

牌例七：
S：A K 10 x x x
H：----
D：A 10 x x
C：x x x

牌例八：
S：A K 10 x x x x
H：x
D：A 10 x
C：x x

南家的牌在东家再叫3C之后，除了红心J和梅花J之外，其他的牌点的效用依然存在。也就是说，东家的争叫并没有使得南家的牌力大幅度地贬值。从南北方的防守牌力来看，黑桃和方块充其量只能提供两个防守赢墩，红心有两到三个。如果同伴的梅花为两张很小的点数牌（难以超过明手的梅花将牌），那么东西方3C定约极可能成约。如果东家的牌型更加畸形，如0-6-1-6，3C定约将能够完成。反观南北家的联手，4S定约，3NT定约（如果北家有梅花K，由北家坐庄），都是可能完成的。那么北家持有梅花K的可能性有多大？我认为比他持有方块K的可能性要小得多。

基于以上的分析和推测，我们看看上述南家几种可选叫品的利与弊：

惩罚性加倍：南家不相信东家的叫牌，如果东家的水平一般，加倍成功的可能性就较大。如果南家加倍东家3C定约宕一，而当他发现南北家4S定约稳打稳成时，可能会后悔。

3H：南家此时扣叫红心大牌控制的目的主要有两个，第一，希望北家持有梅花K x时再叫3NT，毕竟南家已经无法在3NT水平之下扣叫梅花来询问同伴北的梅花挡张情况。北家立即可以得知南家有牌力，他从自身的红心短套上判断，南家不可能也持有红心短套，否则南家极可能持有较长的黑桃套而在第一轮跳加叫黑桃支持，或者持有方块长套而在3阶自由盖叫方块。南家自己不叫3NT而进行扣叫，表示南家梅花和红心中有一门花色没有档张。如果北家红心和梅花都没有档张，他可以再叫3S，否则就叫3NT。但是这样叫是有风险的，因为北家不知道南家究竟在梅花上，还是在红心上有挡张，尽管从北家所持有的一手牌来推断，南家持有红心挡张的可能性很大。第二，表示红心有控制，是显示黑桃支持的满贯早期试探叫。如果同伴草花单张，方块至少为A 10 9 x x的长套，南北方6S定约就能够完成。但是，如果同伴真的持有如下的理想牌（牌例九和牌例十）：

牌例九：
S：A K 10 x x x
H：x
D：A K x x x
C：x

牌例十：
S：A K 10 x x x
H：x
D：A K x x
C：K x

那么几乎可以肯定，同伴的再叫要么是3D（牌例九），要么是3S（牌例十）。他不会拿上述两例的一手牌再叫2S的。如果同伴有5张方块套而没有方块K，6S定约将面临黑桃和方块上的恶劣分布，西家在这两门花色中，只要有一门上的中间张稍微好一点，能够吃到一墩，6S定约就要宕，因此南北方满贯定约的前景并不明朗。如果南家先扣叫3H，再在北家很可能的3S回应之后，叫4S成局，北家就知道南家是在试探满贯，因此当北家持有较好的一手牌时（如牌例四或牌例八），他会设想南家可能持有梅花K，红心A，方块K或方块Q J，黑桃Q x x或Q J而直接叫6S（他可能想，南家不在东家争叫2H之后叫2NT是因为南家的红心只有A x x （x））。当然如果北家叫牌细腻的话，他会用5S进行邀叫。这可能就是3H扣叫的唯一不足所在。如果是队式赛，3H可能是更好的叫品。但是我相信，如果真有满贯，在双人赛中，4S上2的分数一定也不坏，只要东家主打3C只宕一。

3S：由于南家没有在东家争叫2H之后直接叫2S，所以南家此时再叫3S，同伴将会认为你持有较弱的牌，他绝对不会再叫的，南北方将丢失4S成局定约。但是南家叫3S，很可能诱使西家竞叫4C，那么一切机会又都回来了，此时南家可以再叫4S去赚取对方一个加倍，也可以加倍惩罚对方的4C定约，争取宕2。但是南家没有必要去冒这样的风险，毕竟这一切都建立在西家竞叫4C 的基础上，南家没有理由确定西家一定会这样叫牌。

3NT：冒险赌博，南家希望西家首攻红心从而超墩。但是东家可以采用加倍与否指示西家首攻他叫过的哪一门花色。

4S：直接进局，这多少忽略了南北方的满贯定约意向，以及北家持有梅花Kx 主打3NT的可能性。但是前面已经分析过，南北家满贯定约的前景并不明朗，而一旦西家在黑桃上有一个止张，南北方4S还有机会，而3NT几乎就没有什么机会了。
