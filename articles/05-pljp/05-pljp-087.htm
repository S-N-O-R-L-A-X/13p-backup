2002年世界奥林匹克大奖赛（3）


2002年世界奥林匹克大奖赛(IOC Grand Prix)- Part 2b
(BBO，Fred Gitleman)(February 18, 2002)
(Article：2002 IOC Grand Prix Part 2)
(翻译：海老公)

巴西队是以老带新的阵容。南家，Gabriel Chagas（查加斯），是世界上最好的牌手之一，也是我最喜欢的牌手之一。他的搭档是27岁的Diego Brenner，本次比赛出现的一颗新星。比赛刚刚开始，我的搭档Joey，坐西拿到了下面这手牌：

  S 94 H J1087653 D A8 C 85

东西无局，西开叫。
大家应该还记得，在对抗澳大利亚那一轮，我在第三家拿到了类似的牌：也是HJ108领头的7张套，7222牌型和边花的一个Ace，局况也是单方无局。当时我考虑诈开叫1H，但最终我压抑了自己的欲望，开叫了3H。这次Joey第一家拿到了这样的牌，你认为他会叫什么呢？

Joey选择了Pass－－遵守体系纪律。我的搭档喜欢这样的阻击叫风格：第一第二家阻击的时候，边花不能有Ace（有的时候甚至King都不能有），我不能说我同意这个规定（这牌我会开叫3H），但这就是Joey的风格。说实在的，我并不介意同伴的阻击叫风格－－只要他总是遵守他的风格就行！

好，现在看看我（东家）的牌吧。

  S 72 H A9 D J53 C KQJ1096

现在忘掉我同伴的牌。Joey和Diego都Pass，到我第三家，你认为我会叫什么？

我选择3C－－不是很标准的阻击叫。如果是第一家或者第二家，我会开叫1C，但在有利局况的第三家，我的目标是破坏叫牌而非建设叫牌，而1C开叫并不是破坏性的。

你一定再想：Fred这个家伙的阻击叫范围如此之宽，他的同伴怎么还没有疯掉呢？好问题！我的搭档总是认为我的叫牌是正常的，而如果我作出处理的叫牌，我负全责。如果你也想和我一样经常采用“处理叫牌”的话，建议你先找一个善解人意的搭档！

Gabriel 查加斯争叫3D，Joey加叫4C。停！想想Joey的牌，他确实是用两张小草花加叫的4C！我必须说我不会这么叫牌，但Joey认为敌人可能有满贯，他必须叫4C抢掉敌人的扣叫空间。

Diego作负加倍，查加斯叫4S结束了叫牌。整个叫牌过程如下：

Joey Diego Fred Gabriel
－！ －    3C！ 3D
4C！ X     －   4S
－   －    ＝

Joey首攻C8。查加斯面对的坐庄是这样的：
        S KJ1063
        H K2
        D Q72
        C 743

        S AQ85
        H Q4
        D K10964
        C A2
忘掉东西的牌，试着读牌并找找正确的坐庄路线。不过没有奖赏：如果你找到了正确路线，你会打宕4S！

查加斯CA吃住第一墩，SA，S5清将到明手，我和Joey都跟了将牌。查加斯接着送草花给我。我继续出草花给庄家王吃，Joey垫H3。庄家出H4到HK，我HA吃住，又出H送回庄家。现在定约的命运在于寻找到方块王子：DJ。

这里并没有DJ的明显证据，但从查加斯看来，Joey的方块张数应该比我多，我已经暴露的牌型是2张黑桃，2张红心以及6张梅花，我的牌型可能是：2236或者2326。如果我是2236，那么Joey就是2722了。在数学的推理以及Joey的开叫Pass作用下，查加斯认为我应该是2326，2张方块；而Joey是：2632，3张方块。

查加斯决定打Joey持有DJ，他选择用DK送出！Joey用DA吃住，然后继续D8出手，查加斯明手出小飞牌，我的DJ击退4S一墩。

我和Joey在第一轮都采用了不寻常的叫牌，而查加斯在最后寻找DJ的时候被这些叫牌误导了。当时我并不知道我们这些叫牌的作用，但结分的时候我们发现如果Joey开叫3H，或者我开叫1C，查加斯可能都会正确找到DJ。这说明了什么？－－就是当你好运的时候，好运气总是跟着你！我认为这正是重大桥牌比赛取胜的关键：你只需要比对手作多一点的正确事情。在这个级别，每个人都能正确的评估他们的叫牌，并且取走自己的赢墩，一切都看谁的状态更好。我希望我们的好运和状态可以继续！

和巴西队的最后一副牌，我方单方无局，我的牌：

  S AJ62 H K8742 D 8 C A82

我开叫1H，查加斯加倍，Joey Redouble显示实力，Diego叫2D，我Pass保留同伴惩罚2D的机会。查加斯Pass，Joey叫3H。3H什么意思？我已经没精力搞清楚它了，我加叫到4H，成为最后定约。

4H    －      －   ＝

首攻是DK，Joey摊下了一手有趣的明手牌。下面是两手的牌：
        S 8754

        C KQ

        H K8742

我的第一感觉是：我不喜欢这个redouble，我会叫2NT，表示H限制性加叫以上，以及四张王牌支持。后来我明白了Joey为什么redouble，而且喜欢上它了：Joey这样叫是为了减慢叫牌过程，增加我们4H买单的机会。敌人很有可能在低花上有一个好牺牲。

DK取到第一墩；查加斯继续DA，我王吃。想想看，如何坐庄？呵呵，4H是铁成的，但怎样超额一墩呢？一个IMP也是点呢：）
