2003年度最佳叫牌奖：巴特·布拉姆利/西尼·拉扎德(美国)

　

这副牌出现在2002年12月于美国凤凰城举行的蓝丝带双人赛中，这是ACBL秋季大赛最重要的项目之一。

西发牌，东西有局
　　　　
S ----　　　　S A 10 7 5
H AKQ7543　 　H 10 6
D A1053　　 　D Q J
C K7　　　　　C AQJ109

西　　　北　　　东　　　南
拉扎德　　　　布拉姆利
1H　　　1S　　　2C　　　3S
4D　　　/　　　4H　　　　/
4S　　　/　　　6C　　　　/
7H　　　/　　　7NT　　　//

　　拉扎德并未选择2C开叫，因为局况对对手十分有利，并且他的S是缺门。在首开了主套后，如其所料，对手挤占了大量叫牌空间，拉扎德面临着如何选择第二轮叫品的问题。他的4D很好地解决了这时的难题，这一叫品既自然，又逼叫，而且保持了尽量低的阶数。东家的4H示选，看上去稍有保守，但他担心存在着可能的失配，不愿在情况未明时叫得过高。这一示选打消了西家对H套的一丝担心，使他相信该套可以吃通，于是西以4S扣叫(唯一一个非自然叫品)显示满贯兴趣，对这手错综复杂的牌进行循序渐进的试探。这个4S使东颇有”拨开云雾见青天”之感，已经可以坦然进贯了，唯一不能确定的是以哪一套做将牌最好。东的6C又是一个极好的描述性叫品，既呼应了满贯试探，又显示出C套的强度，同时表明6C是一个可供选择的定约。如果西家H稍弱而D稍好，持类似这样的牌：
S ---- H A Q x x x x D A K 10 x x C x x
　　则6C就将是致胜的定约。

　　6C叫品给西以极大的鼓舞，他看到CK也价值连城，于是充满信心地挺进7H。东家随即以同样的自信将定约改为7NT，他知道西必有坚强的H，DA，以及至少一个低花K。

　　注意以上叫牌过程中，除4S外每一声都是自然叫，没有用A问叫，也没有扣A。4S表示S缺门，除了建立满贯试探的局势外，对最终的7NT贡献并非最大；4S后的几个叫品，每一声都是一个可打的定约，达到7NT后，两人均坚信将会铁成。

　　关于打牌进程，还有一点枝节可说：庄家用SA停住首攻时，垫掉了明手一张小H。如果H分配正常，只需要六墩H就可完成定约；万一H是最不幸的4-0分布，他准备在飞DK成功后在南家同时持有H和D的止张时对其施行紧逼。

　　帕特里克·乔丹评论：现在很少见到这样的叫牌过程了，通过七个自然叫品，找到了顶分定约，还是在对手施加了干扰的情况下。两个人都处理得非常棒。
