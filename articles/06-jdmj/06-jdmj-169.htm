想象…充分发挥！－伯纳特．马可（1994）


From 将吃－桥牌天地  （1994－BOLS tips－Bernard Marcoux－Imagine...and capitalise）

    苹果树对人们总是有吸引力， 亚当和夏娃，牛顿……艾力克·柯凯许（Eric Kokish）什么？是的，艾力克有一次曾写过，如果你摇动一棵苹果树（一棵桥牌树？？！），十个好的主打手落下来也许只有一个好的叫牌手。

    不知一个好叫牌手的原理和落在牛顿鼻尖上（也许是脚上）的苹果是否是同样原理？牛顿感到的痛楚使他创造了 “万有引力定律” 。 同样， 宕一墩的痛苦促使马修．格兰诺凡特 （Matthew Gruetter）创造了格兰诺凡脱定律：永远不要叫大满贯，如果你不能数到13墩。

    好，牛顿之后又来了阿尔伯特．爱因斯坦，他曾经讲过：想象比知识更为重要，还说牛顿看到的那个从树上很快落下来的实际上是一个曲线球。

    1990年在日内瓦赢得了世界冠军之后，总是投曲线球的加勃利尔·查加斯（Gabriel Chagas）创造了查加斯原理：如果第一副牌就很好或是有运气，以后事情很可能会向着你的道路发展。

    所有这些噜嗦的意思是：如果你按直线方向思索，你便是在运用“重力诺凡特定律”。

    在一次加尔各答赛的第二场：你拿的第一副牌是：

      S AQ87 H K1098753 D A6 C --

    搭挡开叫 1C，你应叫1H ，搭挡跳叫了3NT表示C是坚强的长套，现在便是运用想象的时候：如果搭挡有QX或JX 的H，你有机会做成6H，你没有办法“知道”，你必须想象。

    下午，你的成绩平平，晚上你必须要好。

    想得愈多知道得愈少，更多的发现使你更想叫，你叫了6H。

      明手                  你
      S K8                S AQ87
      H Q                 H K1098753
      D 752               D A6
      C AKQ7532           C --

    明手的牌比你所想象的少了一张，你用D A赢进首攻D K，SK下桌，兑现CAK手上扔一D 和一S ，现在出HQ，这墩赢进了，将牌捕吃D回手，想象某家持JX ，你出将牌K，J和A同时跌出：＋1430  （121MP）。

    第三轮，在你方阻击叫后，对手方打4S和6S 两副都宕了，你方赢了17个IMP，你觉得你不会输了。

    第五轮，无局对有局，你和你的搭挡又占了一个便宜：

              D:QXXXX

    搭挡出HAK和小H，定约主肃清将牌后出CQ，你盖上K。她用A赢进……手上再出CJ？？！坏招，一下：十131MP。

    是否可能？

    第六轮）对手方爬到了5C，有局被加倍，定约主可以只失200分，但又打错了，你又赢了500分，61个IMP。

    第七轮之后，你已赢了61个iMP，还有一半路。

    第八轮， 对手是好手，这一轮不好，第一副就不利，他们叫了曲线球6D。标准球是3NT），做成7D，这是这一轮的第一个负分，下一副，你持：

      S AX H AKQXXXX  D K  C QX

    搭挡      上家    你    下家
     1C        2S     3H     --
     4H        --     4S     --
     5D        --     5S     --
     5NT       --     6C     --
     6H        --     ？

    4S是罗马关键张问叫，5S是问特殊K；5NT表示有SK但不否定其他的K，6C是问CK而6H表示否定，你怎么办？你能知道搭挡是否搭挡持有SQ或DQ吗？不。

    你处于十字路口；你是否按重力诺凡特定律来行事？这手牌你知道你只能数到12墩。也许你应该运用查加斯原理？在搭挡否定了持C K后，你是否应该叫7NT？即使你未能数到13墩，你是否应该想象13墩？你是否应该与重力诺凡特定律对抗？

    是的，而且不会错；从第一副起，运气就跟着你，你做的每件事结果都好，对手们给了你数以百计的IMP。 你在成功的波浪中航行，在这特殊条件下，查加斯原理胜过了重力诺凡特定律，不要与好的节奏相违背，记住第一副牌。你不会失败的，按曲线想：不要照直线，想象比知识更重要，叫7NT！！！

      明手            你
        S KQXX                S AX
        H JXX                 H AKQXXXX
        D AXX                 D KX
        C AXX                 C QX

    你赢了10个IMP而且总分是第二名十841MP。在一般情况下，应按照重力诺凡特定律：不要叫大满贯如果你不能数到13墩，一个快球打在鼻梁上是很痛的。

    在特殊情况，忘记牛顿，按查加斯原理，按照曲线来想，因此，我的BOLS提示是：

    想象…充分利用！
