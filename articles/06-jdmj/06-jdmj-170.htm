如果大牌无用不妨早些打出－周麒（1994）


From 将吃－桥牌天地 （1994－BOLS tips－Qi Zhou－Play your honour earlier if it is of no use）

    下面这副牌是我多年前在一次双人赛中打出的：

          S 10984
          H 842
          D AQ
          C AQJ7

S 73         N     S 5
H K105     W    E  H J963
D J109862    S     D K754
C K6               C 10982
         S AKQJ62
         H AQ7
         D 3
         C 543

    南家主打6S 定约，我首攻SJ，隔了好一会，定约主从桌上打出了DA和DQ，手上用将牌捕吃，我不知道他在想什么，手下双张，出Q飞；单张，放A。很简单么，我是初来，但是我的搭档狄克似乎和每个人都熟识，他曾告诉我，南北两家都是什么特级大师，所以定约主必定有些道理，接着是两轮将牌，再是手中出小C 。

    不知是因为前面有几副好成绩感到兴奋，还是因为对手是两名特级大师而有些紧张，在跟牌时，一个不小心，几张牌跌落在桌上，我连忙放回手中，CK是牌面向上在最上面，按规则如果被人看见则一定要打出，我吃不准是否被看见，怕以后多麻烦，我索兴将它打出，就当它是暴露张。定约主感到很惊奇，他很不情愿地用A盖上。

    CQJ后桌上出一H ，搭档放H9，定约主未飞，而是放上了A，再小将牌下桌，再出H ，这次他放上了Q，我赢得了两墩H ，定约主说：“一下，防守精彩。”我想，搭档的H9确是不错，但也不是很难的教科书打法。

    后来，我坐在桌子旁等候，狄克告诉我，我们在组内得了第一，而且，那副牌被提名为最佳防守，有些赛员跑来向我祝贺。我想他们一定是错把我当做狄克了。

    只是在我们回家的途中，我才知道也有南家做成了6S定约，前面4墩相同，接着也是手中出C，西家跟小，桌上用J飞成功，将牌回手，再出C ，西家出K，桌上放小让过，西家被投入，做成了定约，这一打法被提名为最佳主打，在我们桌上，南家原来也是准备这样打的，由于我的CK提早打出，使他无法采用这一打法。所以，他提名我为最佳防守。当然，我是不配得到这一荣誉的，但从此以后，我注意到，如果一张无用的大牌提早打出，常会发生意想不到的效果，似乎所有专家们都知道，只是，偶然他们也会忘记。

    下面这幅牌是1963年在那不勒斯举行的一次双人赛人打出的。

          S 2
          H A10
          D KQJ7654
          C J93

S A109876   N     S KQJ543
H Q8      W    E  H 53
D 32        S     D A8
C 854             C AQ10
         S --
         H KJ97643
         D 109
         C K762

          西          北          东          南
                                  1S          4H 
          4S          5S          X           ==

    西家是著名的彼德罗·福盖（Pietro  Forquet），意大利蓝队的主力。如果首攻是C ，东家的C10可以逼出定约主手上的K。以后DA进手，东家可以兑现两墩C击败定约。但不幸的是福盖首攻了D 、东家班尼托·葛罗索（Benito  Garozzo）用A赢进出CA，看到西家非鼓励性的C4，他回了一张D 被D10赢进, 手上出HK，防守方看来已无能为力了，但福盖轻轻地放上了HQ，定约主认为其余两张将牌必都在东家手上，他用桌上的HA盖上了K，再出D ，如果东家用将牌捕吃，手上可以盖捕吃，还有H10可作进手下桌，东家扔一C ，手上扔C 。福盖得到了意外的一墩。

    最后一个例子取自1994年远东桥牌锦标赛，中国台北青年队对泰国青年队：

          S AQ103
          H Q85
          D QJ9
          C A92

S 865       N     S J92
H K1072   W    E  H A942
D K85       S     D 763
C K75             C Q108
         S K72
         H J6
         D A1042
         C J643   

    泰国队的南家打3NT定约。西家怀瑞伦首攻H2到J。S到桌上的Q，再DQ输给了DK，接着是防守方拿三墩H再D出手，再是两轮D ，形势如下：

          S A103
          H --
          D --
          C A

S 86       N     S J9
H --     W    E  H --
D --       S     D --
C K7             C Q10
         S K7
         H --
         D --
         C J6       

    南家出C6，西家平稳地放上K，如果这张是单张则西家必定还有三张S，定约主用SK回手再出S，桌上用10飞。定约被击败。

    因此，我的BOLS提示是：“如果大牌无用，不妨早些打出”。
