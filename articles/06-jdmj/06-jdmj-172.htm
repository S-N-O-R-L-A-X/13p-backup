考虑你的垫牌－派屈立克·乔丹（1994）


考虑你的垫牌－派屈立克·乔丹（英国）
From 将吃－桥牌天地  
（1994－BOLS tips－Patrick Jourdain－Consider the discard） 

    “当有两门将牌配合可作选择时，要考虑垫牌”

    从叫牌中，你知道你方有一门花色是5一4配合且另一门花色是4一4，你应该选择哪一门作为将牌，譬如说，已经到了满贯阶段？

    传统的忠告是选择4一4配合：

    任一家用将牌捕吃一墩都能增加一墩，而5一4配合只有在短将牌的一家用将牌捕吃时才有好处。

    但是，这一忠告把桥手们的注意力引导到了错误的方向，不应该去想将牌捕吃或是盲目的相信4一4配合，你应该集中思想考虑如果长套不是将牌，垫牌是否有用。

    为说明这问题，假想你拿了下面这手牌：

    S KQXXX H KQXX D KQJ C X

    你开叫： 1S，搭挡叫了贾可贝（Jacopy）2NT。表示有成局实力和四张支持，有无理由需要找寻另一门配合？ 没有，如果搭挡有四张H。C可以用将牌捕吃一墩。也没有帮助。在第五张S上没有什么牌需要垫。

    如果搭挡持：

    S AXXX H AXXX D AX C QXX

    打6S要比6H好，因为你可以用D垫去一张H而不必担心外面H是4一1分布。相反，如果你持：

    S KQXXX  H KQXX  D AXX  C X

    叫牌仍是：1S一2NT。现在H作将牌就有好处了。如果明手是：

    S AXXX H AXXX D KXX C QX

    6H非常好而6S则是毫无机会。

    贾可贝2NT是很常用的约定， 但是，不少的桥手在后面增加了许多虚叫，以致于难以找到另一门花色的配合，下面这副牌取自1991年，英国队的选拔赛：

      南发牌，南北有局

              H:K843
              D:J5
              C:K74
S:87                 S:J
H:75          北     H:J92
D:KQ643     西  东   D:9872
C:AQ105       南     C:J9632
              S:A105432
              H:AQ106
              D:A10
              C:8

    除一桌外，其余都是以S为将牌（满贯，只有一桌叫了5S）。唯一例外是：

    1S—2NT—3H—4H一4NT—5D—6H都放过。 2NT是逼叫成局。H有支持，3H和4H都是自然。4NT是罗马关键张问。H为将牌。

    西家首攻DK。S做将牌只能拿11墩。

    在上述叫牌中，已知北家没有CA，否贝）他会在3H后作扣叫（扣叫表示S配合，如果H不能支持，北家应该先叫3S）。

    最好是两门高级花色的K都视为关键张，但5D只表示HK，未谈到SK，依靠H能连拿，南家叫了6H。可看出这是一个容易的定约，西家首攻DK，将牌肃清后明手的D垫在S下，．低级花色将牌捕吃就是12墩了。

    再假设南家的牌是：

    S A105432 H AQ76 D AK C 8

    如果叫牌开始是：1S一2NT，建议H作将牌是无意义的：从S下垫牌，唯一有用的将是C。同样的明手，6S比6H要优越。因为如果H外面是4一1分布，你还有机会做好CK作为第12墩。

     （作者是南家，北家是艾德林·托马斯）

    南家勃来安·格鲁卜克和北家埃德加·凯普尘在未受干扰的情况下叫牌如下：

    1D—2S一3S—4D一5NT—6C—7D（6C表示在D的三张大牌中有两张）

    格鲁卜克知道他们两手至少有9张S。他的C可以垫在第五张S下。如果搭挡的布型是6一2一4一1，便可以垫掉手上的两张牌。 因此，他很聪明地选择了D作为将牌，当7D很容易而7S做不成时得到了报偿。

      但如果南家的牌是：

      S K1097 H KQ2 D Q843 C AK

    而叫牌开始是：1D一2S一3S一4D，南家可以看到，唯一从S垫牌有用的是D（有可能外面是4一1分布）。同样北家的牌，6S（或6NT）将会比6D好。

    试试我们的新概念。持：

    S KQXXX H AKXX D KQJX C —

    你开叫1S，听到搭挡叫贾可贝2NT。你是否要找另一门配合？如果要找，找哪一门？

    唯一在第五张S上垫牌有用的是H。因此，你不要建议H作为将牌。如搭挡持：

    S AJXX HXXXX D XX C AKX

    四张S支持，但6S容易，6H毫无机会。

    但是D作为将牌可以多做出一墩。因为S失张可以从H中垫掉，如搭挡持：

    S AJXX  HXXX DA10XX c AX

    打H或无将定约，你只有不多不少12墩，而7D可以摊牌。

    因此，当选择将牌时，想象连出长套后另一手牌垫什么有用。答案常能告诉你哪一门花色作为将牌最好。
