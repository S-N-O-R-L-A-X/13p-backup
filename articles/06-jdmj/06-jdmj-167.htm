豹子胆加倍－齐亚（1993）


（1993－BOLS tips－Zia Mahmood－The panther double）
    
    你热爱桥牌，就象我们之中的其它人一般沉溺其中，但你可曾想过，桥牌中的那一部分最使你感到兴奋呢? 可能是对精细叫牌的喜爱，优良主打所带给你的喜乐，艺术般美妙的防守杀着，或者也许是以上的结合。它们确实都有着不可抗拒的魔力。

    事实上，对我而言，还有第4种，那甚至是更具吸引力的:心理的谋略。我发现它是非常迷人的，而令人惊讶的，虽然已经出版了无数关于叫牌和主打的书籍，几乎没有任何在这主题上的文章。这对我的BOLS心得是个很好的介绍，它直接来自心中。

    心理的谋略能以许多方式表现， 但由于成功的突然而有效的一击而来的满足是最大的。只要问任何庄家，他曾经籍着先攻击他的最弱花色而误导敌方;或是任何冒险者，在通过满贯的路上扣叫他没有的控制，因此阻止了致命的首攻，便可得知。

    这是何等的极乐， 一种无可比拟的心醉神迷，在施展过后历经犹存，令人回味。这些打法的最大优点， 不象牌张技巧的改善需时费年才致完美，它们可以这一般桥友立即使用。采用我的心得:我称为豹子胆加倍。这是一种心理的惩罚性加倍，并不是基于你的牌张证据，而是其它因素:时效，敌方的叫牌，或是他们在牌桌上的举动。

    你可以使用这样的加倍，当下面的状况发生时，而你有的是最微不足道的籍口。

    1。不可能听到的叫牌过程:
    身为南家，你持有:
    S 976 H AQJ103 D 8 C 10842
    叫牌过程
    西       北      东        南
   左敌     同伴    右敌       你
             --      1S        --
   2S        --      --        3H
   --        --      3S        --
   4S        --      --        X
   --        --      ==

    敌方原本停在2S上， 而现在西家要惩罚东的竞叫，有些事出岔了:加倍他们。你有一个很好的首攻，十次中他们会做成一次，这其他九次收获丰硕的加倍将可大大地弥补过来。

    2。你希望敌方逃跑
    这开始于类似的叫牌过程，敌方挣扎到一个定约，而你知道他们可能会因幸运而成约:加倍他们。大量失分的恐惧将使他们移至一个"更安全"的萋息之处。我和大卫。伯克维茨为伴。范德贝尔德大赛。他持有:S J43 H K103 D J104 C J1095

    坐在东家听到下面的叫牌过程

    西       北      东        南
    我              大卫
                               1C
    1H       2H(1)   X         2S
    --       3C      --        3H(2)
    --       3S      --        4S
    --       --      X(3)      5C
    --       --      X(4)      ==
    1。梅花配合，逼叫，否认4张S
    2。寻求无将
    3。完全虚张声势
    4。快乐得不得了

    大卫预先看到4-3配的4S将会因友善的将牌分布而完成，他也知道敌方不知道这件事，所以他加倍; 恶劣的分配将会导致庄家将牌失控以及遭受重大损失，所以南家合理地跑到已知是"安全"的梅花配合。

    多么聪慧--然而他所需的只是小心地倾听叫牌过程， 以及做这豹子胆加倍的勇气。你不需要看起来象只豹子般敏捷，就可以做出象这样的加倍。

    类似的机会发生在如果有一带有长低花的敌方表示出有限牌力，然后在竞叫中"顺道:叫上3NT。

    维持他继续走在他的路上并且加倍， 特别地如果你能看到牌张长得很友善，他可能做成时--机会是他将跑至4线低花。

    以下是典型的例子，身为南家，你持有:   S Q732 H Q62 D KJ7 C 843
    
    西       北      东      南
                             你
                     1C      --
    1D       1H      2C      2H
    3C       3H      3NT     X(1)
    --       --      4C      ==

    1。 敌方约有22--23P(如果假设同伴有8-9P)，但梅花位置看来不利，如果他们赌看看，你也可赌看看，对这种收获而冒险是很值得的。

    3。邀请的叫牌过程

    这是我个人的最爱，无论何时一手有限牌接受邀请时，同伴应准备面对最微不足道的藉口，藉口右能是全然无力的，象是最后的叫品是在长考后才叫的，强列显示叫过头的，或至少是个吃紧的定约。

    西      北      东       南
    你
           1NT      --       2NT
    --     3NT      --       --
          犹豫的
    X      ==

    加倍藉由增加得失， 加诸很大的压力在庄家身上，他现在将会错置大牌之所在，此外，他的心中充满了鬼影及坏分配的假象，将不能明智地做庄。

    如果上面的程序是对于较为冒险的人， 这较为一般的叫牌过程，防家能预见对庄家不利的牌张和分配，则是不可抗拒的，此时敌方象是无助的待宰羔羊，几乎让人感到不忍心下手。

    带着你的新设备，你决定打上一场高赌金的盘式桥牌，很自然地你拿到了一手典型的牌:  S 6 H J982 D Q1063 C KJ98

    叫牌过程:
    西      北      东       南
                             你
    --      --      1C       --(1)
    1S      --      2S       --
    3D      --      3NT      X (2)
    4S      X(3)    ==

    1。当东叫出梅花，这手无趣的牌变得没那么无趣

    2。 豹子胆加倍，叫牌过程是有限的，更有甚之，两门牌都分偏了，这是逮住他们的好机会--或许他们会逃掉

    3。他们真的跑了，很有趣，这手牌是

          S Q1094
          H Q7654
          D A5
          C 74

S KJ853       北      S A72
H K10       西   东   H A3
D K972        南      D J84
C 65                  C AQ1032
         S 6
         H J982
         D Q1063
         C KJ98

    同伴首攻C7，庄家飞牌到K，西手上赢得H换攻，打S到SA，再打一张S，当你垫D时他赢吃SK，然后H出手到明手的HA， 庄家感到很失望，在没有桥路到明手的情况下，他试着打D到DK，北赢吃，西剩下的赢墩只有将牌了。

    结果西吃到3墩S，2墩H和一墩C，倒4，南北得到800分。

    或许庄家可以打得更好些，但大多数人在压力下都不能表现得很好。

    现在诚实点， 如果在还没看过这篇文章前， 你拿到南家看起来不怎么样的牌，你会不会让你的800分溜走呢? 生命中没有一定可保证的，我承认有时候豹子胆加倍会弄巧成拙，反而危害到自身安全，但是请勿担心，如果你的同伴因此向你咆哮，就怪到我身上吧，毕竟这是我的BOLS桥牌心得。
