勇敢的首攻

　

世青赛牌例——以色列 对 阿根廷(摘自世青赛公报)

Board 3. Dealer South. East/West Vul.  

          S A643
          H 10864
          D 86
          C AJ6 
S J5                 S K10982
H Q9                 H J3
D AQJ543             D 109
C Q87                C 10942 
          S Q7
          H AK752
          D K72
          C K53 

开室： 
West  North  East South
                   1H
 2D    3D    Pass  4H
All Pass
西首攻SJ，定约人没有留下机会，用A止住然后2轮将牌。梅花飞牌成功，以色列队得+420。

闭室： 
West  North  East South
                   1H
Pass   2NT*  Pass  3D
Pass   4H   All Pass
2NT表示好的H加叫或更好。

西拒绝争叫并且没有加倍3D。现在定约人很可能放过S首攻，让东赢得并回D，但西家有更与众不同的想法，他攻出H9！
定约人明手放10并用K吃住东家的J。他用梅花飞牌到明手并打出H8。
当定约人看到西家用Q拿住这墩H不免大惊失色了！以色列队赢得了10IMP。

在2000年奥林匹克决赛意大利对波兰也出现过类似的牌。当时Tuszynski作了类似的勇敢首攻，但经验丰富的庄家认为防守方不象会攻出单张将牌，可怕的Lorenzo Lauria象能透视牌一样击下了Q！
