BridgeBase #161 Article

　

本周的牌是一副特殊的坐庄牌。是由Ben Zeidenberg提供给我的，是Steve Mackay最初给了Ben这副牌。我们不知道是谁提供给Steve Mackay的。

你坐在南家，拿到了一副怪牌： S AK8742 H AK9762 D J C --

局况是东西有局，西开叫。

西    北    东    南
pass  pass  5C    6C
pass  6D    pass  6H
pass  pass  pass

西和北pass后，东家开叫5C，你选择扣叫6C，在同伴叫6D后，你将定约修正为6H，希望同伴知道你是很强的双高套牌。最后的定约是6H，西家首攻了CJ。

明手摊下了一手不错的牌：
        S 95
        H Q854
        D Q532
        C Q104
        
        S AK8742
        H AK9762
        D J
        C --

你让明手出CQ盖住CJ，东出CK，你用H2王吃。定约看起来不错，你决定开始清将，出H6到明手的HQ。坏消息，东家垫了一张C，王牌是3-0分配。不过这也没有关系，你开始处理黑心套，从明手出S5，东跟SJ，你上SA，再拔SK，东家又示缺了，这次他又填了一张小草花。不过这没有关系，你继续黑心，明手H4王吃；打H5到暗手的HA，再王吃一轮黑心，将黑心套树立起来了。

此时，你在想如何回手清光将牌，兑现黑心赢墩。。。。。。

“出草花回手多半不行，东家开叫5C很可能有9张草花，还是送D，希望能回手吧！”，你让明手出D2，希望西家赢得这一墩。。。。。。

坏消息：东家确实是9张草花！
更坏的消息：东家出了DK！
最坏的消息：东家还了一张小草花，西家的HJ升级了！
定约下一。

四手牌是这样的：
        S 95
        H Q854
        D Q532
        C Q104
S Q1063            S J
H J103             H --
D A10986           D K74
C J                C AK9876532
        S AK8742
        H AK9762
        D J
        C --

让我们回到第二墩，刚刚王吃过草花，我们该怎样才能完成定约呢？

已经太晚了！致命的错误是在第一墩！我们本来可以完成这个6H定约的！

现在回到第一墩，有两个关键的牌张，我们一定要打对才行：首先，明手要出C4！！假设东家跟了一张小C。。。。。。第二张关键牌：南家垫掉DJ！！庄家接下来的打法和前述的打法相同，但这一次你可以安全的王吃方板回手清将了。南家垫掉DJ的打法是“剪刀妙招”的一种，它切断了防家王牌升级的桥路。

那么，你是否在想，如果东家第一墩用CK超过同伴的CJ，他就能够最后击败你的定约而笑到最后？

不！

南家必须很小心的处理这副牌，第一墩用H2王吃，第二墩用手里的HA清将，发现王牌3-0分配后，提SA，SK，明手王吃小黑心，明手续出CQ，东家盖上CA，你用王牌K将吃，再让明手王吃小黑心。此时，兑现明手的C10，暗手垫掉DJ！此时定约已经到家了：西家如果王吃这一墩，剩下的牌你就都好了；如果西家不王吃，你就简单提掉HQ，将吃方板回手，兑现黑心就可以仅失一墩王牌而成约了。

怎么样？你是否让你的搭档失望了？！
