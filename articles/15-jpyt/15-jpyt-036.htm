沃尔夫止叫信号(Wolff Signoff)


许多打自然制的桥手在同伴开叫低花之後，拿了四五个点，怕同伴会有19个点以上时，可能会 lose game，所以常会作不足点的答叫；或是在同伴所开叫的低花只有两张以下时，做一下保护性的叫牌。但如果开叫人接下来跳叫2NT时，答叫人要如何适当的表示牌形与点力呢?对於开叫1NT为15-17点的桥对而言，这样的再叫2NT通常为18-19点，在答叫人作不足点的答叫时，2NT通常是个不会活的定约(若开叫1NT为16-18点，则再叫2NT应为19-21点) ，如果答叫人为不平均牌形，打三线的部分定约应该会比2NT好很多。 但问题是传统自然叫在开叫人的2NT再叫後，答叫人的再叫都是逼叫成局的，有鉴於此，著名桥手Bobby Wolff提出了以3C为 relay 的转换叫，可以让答叫人清楚的表示牌形与点力，并以他的名字称之为 Wolff Signoff--沃尔夫止叫信号.

一.版本一的规则

    Wolff Signoff 3C 的用法其实与Checkback Stayman的2C用意是类似的，只是一个是用在开叫人再叫2NT，一个是用在答叫人再叫1NT。
    在1x-1y;2NT之後，答叫人的再叫规则为∶
    1.3C转换3D
    2.3D以上叫品为逼叫成局
1.1 1C-1D;2NT的後续发展
    1C-1D;2NT-3C = transfer 3D
              3D = 寻问高花，方块只保证四张
              3H = 四张红心与五张方块
              3S = 四张黑桃与五张方块
              3NT= to play
    1C-1D;2NT-3C;
    3D-pass      = 长方块的弱牌
       3H/3S     = 两门低花套，示叫高花控制，有满贯兴趣
       3NT       = 两门低花套，高花无控制
    1C-1D;2NT-3D
    3H/3S        = 四张套
    1C-1D;2NT-3H
    3S           = 四张黑桃
1.2 1m-1H;2NT的後续发展
    1m-1H;2NT-3C = transfer 3D
              3D = 方块套，如果不使用Walsh答叫则另保证五张红心
              3H = 五张红心
              3S = 五张红心与四张黑桃
              3NT= to play
    1m-1H;2NT-3C;
    3D-pass      = 通常是四张红心与六张以上方块的弱牌
       3H        = 红心套的弱牌，止叫
       3S        = 四张红心与四张黑桃，在转换3D之後叫出超越第一次
                   答花色的叫品为逼叫成局
       3NT       = 至少四张红心与四张梅花逼叫成局的牌，开叫人可尝
                   试5C或6C，或Pass
       4m        = 对同伴低低花的强烈 slam try
    1m-1H;2NT-3D;
    3H           = 三张红心(可能还有四张黑桃)
    3S           = 四张黑桃
    3NT          = none of the above
1.3 1m-1S;2NT的後续发展
    1m-1S;2NT-3C = transfer 3D
              3D = 方块套，如果不使用Walsh答叫则另保证五张黑桃
              3H = 五张黑桃与四张红心
              3S = 五张黑桃，绝无四张红心
              3NT= to play
    1m-1S;2NT-3C;
    3D-pass      = 通常是四张黑桃与六张以上方块的弱牌
       3H        = 五张黑桃与四张以上红心套的弱牌，开叫人可以Pass或修正为3S
       3S        = 黑桃牌祖的弱牌
       3NT       = 至少四张黑与四张梅花逼叫成局的牌，开叫人可尝试5C或6C，或Pass
       4m        = 对同伴低低花的强烈 slam try
    1m-1S;2NT-3D;
    3H           = 四张红心(可能还有三张黑桃)
    3S           = 三张黑桃
    3NT          = none of the above


二.版本二的规则
2.1 1C-1D;2NT的後续发展
    (与版本一相同)
    1C-1D;2NT-3C = transfer 3D
              3D = 询问高花，方块只保证四张
              3H = 四张红心与五张方块
              3S = 四张黑桃与五张方块
              3NT= to play
    1C-1D;2NT-3C;
    3D-pass      = 长方块的弱牌
       3H/3S     = 两门低花套，示叫高花控制，有满贯兴趣
       3NT       = 两门低花套，高花无控制
    1C-1D;2NT-3D
    3H/3S        = 四张套

    1C-1D;2NT-3H
    3S           = 四张黑桃
2.2 1m-1H;2NT的後续发展
    1m-1H;2NT-3C = transfer 3D
              3D = 询问高花，方块不保证
              3H = 六张红心
              3S = 五张红心与四张黑桃
              3NT= to play
    1m-1H;2NT-3C;
    3D-pass      = 通常是四张红心与六张以上方块的弱牌
       3H        = 红心套的弱牌，止叫
       3S        = 对另一门低花的slam try
       3NT       = 对同伴低花的slam try
       4m        = 对同伴低花的强烈slam try
    1m-1H;2NT-3D;
    3H           = 三张红心(可能还有四张黑桃)
    3S           = 四张黑桃
    3NT          = none of the above

    1m-1H;2NT-3D;
    3H-3S        = 答叫人为四四高花
2.3 1m-1S;2NT的後续发展
    1m-1S;2NT-3C = transfer 3D
              3D = 询问高花，方块不保证
              3H = 五张黑桃与四张红心的弱牌!
              3S = 六张黑桃，绝无四张红心
              3NT= to play
    1m-1S;2NT-3C;
    3D-pass      = 通常是四张黑桃与六张以上方块的弱牌
       3H        = 对另一门低花的slam try
       3S        = 黑桃牌祖的弱牌
       3NT       = 对同伴低花的slam try
       4m        = 对同伴低花的强烈slam try
    1m-1S;2NT-3D;
    3H           = 四张红心(可能还有三张黑桃)
    3S           = 三张黑桃，绝无四张红心
    3NT          = none of the above
    1m-1S;2NT-3D;
    3H-3S        = 五张黑桃，但无四张红心
