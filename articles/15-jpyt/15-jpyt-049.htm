自然二盖一应叫后的结构


以下整理自劳伦斯著《二盖一体系-叫牌测验与解答》，并略作修改。

二盖一应叫以后，由于双方已经处于基本逼叫到局的序列中，此时的“速达原则”和基本的“速达原则”是完全不同的。二盖一后的速达以及各种跳叫等占用大量空间的叫牌，都是为了表示“某种特征”的牌，而不是表示自己是低限。

1 开叫人加叫同伴的应叫花色进局。典型序列：
1S 2H
4H 
表示好支持，开叫花色套也不错，但其余两个边花都没有第二轮以上的控制。这个限制条件可以让同伴马上评估出是否有满贯。
典型的牌：S AKJxx H QJxx D Qx C xx （D，C两套都没有控制）

2 开叫人简单加叫同伴的应叫花色：
1S 2C
3C
这么叫显示支持和额外实力或者是低限牌，但支持很好。典型的牌：
S AKJxx H Qx D Ax C Jxxx （支持一般，但有额外实力）
S Axxxx H xx D xx C AKxx （低限，但支持很好）

3 开叫人跳叫原花色，表示是6张以上的坚固套和满贯兴趣。这个叫品确定了将牌，并要求双方开始展开扣叫。例如：
1S 2C
3S
开叫人的牌：S AKQJxxx H Axx D xx C x 
搭档双方应该约定，跳叫的套最多缺少一个关键张，最多输一墩牌。例如：KQJxxxx，AQJ10xx，AKQJxx，是合适的套。

4 开叫人跳叫自己的原花色进局，如：
1S 2C
4S
这种叫牌显示坚固套，否定满贯兴趣，未叫的两个套没有第二轮控制。典型持牌如：S AKQJxxx H Qxx D xx C x （H，D两套都没有控制）

5 应叫人跳加叫开叫人的第一花色，显示满贯兴趣，并邀请同伴开始展开扣叫，例如：
   1S 2C
   2D 3S：满贯兴趣，邀请同伴展开扣叫
   典型持牌：S K10x H Axx D xx C AQJxx

6 应叫人跳加叫开叫人的第一花色到局，显示温和的满贯兴趣和将牌好支持，但未叫花色没有前两轮控制，而且二盖一的花色套也是不错的套，例如：
   1S 2C
   2D 4S：持牌：S AKxx H xx D xx C AQJxx（H、D没有前两轮控制）

7 应叫人简单加叫开叫人的第一花色，显示低限或者不适合5，6描述的牌，例如：
   1S 2C
   2D 2S
   典型持牌：
   S Kxx H xxx D xx C AKJxx；
   S Q10x H xx D Axx C KQxxx；

8 开叫人二阶重叫原花，显示低限示弱，或者6张以上套，低限，例如：
   1S 2C
   2S：低限，S可能6张以上，但5张低限也是这么叫；

9 开叫人的低逆叫或者高逆叫需要额外实力，例如：
   1H 2D
   2S：低逆叫，需要额外实力；

   1H 2D
   3C：高逆叫，需要额外实力；

10 开叫人3阶跳叫新花色，显示55好套，例如：
   1S 2C
   3D：

11 开叫人再叫2NT，显示12-14均型，或者18-19均型，例如：
   1S 2C
   2NT
   如果开叫人是18－19均型，他将再叫4NT；

12 开叫人跳再叫3NT，显示15－17均型，例如：
   1S 2C
   3NT

13 应叫人的2NT再叫只是一个中继性质的叫牌，可以是低限牌，也可能是很强的均型牌，等待同伴继续显示。开叫人此时的3阶叫牌都是自然的，例如：
   1S 2C
   2S 2NT
   3D

14 应叫人在3阶出新花色，需要额外实力，例如：
    1S 2D
    2S 3C

15 如果开叫人的叫品建立了逼局局面，则应叫人在3阶重叫原花色，需要额外实力，例如：
    1S  2H
    2NT 3H

16 应叫人再叫3NT，显示15-17，均型。例如：
    1S 2D
    2S 3NT

17 应叫人跳叫原花成局，显示低限，但独立可打的坚固套，例如：
    1S 2H
    2S 4H

18 第四花色叫牌是自然的，需要额外实力，例如：
    1H 2C
    2D 2S
