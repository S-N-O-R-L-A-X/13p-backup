鲁本索尔（Rubinsohl）约定叫


    Rubinsohl 的使用时机为开叫平均牌的无将遭到敌方插叫後，其观念据说来自Ruben Advance与Lebensohl，其用法采用了Lebensohl观念之一的“二线逼叫”与“2NT转换3C”；另外在三线的叫品则采用了类似 Ruben Advance式的转换叫品，所以亦称为Transfer Lebensohl。

    我们知道Lebensohl的问题之一为无法表示出邀请的牌力， 而在强无将或是Ruben Advance的发展中，转换叫的使用让答叫人或盖叫者的同伴得到了再叫的机会来表示牌力。Rubinsohl即是以三线转换叫来表示邀请以上的牌力。举例来说∶
        1NT (2S) 3C = 方块邀请以上
                 3D = 红心邀请以上
开叫人在持无将低限牌力时，可以叫答叫的花色，答叫人若仅有邀请牌力，便派司，若有逼叫成局牌力便叫新花(套)、无将、或示叫问挡(或在无空间时叫3S可能为问挡，即使敌叫组不为黑桃)，例如∶
        1NT (2S) 3C  (P)
        3D  (P)  P

        1NT (2S) 3C  (P)
        3D  (P)  3H      = 还有四张红心
                 3S      = 询问挡张
                 3NT     = 有挡张，要打

若开叫人持无将高限牌力时，可以叫叫新花(套)、无将、或示叫问挡，例如∶
        1NT (2S) 3C (P)
        3H               = 四张红心
        3S               = 询问挡张
        3NT              = 有挡张，要打

    由於2NT是用来转换3C，接下来用以表示弱牌的状况(Lebensohl的状况)，所以梅花套的邀请便无法表示出来；另外，Rubinsohl无法解决的状况尚有∶
    1.平均牌邀请牌力(不论是否有四张高花要对)
    2.敌方以低花插叫持，五四高花的邀请牌力
    3.如同传统Lebensohl一样，逼叫或邀请的套只有五张，可能因此在不接受邀请的状况下，打到5-2王牌三线定约。

所以说∶Rubinsohl并不是万能的。

    如同许多特约的发展有不同的变化，Rubinsohl也有不同的发展，以下发展是参考本社社友孙世伟的制度修改所得，如有人觉得有更好的发展敬请来信指教。

一、1NT (2C) 之後
(假设插叫为自然)
1NT (2C) 2D/M = to play
         2NT  = 方块套，邀请以上牌力
                 开叫人再叫3D表示不接受邀请，
                         叫3C表示接受邀请，有四张高花，梅花无挡
                         叫3M表示接受邀请，四张花色与梅花挡张
                         叫3N表示接受邀请，梅花有挡，无四张高花
         3C  = Stayman，逼叫成局；
                  开叫人无四张高花时，3D/3NT=无/有挡张
         3D  = 红心套，邀请以上牌力
                  开叫人再叫3H表示不接受邀请
                          叫3S表示接受邀请，四张花色与梅花挡张
         3H  = 黑桃套，邀请以上牌力
                  开叫人再叫3S表示不接受邀请
                          叫3N表示接受邀请，梅花有挡
         3S  = 平均牌，无梅花挡张
         3NT = 有梅花挡张(不一定为平均牌)
         4M  = to play
         4C  = 55M
         4D  = 5M+5D

1NT (2C) 2NT
3C  (P)  3D   = 无挡张的邀请牌力，不逼叫
         3M   = 四张花色
         3NT  = 有挡张，不想对高花
         4x   = 方块满贯试探的示叫


二、1NT (2D) 之後
1NT (2D) 2X   = to play
         2NT  = transfer 3C
         3C  = 因为转换至敌叫组，stayman
         3D  = 红心邀请以上，发展同2C插叫後
         3H  = 黑桃邀请以上，发展同2C插叫後
         3S  = 平均牌，无方块挡张
         3NT  = 有方块挡张(不一定为平均牌)

1NT  (2D) 2NT
3C   (P)   P   = 梅花弱牌
           3D  = 梅花套，无方块挡张，逼叫牌力，可能有四张高花
                 (无法表示仅有梅花邀请牌力)
           3M  = 梅花套与四张高花，有方块挡张
           3NT = 梅花套，有方块挡张

1NT  (2D) 3C
3D            = 无四张高花，无方块挡张
3M            = 四张高花
3NT           = 无四张高花，有方块挡张


三、1NT (2H) 之後
1NT  (2H)  2S  = to play
           2NT  = transfer 3C
           3C  = 方块邀请以上
           3D  = 转换至敌叫组，stayman，四张黑桃
           3H  = 黑桃邀请以上
           3S  = 平均牌，无四张黑桃，红心无挡张
           3NT  = 红心有档，要打

1NT  (2H) 2NT
3C   (P)   P/3D = to play
           3H   = 梅花套，逼叫，红心无挡
           3S   = 梅花套与四张黑桃，红心有挡
           3NT  = 梅花套，红心有挡


四、1NT (2S) 之後
1NT  (2S)  2NT  = transfer 3C
            3C  = 方块邀请以上
                   开叫人再叫3D表示不接受邀请，
                           叫3H表示接受邀请，有四张红心
                           叫3S表示接受邀请，黑桃无挡
                           叫3N表示接受邀请，黑桃有挡，无四张高花
            3D  = 红心邀请以上
                   开叫人再叫3H表示不接受邀请，
                           叫3S表示接受邀请，黑桃无挡
                           叫3N表示接受邀请，黑桃有挡
            3H  = 四张红心，黑桃没挡，逼叫成局牌力；答叫人持四张红心
                   与黑桃有挡张时应从2NT转换3C开始，然後再叫3S
            3S  = 平均牌，无四张红心，黑桃无挡张
            3NT = 有黑桃挡张(不一定为平均牌)

1NT  (2S)  2NT
3C   (P)    Pass/3D/3H = to play
            3S  = 梅花套，逼叫，黑桃无挡
            3NT  = 梅花套，逼叫，黑桃有挡
