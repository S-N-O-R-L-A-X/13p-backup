1阶花色开叫后的约定性应叫：应叫人最经济的跳叫


　　立即跳应叫2NT、限制性3阶加叫及新花示强跳应叫等叫品的一个缺点是，当持强牌且同伴花色也是自己的套时，就没有合适的叫品可以资用了。如：
　　S A4
　　H K52
　　D QJ9843
　　C K3
　　当听到同伴开叫1D后，你就遇到难题了，1H/1S太危险，3NT和3S（SPLINTER）又会误导同伴，因此只有叫双张的2C可以勉强对付。但你将无法在安全的前提下显示你的D好支持。

    简单的解决办法是运用“最经济的跳叫新花”来表示这类牌。下列跳应叫均表示同伴花色好支持的进局逼叫：
　　1C－－2D
　　1D－－2H
　　1H－－2S
　　1S－－3C

　　在这类跳叫之后，开叫人的下一轮叫牌（非跳叫）原则上是自然性的（所有均牌可能的话都应该应叫2NT），而跳叫（或“加叫”应叫人的花色）则为斯普林特（Splinter）扣单缺。例如：

　　1C　　2D
　　3C=表示至少5张套。

　　1D　　2H
　　3C=表示双低花至少9张。

　　1H　　2S
　　3H=表示5张以上H，通常不是5332，但可能是4522或4513牌型。

　　1S　　3C
　　3S=表示5332型或6张以上S。

　　1S　　3C
　　3NT=表示4张以上C。

　　1D　　2H
　　3H=表示单张H。

　　1D　　2H
　　3NT=表示单张c。

　　1H　　2S
　　3NT=表示单张C（D为单张时无法表示，只得叫3H）。

　　1C　　2D
　　3D=表示单张D。

　　所有的4阶花色跳叫均显示缺门。
　　在低花同意之后，3NT经常仍可能是最后定约。双方可在3阶上叫有止张或有大牌的花色，但若一方在3阶叫回已同意的将牌花色，如：
　　1C　　2D
　　2S　　3C
　　或
　　1C　　2D
　　2S　　2NT
　　3C
　　则表示没有兴趣做3NT，之后双方应扣叫继续叫牌。
　　在高花同意之后，3NT不可能成为最后定约，除非同伴双方均持4333牌型，那样的话叫程可能如此进行：
　　1H　　2S
　　2NT　　3NT
　　在其他情况下，应叫人在第一次有机会时叫出的3NT为单张问叫。如：
　　1H 　　　　　　　 2S
　　3C/3D／３H　　　3NT
　　或
　　1S　　　 　　 　　3C
　　3D/3H/3S　　　　3NT

　　在其他情况下，高花配合后的3NT表示有满贯兴趣但担心将牌花色的质量。
