双路重询 Stayman


2 Way Checkback Stayman 

    當開叫一線，同伴答叫一線高花，開叫人再叫 1NT。答叫人的點力範圍已經限定(視強弱無王等而有所不同)，並表示在答叫人的高花沒有四張支持，此時如有較低階的迫叫叫品，便可以幫助雙方尋找最佳的合約。舉例來說，如果答叫人持有五張黑桃5332牌型，邀請與迫叫成局的牌力分別要如何表示? 如果未使用特約，答叫人將沒有第二門花色可叫，叫出2S一般又多為束叫，3S不論是邀請或迫叫成局均易被誤認為 6張以上的花色，同時又容易塞走己方的叫牌空間，於是便發展出這些特約：
    1.New Minor Forcing
    2.Checkback Stayman
    3.Two-way Checkback Stayman


一.答叫人再叫的意義
    以下介紹2 Way Checkback Stayman。簡單的來說，叫 2C表示最多邀請的牌力，叫兩方塊為迫叫成局，二線高花為弱牌，3D以上亦為迫叫成局。
    2 Way Checkback Stayman 在叫牌過程為：1x-1M; 1NT之後，各叫品的意義為：
    1X - 1M; 
    1NT- 2C = 迫開叫人再叫 2D，邀請的牌力
         2D = 迫叫成局
         2M = 弱牌(開叫人通常都要pass)
        2NT = 平均牌的邀請，首次答叫的高花只有四張
         3C = 長梅花弱牌(開叫人通常都要pass)
        4om = 長門 M，另一門低花的 splinter
     3NT/4M = 我就是要打...
    4NT/5NT = 計量邀請6NT/7NT

    1m - 1S;
    1NT- 2H = 五張黑桃與四張以上紅心的弱牌(開叫人pass或改為2S)

    1C - 1M;
    1NT- 3D = 五張高花與五張方塊，迫叫成局

    1D - 1M;
    1NT- 3D = 方塊配合，迫叫成局

    1m - 1M;
    1NT- 3M = 六張牌組，迫叫成局

    1m - 1S;
    1NT- 3H = 五五高花，迫叫成局

    1m - 1H;
    1NT- 2S = 照自然意義推演的話，應是六張紅心與五張黑桃的弱牌?
         3S = 長紅心，黑桃 splinter


二.2C之後的叫牌
    答叫人再叫2C後，開叫人被迫要叫2D，然後答叫人第三次叫牌的意義為：
    1x - 1M; 1NT- 2C;
    2D - pass = 有方塊的弱牌(不論開叫人是否開叫1D)
         2M   = 五張的邀請
         2NT  = 對開叫人花色四張支持的平均牌邀請，也就是說有一門可能沒擋
         3M   = 六張的邀請

    1x - 1S; 1NT- 2C;
    2D - 2H   = 五張黑桃與四張紅心的邀請
         3H   = 五五高花的邀請

    1x - 1H; 1NT- 2C;
    2D - 2S   = 五張紅心與四張黑桃的邀請

    1m - 1M; 1NT- 2C;
    2D - 3m   = 五張支持的邀請
         3om  = 五張高花與另一門低花五張


三.2D之後的叫牌
    1m - 1S; 1NT- 2D;
    2H  = 四張紅心(可能還有三張黑桃)
    2S  = 三張黑桃(絕無四張紅心)
    2NT = 沒有四張紅心，也沒有三張黑桃


    1m - 1H; 1NT- 2D;
    2H  = 三張紅心(可能還有四張黑桃)
    2S  = 四張黑桃(絕無三張紅心)
    2NT = 沒有三張紅心，也沒有四張黑桃

    1D - 1M; 1NT- 2D;
    3C  = 沒有四張紅心，也沒有三張黑桃，四張好梅花，高限
    3D  = 五張方塊

    1C - 1M; 1NT- 2D;
    3C  = 五張梅花
