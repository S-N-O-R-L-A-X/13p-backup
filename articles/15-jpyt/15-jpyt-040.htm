CAPP约定叫与其破解之道


□  在敌方开叫 1NT 之後, CAPPELLETTI 或 HAMILTON 的防御叫牌:
 * 赌倍表示处罚, 通常对弱无王要好的十四点, 强无王则要好的十五六点。
 * 盖叫 2C 表示某一门(通常应该是六张)牌组。
 * 盖叫 2D 表示两门高花, 通常至少两门九张以上。 对此答叫 2N 是询问叫。
 * 盖叫 2M 表示该高花带一门低花; 通常是高花正好五张, 低花只保证四张。(Mike Cappelletti 本人推荐大家: 四张高花五张低花也可以如此叫)
 * 盖叫 2N 表示两门低花。
 * 盖叫 3x 为弱性跳盖叫。
 * 第四家的防御叫牌与第二家同。

□ 超叫人同伴的行动
 * 对 2C 的答叫:
   # 通常答叫 2D 让超叫人叫出牌组; 当然, 同伴自己是长门梅花也可以 pass, 但是机会很少就是了。 超叫人再叫 2M 之後, 答叫人以三线新花表示配合与力量所在, 2N 则为lebensohl (三线上束叫的不错牌组, 约 8-10 hcp)。
   # 答叫 2N 为好牌, 如同 DONT 的赌倍一样, 超叫人再叫用转换的 (3S=梅花高限而 3N=梅花低限), 其後叫出新花为迫叫, 接受其转换通常不迫叫。
   # 答叫 2M 自然, 有邀请意味但不迫叫, 大约 11-13 点, 3M 亦同, 好六张牌组!
   # 如 2C被赌倍时, 以再赌倍表示自己有一门很可能比同伴的花色好的牌组。
   # 敌方插叫时, 超叫人若是叫出2NT则为 lebensohl[1nt-2c-2h-pass-pass-2nt]; 答叫人赌倍为竞叫性赌倍,表示可在三线上竞叫, 2N 则仍为好牌的迫叫。

 * 对 2D 的答叫:
   # 若 2D 被赌倍, 则 pass 表示五张以上的方块, 若超叫人再赌倍表示长门黑桃!若直接再赌倍则请同伴选择一门高花。
   # 若同伴答叫 2M, 则好牌时可以自然或半自然的叫品试探成局:
     (1N)2D-2H;                         (1N)2D-2S;
     2S= 五张黑桃                       2N= 只有四张黑桃
     2N= 4-5-2-2 (通常)                 3C= 5-4-1-3
     3C= 4-5-1-3 或 4-4-1-4             3D= 5-4-3-1
     3D= 4-5-3-1 或 4-4-4-1             3H= 4-6 高花
     3H= 4-6 高花                       3S= 5-4-2-2 或六张黑桃
   # 答叫 2N 为询问叫, 好牌邀请。 以下再叫是:
     3C= 低限; 同伴束叫或 3D 迫叫 ─┬ 3H= 四张红心; 可能五张黑桃
     3D= 高限, 4-5 高花             ├ 3S= 五张红心四张黑桃
     3H= 高限, 5-4 高花             └ 3N= 五五以上高花
     3S= 高限, 4-4 高花
     3N= 高限, 5-5 高花
   # 答叫 3m 为邀请性, 不错的长门低花。
   # 答叫 3M 表示五张以上的牌组, 约 8-10 的竞叫性。

 * 对 2M 的答叫
   # 答叫 2N 询问: 再叫 3m=低限, 3M 高限三张, 3M/3N= 高限 C/D!
   # 其他的答叫通常为建设性, 自然不迫叫。

□ CAPP 的弱点何在, 我们又如何去做最大利用呢?

⒈:在对抗1N的防御叫牌之分类上, CAPP 与 DONT, BroZel 同属 Gladiator 形式，其最大共通点是六张牌组无法立即表示，必须经由一个 PUPPET 或是 
Gladiator 的人为叫品 (在 CAPP 是 2C,在 DONT 是赌倍)，才能在第二圈叫出。〔请注意，Woolsey (或称 Multi-Landy) 用2D 表示一门高花的超叫，意思不太一样，因为後者在敌方插叫一门高花时可立即判别超叫人为另一门高花。〕
   这个六张高花, 也是竞叫方最有力的资产无法立刻表示这一点, 是 CAPP 与 DONT   相同的最大致命伤，而对付的方法也如出一辙，就是『强牌追杀，弱牌阻塞』。
   如果你拿了 (a) S:xx H:Kxxxx D:xxx C:xxx 或 (b) S:xx H:Kx D:Kxx C:xxxxxx 同伴开叫强无王, 下家 2C (CAPP)你怎么办? 答案很简单: 当你拿了 (a), 你自由答叫 2H; 同伴若有四张支持, 他会加叫 3H,等拿了六张黑桃的上家又轮到时, 他会後悔为什么选这个特约。 当你拿了 (b), 立刻自由答叫3C, 束叫! 当下家看著手上的 2-4-5-2, 搞不清同伴是拿黑桃或红心时, 他会暗自问候你的祖宗。 如果你看过前面有关 DONT 的文章, 你就会发现, 这只是简单的『攻彼之短』。

   在 CAPP 的 2C 超叫之後, 很多书上告诉你要按照正常的方式答叫, 也就是赌倍=   原来的Stayman, 其他不变, 我想在此介绍另一种, 比较有效的方法!
   p=  可能弱牌, 但也可能是短梅花, 还不错的牌, 下一圈赌倍就表示这种牌。
   其他的叫品就是表示有两门以上可以打的竞叫。
   X=  至少两张梅花, 通常会有三张, 有邀请力量的牌, 并且使用 Forcing Pass!
   等一下再叫 3C 就是邀请带五张以上的梅花! 当超叫一方开始叫出牌组後,平衡位置上可以拿QTx 之类的王牌赌倍, 直接位置的要稍好些。
   2D= 短梅花的竞叫性, 视为对其他三门的迫伴, 但也可能如 2-4-5-2。 开叫人有高花叫高花, 没有时可束叫 2N 或 PASS (你应有四张)。 敌方叫出牌组时,开叫人可以做合作性的赌倍。
   2M= 自然, 五张以上牌组, 不迫叫, 但开叫人有四张一律加叫 (新花=配合)!
   2N= 方块牌组, 或是上三门的迫伴且迫叫成局 (刚才不敢赌倍)! 开叫人持了方块配合就叫出 3C, 答叫人此时如果去叫高花就是短梅花的 Stayman 了。
   3C= 牌组, 竞叫性。
   3D/H=红心与黑桃牌组的转换叫, 邀请以上的力量!
   3S= 梅花牌组的好牌
   3N= 束叫, 可能是在搞鬼 (在对手的超叫强弱未明时先塞掉他们的空间)。
   4CD=转换叫到 HS! 4S 也是要打但是直接束叫!
   4H= 两高请同伴选一门打!!
   4N= 两低天牌!

   重点是: 在对手能搞清楚配合之前, 先下手把线位拉高。 
 * 如果你拿了 (c) S:xxx   H:x D:KQxxx C:xxxx 就叫 2N (方块)。 同伴有配合的话会叫 3C, 否则束叫 3D!   不论如何, 你应该会打到 3D, 如果你的任一敌方迟疑了半晌才 pass, 记得要请裁判来哟, 国际的锺先生或王先生都很清楚这种状况, 当敌方平衡会还你公道的!

 * 拿了 (d) S:QTx H:KJx D:JTx C:Txxx 你应该赌倍。 等一下不管发生什么事到你大致仍然是赌倍。 你知道你有一局的话, 敌方应该已经翻车了, 何况这个局还未必会在另一桌叫到打成。 

 * 如果拿了(e) S:KTxx H:xx D:QJxx C:QTx 就不要赌倍   两红心, 不过可以放过去给同伴, 如果他不能动刀, 他会叫出四张的黑桃, 你还   可以加叫邀请。 如两高互换, 同伴在敌方的两黑桃之後不能赌倍而有四张红心,他会叫两无王 (take-out), 你就回到原属於你的合约了。

 * 拿了 (f) S:x H:Kxxxxx D:Qxxx C:xx 你就跳叫 3D, 红心牌组的邀请 (这里假设你用15-17 的无王)。 此牌未必是你家的牌, 但是你知道敌方的黑桃已经完了。

 * 拿了 (g) S:x H:xx D:xxx C:Kxxxxxx 跳叫三无王! 超叫人从 7-15 hcp 都有可能   (你要随时注意敌方可能传递的非合法讯息), 虽然敌方的 4M 可能 make, 很可能就被你买到俗又大晚的合约 (信不信由你, 美国的正点王 Paul Soloway 就这样被   我搞到一把双 game ... 同伴的梅花是 Axx, 阿门)。

 * 拿了 (h) S:AQxx H:KJx D:AQxx C:xx 开叫 1N, 却听到 (2C)-P-(P); 此时一定要   以 2D 来重开, 同伴非常可能是也拿短梅花不错的牌。 如果黑花对调则赌倍, 你知道同伴很可能拿了 S:KTxx H:Txxx C:Kxx C:xx, 不要放过他们。

⒉:当敌方用 2D 表示两高时, 敌方最大的弱点乃是对不到较长门的高花 (用ASPTRO 或 Woolsey 甚至老牌的 Landy 都没有这个问题)。 因此你拿了平均牌时, 记得不要轻举妄动... 例如 (i) S:Q9x H:QTx D:ATxx C:xxx, 赶快 pass!! 敌方选   2H 之後才赌倍, 他们打到的很可能是高花 5-4 对 3-3 最可怜的合约, 同伴如同为平均牌, 可能敌方就莫名其妙翻车了。 拿 (j) S:KT9x H:Qx D:AJxx C:xxx 就   赌倍, 你知道如果他们跑到红心, 要杀就不能你自己来出面! 其实只要同伴赏脸,红心长个 Kxx, 告个三圈王下来, 敌方的两红心想不当一屁股似乎还不太容易。
 
 * 当你拿了不平均牌又该如何呢? 我的推  啦 ... 三低花就是要打啦, 多利用敌方两高的长度没出来这个特质, 所以 (k) S:xx H:xxxx D:KJxxxx C:x 就叫 3D, 看拿了 3-3 高花的下家如何知道该打黑桃而非红心。 两红心表示方块, 邀请点的牌, 可能是两低或者独门; 两黑桃表示梅花。 在这个转换叫之後, 开叫人都可以束叫 2N, 在 2H 之後还可以用 2S 询问牌情。 当接受了邀请, 两个人三线叫出高花都是询问 (不是表示喔) 挡张。 像是 (l) S:xxx H:Kx H:KJxxxx C:xx 就是两红心的典型罗。 直接的两无王是竞叫性, 表示两高有点东西, 若是无身价时,   敌方可能 110 或 140 被你以 50 买到。 直接跳叫 3M 有两种用法, 我偏好自然   叫 (预防敌方搞鬼或是碰到四小 ...) 不过你要打问挡张我也不反对, 如果是後者那你长门黑桃就必须先赌倍 2D 然後再叫黑桃了。 不论如何, 4M 就是要打。

⒊:两高花的盖叫代表五张高花带四张低花, 看来不错! 但是这是一个『死亡陷阱』!   请问设身处地的作为超叫人的同伴, 你何时能确认打同伴的低花, 合约比高花来   得好   这就对啦, 你需要至少四四以上的低花, 还得多打一线! 这就是作为无王   这一方的优势所在! 当我方握有较多的大牌点时, 往往可以使敌方的旁门吃不到, 这就是对抗 Cappelletti (与相同的 Woolsey 又称 Multi-Landy) 的两高花超叫所需要掌握的关键。 当你的上家超叫两黑桃, 你持了以下的牌, 都应该先赌倍:

   (m) S:Qxx H:xx D:KJ9x C:QTxx    (n) S:KTx H:Q9xx D:QJxx C:xx
   (o) S:Qxxx H:Kxx D:Kxx C:Kxx    (p) S:xx H:xxx D:KTxx C:KJ9x

   持了两小去赌倍或许有些令人不安。 实际上这种赌倍, 是所谓的『合作性』或者『两用』赌倍。 同伴持了两张黑桃或某些牌值不利防守的三小黑桃, 不会留下的!   当你赌倍, 你表示了至少加叫到 2N 的牌值, 同伴在无法罚放时可以自己补到 3N   或以巧叫寻求高花配合。 持了 (o) 之所以赌倍乃是因为牌值太好之故。 如改成   (q) S:QTxx H:xx D:Kxx C:Jxxx 就先放过去, 静待同伴的重开叫。 如果不喜欢,   或者无法接受这样的赌倍, 我建议你改打『虽仅三张王牌, 但靠牌值就可以赌倍』这样的方式, 如此你势必要对 (p) 的牌另寻发展, 那就看你怎么干了。 如果你平常对两高花超叫用 lebensohl, 此时当然也应该照常使用。 不论如何, 或许会惊悸於持 (p) 与 (o) 做相同的赌倍, 实际上当你持了(o) 而赌倍, 在同伴牌组短门的下家往往会冒险逃跑, 於是送给你们一个礼物。

 * 在此我再度强调, 当 1N(2M)X 到了第四家时, 他如果长考才 pass 是个非常有危险性的举动, 因为他传递了『我只有两张王, 我怕被罚放; 可是我出来对低花又怕碰到 4-3 多打一线』的讯息给同伴。 如果超叫人此时逃跑, 和前面一样, 记得要请裁判来。 在非合法讯息传递之後拉出自己的低花已成为『受限叫品』了, 除非他低花真的有六张之多, 或是加一百 的漂亮牌组, 否则即使是逃到较佳的配合, 你都有理由要求裁判主持正义的。

   一般而言, 在低阶的合约被赌倍时, 赛员应尽的礼节乃是在平稳的节奏(约二三秒的间隔)下 pass 或作出逃跑的动作。 过快或是过慢的 pass 都是很不恰当的。

⒋:对付不寻常无王的方法大多数情况下相同。 使用不寻常无王的人, 通常不会正确   的注意两门牌组的超叫所应有的攻击力。我的推  是: 3C 代表黑桃或两高, 3D 代表红心, 两者都有邀请点; 3M 就是竞叫性, 4CDH 则如同 2C 之後。 赌倍的牌如 (r) S:Qxx H:xxxx D:xx C:AJxx 或 (s) S:AJx H:xx D:J9xx C:Qxxx 般, 至   少对一门低花具有强烈的处罚意愿, 而先 pass 之後下圈赌倍则表示有防御力量, 但如同 (t) S:Kxxx H:Kxxx D:xxx C:Ax 这样, 处罚的意愿并不那么强烈。

□ 祝你对抗 CAPP 愉快! 在此要请你注意, 我并没有保证你每一付牌都会成功, 而且如果你照我所说的去做, 你必然的会碰到一些 -470, -730 这样的尴尬场面, 队友可能也会责怪你过分冒险。 但正如 Kit Woolsey 所说, 如果你赌倍十次, 没有失败一次, 那显然是你赌倍的太少了; 也顺便磨练一下自己的防御吧!!
