非典型莱本索尔约定叫之一

原创：bridge_commentator(一叶知秋)

  大家都知道，桥牌中的防守是最难的。将防守比喻为两个在黑暗中摸索的人想找到一个唯一的出口并不为过，在防守过程中，无法通过声音、灯火来提醒，只能靠临场判断，这个时候，防守叫牌中一些有效的约定（不是密约！），可以对同伴的防守思路产生引导作用。可惜，任何约定叫都要求同伴以及对手有一个心里接受的限度。我个人认为，在网络桥牌上，不可能使用自己研究的叫牌体系，因为没有那么多的时间去用于对每一个特约进行解释（有些人为叫牌体系过于复杂，可能连使用者本身都不一定说得清楚，很容易给对手造成一个隐瞒和欺骗的现象，久而久之，这对牌手也会招人牙垢）。但是，桥牌的发展和钻研离不开约定叫，高水平的牌手更需要约定叫。选择一些众所周知的约定叫，经过消化后充实到自己的叫牌尤其是防守叫牌中，不失为一种比较好的办法。首先它不影响整个叫牌体系的根基（总体上还是应该精确/自然）,其次这些约定叫的使用频率应该不会太高，解释的次数较少，这样容易为他人接受。莱本索尔约定叫及其变化就是一个非常好的约定叫。

（一）莱本索尔约定叫
  　在同伴开叫1NT后，如果对方争叫二阶花色，开叫方一般会遇到一系列问题：
            （1）加倍是否惩罚性？
            （2）新花是否逼叫？
            （3）敌花止张如何表明？ 
            （4）四张高套如何表明？
    在诸多约定性对策中，莱本索尔约定叫的处理方案较好地解决了上述难点，现在莱本索尔约定叫已经是无将类中除了斯台曼问叫和雅克比转移叫之外最常用的约定叫了。
    本人推崇莱本索尔约定叫，盖因其中有一个令人难忘的非自然约定性叫品――2NT。通过这个叫品的设计，应叫人可以表达出大约10多种的牌情。应叫人除了让开叫人作出一个3C“傀儡叫”后通过其再应叫来区分应叫人持牌强弱程度之外，还有一个“止张表明”和“高花表明”的特色值得一提，见下表：
            西        北         东        南       敌花止张和另外一套高花的情况
            －－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－
      (1)    /        1NT        2H        2NT      
             /        3C         /         3NT      有H止张，否定4张S 
       
      (2)    /        1NT        2H        3NT      否定有H止张，否定4张S
       
      (3)    /        1NT        2H        2NT      
             /        3C         /         3H       扣叫表示4张S且H有止张
       
      (4)    /        1NT        2H        3H       H没有止张，但有4张S
       
    莱本索尔约定叫是如此的令人爱不释手，如果仅用于NT类开叫是不是有点可惜？我不自量力将它嫁接到叫牌的其他方面中尤其是防守叫牌的一些环节中，供大家参考，就称之为【非典型型莱本索尔约定叫】吧。

（二）非典型莱本索尔约定叫之【关键张】加叫

　　我认为，莱本索尔约定叫的精华是2NT这个叫品，虽然这个2NT已经失去了它自然的含义，而且在选择使用莱本索尔约定叫后，并不是每一个叫牌过程都会出现2NT这个叫品，但是却因为它的存在，搭档之间还是可以通过“你用”、“你可用而不用”等现象来判断整个叫牌的意思。所以2NT这个叫品可以算是莱本索尔约定叫的【魂】。
　　所谓非典型莱本索尔关键张约定叫就是借鉴莱本索尔约定叫中对于止张报明的设计原则，注重于防守叫牌过程中对于争叫花色大牌的情况描述，从而达到指明防守路线的目的。

1.  非典型莱本索尔【关键张】加叫的理论
    这是省级桥牌锦标赛上的实战牌例。
      南        西        北        东  
      1S        2D        2S        3D（*）
      /         3NT       /         //

    首攻黑桃之后，东西方获取了9墩牌。东西家持牌是这样的：
               西              东   
      　　　　S K4           S 872
      　　　　H 1085         H Q743
      　　　　D KQ9873       D A104
      　　　　C A5           C K103
         
    东西方当时就是采取了一种约定叫（3D是提醒了的――表示有大牌支持的加叫）。在赛后的分析中，大家都认可了这种关键张加叫在西家决定叫3NT时所起的作用，因为如果东家有A支持，西家基本上看到了8墩牌（在黑桃首攻的前提下），叫3NT就是合理的；否则，即使同伴有四张方片支持，要是没有A，方片必须脱手，那么3NT就基本上是不容易成的，甚至可能遭到加倍输掉大分。可以说，有了这样的约定叫，才给了西家敢叫的胆量和打成的把握。我在这里要向大家推介的关键张莱本索尔约定叫就是：
       
    在防守叫牌过程中，当你想加叫同伴的花色时（如果担心暴露牌情而不作加叫令当别论），首先要考虑的是你的叫品是否对同伴之间可能的防守有用。采取关键张莱本索尔约定性加叫的目的就是让同伴了解你在争叫花色中的关键大牌情况，以利于他作出正确的决定。那么如何表明这个大牌情况呢：还是用这个叫牌过程为例：

      南        西        北         东  
      －－－－－－－－－－－－－－－－－
      1S        2D        2S      　 ？

    如果东家想加叫方片，应当采取：
         3D（*）＝ 有D关键张（A/K）的支持加叫
         2NT（*）＝ 要求同伴先要叫出3C这个傀儡叫，如果下轮东家叫回3D，则表明这个加叫是没有关键大牌的加叫。
    这就是非典型的莱本索尔约定叫的一种临床应用，它最大的用途是有助于你们的防守。例如你是西家持牌：
         54
         K85
         Q73
         AQ982
      南       西        北        东  
      1S       2C        2S        3C（*）
      4S       /         /         //
         
    作为首攻人，面临着是否攻梅花的犹豫。不过如果知道同伴有梅花关键大牌（CA/CK），你当然可以放心出梅花了，要是希望同伴第二轮以后进手，你可以先提A，然后让同伴K上手；如果你希望同伴第一轮就进手换攻花色，也可以先出其他牌张，甚至可以要花色呢。

      a)  54                                  
       　 985                                                
       　 AQ7                                              
       　 AQ982 

      b)  54
          AQ7                                          
          853
          AQ982
    你持a)牌可以选择首攻C2来隐含要求回攻方片。你持b)牌可以选择首攻C9来隐含要求回攻红心。 反之，如果叫牌过程是：
      南        西        北        东  
      1S        2C        2S       2NT（*）
      4S    　  /         /        //

         
    东家支持了梅花，但是通过这个示量加叫表示了一个非常关键的信息：就是东家没有梅花A/K之一，这样，西家当然不会去首攻梅花了，只能等待同伴穿越暗手可能的梅花K了，这也是通过防守叫牌指导首攻的一种范例。

2.  高阶非典型莱本索尔【关键张】加叫
 　 掌握了第一种非典型莱本索尔约定叫的原理后，举一反三，在更高的高度上同样可以随心所欲地使用这种约定：
      南        西         北        东  
      1S        2C         4S        5C（*）表示有梅花关键大牌（CA/CK）的牺牲叫

　　跟这个进程的不同点就很清楚了：
      南        西         北        东  
      1S        2C         4S        4NT（*）表示无梅花关键大牌（CA/CK）的牺牲叫

    这是实战中出现的一个牌例：
      南        西         北        东  
      1S        2C         4S        5C（*）
      5S        /       　 /         //
      四家牌为：
                    KJ87
                    AQ952
                    J96
                    3 
          64　　　　　　　　　　 10
          74　　　　　　　　　　 1086
          AQ5　　　　　　　　 　 108743
          AQ8762　　　　　　　　 K10943
                    AQ9532
                    KJ3
                    K2
                    J5
         
    要是没有东家的5，西家可能会选择高花首攻，将会贻误战机。在使用非典型莱本索尔约定叫的叫牌过程中，这个5是表示梅花顶张大牌的叫法，西家自然可以放心低出小梅花让东家进手来穿越暗手可能的方片软肋。

3.  低阶非典型莱本索尔【关键张】加叫
    有时甚至可以在低阶（一阶）叫牌上使用这种约定(但必须是老搭档而且提醒对手)：
      南        西        北        东  
      －－－－－－－－－－－－－－－－
      1C（JQ）  1S        X         1NT/2NT/2S/3S
         
    这里解释一下：在明确对手的实力在自己方之上后，东西的叫牌的重点在于剥夺对手的叫牌空间，阻止对手在低阶上交换信息，以及为自己方找到一个可能最好的防守路线。
         (1) 2S/3S ＝ 属于正常的阻击加叫，但是保证黑桃顶张大牌支持，花色支持保证3/4/4张以上。
         (2) 1NT   ＝ 可以加叫到2S的牌，但是黑桃花色应该没有顶张支持，可能只有三张支持。
         (3) 2NT   ＝ 可以加叫到3S的牌，但是黑桃花色应该没有顶张支持，至少四张支持。

    对于阻击叫的加叫也可以采用这个方法：
      南        西        北        东  
      1C        2S        X        2NT/3S/4S
    同理东家的叫品：
         (1) 3S/4S ＝ 属于正常的阻击加叫，应该保证黑桃顶张大牌支持，花色支持保证3张以上。
         (2) 2NT   ＝ 本想加叫到3S的牌，但是黑桃花色没有顶张支持，所以用2NT来过渡。

    说明：低阶非典型莱本索尔约定叫与前面几种不是完全一样，原因在于前几种局面基本上已经知道了对手的将牌花色是什么，首攻人是谁了，所以使用非典型莱本索尔关键张加叫来给首攻人以信息。而在低阶情况下，对手很可能尚未定将，你们两个人谁是首攻人也不能肯定，所以叫牌的不确定性非常大。无论如何人，争叫人的同伴通过非典型莱本索尔关键张加叫的约定，可以为同伴的叫牌或防守判断作出一些有益帮助，也是值得费点脑筋的。
         
    本篇的结束语：事实上，非典型莱本索尔约定叫的用途，不仅仅是在做【关键张加叫】这种情况下使用，当联手日趋默契后，你们还可以采用更多的非典型莱本索尔约定叫。

　

非典型莱本索尔约定叫系列之二

－－【强弱区别性莱本索尔】约定叫

改编：bridge_commentator（一叶知秋）

   【强弱区别性莱本索尔】约定叫是一种完全实战型的约定叫，主要是为了解决敌方弱二阻击开叫之后，联手在防守叫牌过程中可能出现的牌力强弱不易区分的难点。该部分的内容原创于约翰﹒布鲁博格，但有修补。 

    我们常常会面对“对方弱二开叫，同伴技术性加倍后”的局面：

      南        同伴        北        你 
      ------------------------------------- 
      2S         X          /         3H    （这个3H的牌点上下限是多少？）
      /         ？？？                         
  
    在传统的防守叫牌进程中，对于你的3H，同伴只好猜测你的牌（0－11点），它可能是下面几个例子之一（实际情况更多）： 

      a)    542      
            9873                      
            Q54                        
            J54  

      b)    42                                  
            K9873                           
            Q54                        
            J54           

      c)    Q4                                 
            KQ73                     
            9542                       
            A105                              

    a)例是如此的弱（11个输墩），估计什么定约也不成，可是你不叫3H又能如何呢，只能祈祷同伴不要贸然上4H 。
    b)例红心不错，虽然牌点较少，如果同伴有4张红心，即使是低限，完成3H也有希望，从输墩计算，有9个之多，联手想成局，要求同伴要有牌型。
    c)按照防守叫牌导论的说法，c)例的牌点足够直接跳叫4H，但效果也许不如联手打3NT（因为同伴可能是加倍再出套的牌），所以有些PD还是会选择叫3H，但不能承受加倍人放过之痛。

    这里的3H，的确会给加倍人的判断上增添困扰。我想推荐的就是一种非典型莱本索尔形式，它的要点是：同伴加倍人对手的弱二阻击叫后，你： 

      1.  在三阶上直接叫出新花色为进局邀叫，通常持有8－11个大牌点。 
      2.  直接扣叫敌花  ＝ 建立一个进局逼叫的形势，牌情不明 
      3.  直接3NT  ＝ 均型牌至少有敌花止张 
      4.  跳叫某个花色进局＝ 关煞叫 
      5.  2NT＝这里称之为【非典型莱本索尔强弱约定叫】。要求同伴叫出3C这个“傀儡叫品”（除非加倍人是加倍出套的好牌可以不遵守这个约定），之后你再作牌力的强弱区别。 

      南        同伴        北        你           
      --------------------------------------
      2S         X          /         2NT 
      /          3C         /         ? 

      你（东家）：
          (1) PASS   ＝ 就是梅花弱套牌，0－7点 
          (2) 3D/3H  ＝ 方片/红心弱套牌，0－7点
          (3) 3S/3NT ＝ 一般11点以上 

    我将这些相关的内容加以补充，整理成以下两个表格。
    表1是围绕另外一门高花套中的强弱区别叫的叫牌序列: 

          敌        同伴        敌        你     高花情况的描述 
      -----------------------------------------|----------------------------
          2S         x          /         2NT 
          /          3C         /         3H  = 最弱的叫法，止叫（0－7点） 
       
          2S         x          /         3H  =  8－11点，四张以上套的邀请叫。    
       
          2S         x          /         2NT 
          /          3C         /         4H  = 牌力在11点左右。
                                                为防同伴是加倍出套，先用2NT确认， 
                                                否定后直接选定红心封局，一般5+H.  

          2S         x          /         4H  =  红心6+张，其他定约没有兴趣，止叫.

    表2是针对敌花止张情况的非典型莱本索尔: 

          敌方       同伴      敌方       你     敌花止张和另一高花张数的间接描述 
      -----------------------------------------|------------------------------
          2S          X         /         2NT    
          /           3C        /         3NT    有S止张无四张H。想确认同伴牌力
                                                 是否加倍出套后再决定最后定约。
       
          2S          X         /         3NT    敌花止张好，无4+H,不关心其他. 

          2S          X         /         2NT 
          /           3C        /         3S     有S止张，H四张，牌力比直接3H强，
                                                 进局逼叫.
        
          2S          X         /         3S     没S止张，H张数不明
                                                （但可以承受同伴在敌花无止时叫新花
                                                 或者4H,否则就是有别的长套），
                                                 进局逼叫，牌力比直接封局强。 
       
    在非典型莱本索尔强弱区别叫中，针对2H阻击叫有特殊之处需要提及： 
          南         同伴       北        你 
      ------------------------------------------
          2H          X         /         ? 

    你的可能叫品注释：
         (1) 如果你直接应叫是2S，是示弱，如d)例
         (2) 直接应叫3S，是进局逼叫，类似于同伴开叫精确1NT之后你直接跳叫3S，如e)例 
         (3) 先应叫2NT要求加倍人叫3C之后，你再叫3S则是邀请叫，如f)例。
