对加倍一阶高花的约定性1NT应叫（MDN）


作者：张德生

    西    北    东    南
    1S    X     －    ？

    这是一种最常见的叫牌进程，如果你是南家，是否考虑过持下面三手牌时该怎样应叫呢？

（a）S J84  H AJ83 D K1062 C Q3
（b）S 953  H 103  D K984  C Q1062
（c）S 1073 H K107 D A93   C A862

牌（a）：无疑，你只能在2H和3H中作出选择。2H实在委屈，当同伴持：S 82 H K1094 D AJ7 C AJ82，你将错过一个机会不错的4H定约。但是，如果你跳叫3H，由于同伴无法确定你这个3H跳叫究竟是5张H还是4张H，当他仅有3张H支持时会无所适从。如：S 92 H K106 D AQ74 C AJ94；在你持5张H时，他会愉快地加叫4H；而当你持4张H时，3H也许都不安全了。

牌（b）：选叫哪门低花呢？从理论上讲，持6～8点时叫级别较高的那门4张套，但同伴可能持2434型；选叫2C吧，同伴又可能是2443型。总之，你只有50％的机会猜对。

牌（c）：虽然大牌实力可以邀请，但以4张弱草花跳叫实在太可笑了；黑桃没有档张又不能跳叫2NT；2C又是一个极度的低叫。你会作何选择呢？大概谁也没有办法。

    上面这三种持牌类型，使用标准的叫牌方法，几乎是没有可能解决的。我曾经在美国的桥牌杂志《Bridge World》中，“大师解答”的专栏中见过类似（a）的问题，专家们的看法各不相同，有的叫2H，有的叫3H，还有的叫1NT（表示实力）。至于（b）和（c）这两种情况，连题目也没有见过，也许他们觉得不可能有正确的解决途径，因而不拿出来供专家讨论吧。

    那么，就真的没有科学的解决途径了么？大约五、六年前，我偶然想到，如果使用人为的1NT应叫，不就可以解决上面三种按标准处理方式无法解决的难题吗？按照这个思路，经过一段时间的研究，我设计出一套简单实用的约定叫，请看：

    西    北    东    南
    1S    X     －    ？

2C/2D/2H：0－8或差的9p，自然叫，和标准的处理方式完全一样；
2S：人为的进局逼叫，任何牌型；
2NT：自然叫，邀请3NT，并保证黑桃有两档张；
3C/3D/3H：自然叫，邀请实力，保证5张以上；
1NT：人为约定叫，包括了前面三种不好处理的持牌类型，和其他几种特殊的持牌，高限为邀请实力。加倍者首先假设同伴是要求他示选一门低花。在开叫者的低花示选后，应叫人持（b）牌Pass，所有的任何其他再叫都显示邀请实力。

    西    北    东    南
    1S    X     －    1NT
    －    2C/2D －    ？

Pass：弱牌，（b）牌类型；
2H：4张H，邀请实力，通常10－11p；
2S：（c）牌类型，邀请实力，但没有4张H，也没有5张低花，S上也没有档张；
2NT：自然叫，邀请3NT，S上仅有一个档张，也没有4张H；
3C/3D：所叫低花5张＋4张H，邀请实力（在加倍者示选2C时，你只需再叫2D就可以描述这类持牌）。当你同伴是持3张H作的技术性加倍时，你很容易就可以找到合适的定约了。
3H：5张H加5张低花，邀请实力。


    一个好的约定叫，必须符合下面三个条件：

（1）实用，确实能解决用其他方法无法解决的问题，并且针对的情况出现机会不能很小；
（2）记忆容易，逻辑性强；
（3）该约定叫所占用的那个叫品（这里是1NT），如果不能取消，就得用其他叫品替代；如果无法用其他叫品替代，你就比较二者之间的优劣，得多失少时使用该约定叫，反之就放弃。

    让我们根据上面的三大原则来作一检查。无疑，本约定叫是及其使用的，它能解决标准方法无法解决的问题，而且解决的问题出现机会还很大。至于记忆负担方面，大概无需去专门记忆。最后，看来你们无法停在1NT上了。但二阶低花定约也不一定劣于1NT。这样损失1NT的自然含义，危害就很轻了。比起上面三种难处理的局势，绝对是得多失少。

    第二个问题，如果加倍者是加倍出套或者加倍再叫2NT的强牌该怎么办呢？很容易，在应叫方的1NT后，你只需再叫2H/2NT/3C/3D即可显示一手强牌，但这并不是逼叫的。

    最后，在对方开叫1H，你同伴作技术性加倍，第三家pass后，同伴间仍然可以使用本约定叫，根本无须作任何变动。

    如果第三家在同伴作技术性加倍后叫再加倍，则关闭上面的约定叫，第四家的所有再叫完全按照常规处理即可。如：跳叫新花色是阻击叫，1NT要求同伴示选低花等等。

    为了便于记忆，并与内容一致，我们不妨将这个约定叫命名为“MDN”，即高花、加倍、无将。
