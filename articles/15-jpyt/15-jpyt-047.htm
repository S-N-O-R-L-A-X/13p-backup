重开叫1NT


——劳伦斯

    大多数情况下，多数牌手对重开叫1NT处理的比较好。不过，我见过不少实战的牌局，其中叫1NT的人所想显示的牌力范围与他的同伴所期望的大相径庭。好在这一点是容易纠正的。
    1NT应该显示多少点？假如你问十位好牌手，你可能发现其意见的一致性少于你所预期的，假如你去问十位经验较少的牌手，其分歧可能还要大得多。我所推荐的牌力范围是我的大多数搭档的共识，你可以加以调整使之与你的设想相适应。
    要是你觉得我既说意见的一致性很差，同时又说多数人能处理好1NT重开叫有些自相矛盾的话，请听我解释。我想多数搭档间的一方对叫牌的同伴的牌有合理的估计，但我认为他们用的牌点范围不佳。假如你约定1NT显示16－18点，当这种情况出现时是挺好，但当你拿着远为更常见的12－13点牌时就有问题了。我打算显示的牌点范围既安全又有效，与某些其他范围相比它有用武之地的机会要多得多。
    
    我建议你的最低界限大约是非常好的11点。
    你的上限是若在1C或1D后重开叫1NT大约为14点，但在1H或1S之后大约为16点。
    （即：1C/1D后，重开叫1NT为：12－14；1H/1S后，为：12－16）

    这一区别的理由是，如果你拿着15－16点在对方1S之后不叫1NT，而代之以加倍，同伴大概会叫二阶某花色。这时，为了真实表示你的牌点，你将不得不叫2NT，而如果同伴的牌力较差，你已经毫无特殊原因的叫的过高了。持有15－16点时稍许保守一点，叫1NT会好的多。丢局？别担心。你将在许多副这样的牌中免遭灾难，拿这种牌你的极限定约就是1NT。同时请注意，当左手敌方开叫1H或1S时，估计那是他的五张以上套，这可能为防守方提供了有效的首攻。1C或1D并不保证这样的牌情。

    所以我推荐的重开叫无将阶梯是：
    1C/1D -  -  ?
                12-14: 1NT;
                15-18: 加倍后平叫无将；
                19-20: 2NT;
    1H/1S -  -  ?
                12-16: 1NT;
                17-18: 加倍后平叫无将；
                19-20: 2NT;

牌例：
(1)
1C pass pass ?
S K8 H 1065 D KJ87 C AJ54
低限的1NT。加倍不好，因为你将在1S应叫后十分尴尬的不叫，而加倍之后再叫1NT所显示的又超出你的牌力。

(2)
1C pass pass ?
S K76 H AQ2 D KJ4 C QJ83
加倍，下次叫1NT。这种叫法并不显示你直接位置加倍再叫1NT的同等牌例，而是16－18。

(3)
1C pass pass ?
S K2 H AQ D QJ1087 C AK106
2NT。你还是不希望在加倍后听见同伴叫黑桃或红桃。

(4)
1D pass pass ?
S 42 H KJ7 D QJ987 C AJ3
Pass。你有叫1NT的牌点，但不叫是最好的，尤其是对手有局的时候。

(5)
1H pass pass ?
S J3 H Q654 D AQ10 C K542
1NT。你真的希望有张H9或者H10，但这是可以接受的低限。

(6)
1H pass pass ?
S K4 H 1062 D AJ54 C AQ63
不好办。只有两张S，加倍不好，不叫又太保守。我会试叫1NT，同时很进张。请注意，无止张时叫1NT要求有高限的大牌点以证明其正当性，同时，你在敌方花色的牌决不能少于三张。如果1NT被放打，应该没有大问题，即使他们的红桃套畅通无阻。如果遭到加倍，你大概应该有第二方案，准备逃叫。你的再加倍是技术性的，显示低花，大致正如本例中你持有的牌。

(7)
1H pass pass ?
S K76 H QJ63 D A54 C KQ7
1NT。好牌，但不值得先叫加倍。同伴很可能叫2C或2D，你将或者不叫并有可能丢局，或者试叫2NT而打宕。此时最佳叫品是保守性的1NT。

(8)
1H pass pass ?
S AQ2 H 1064 D AK107 C AK4
加倍。1NT、2NT必须保证有止张，因此加倍是你最好的选择。在同伴应叫后，如果你还想试探成局，你可以扣叫。

(9)
1S pass pass ?
S KQ9 H 765 D AK86 C A42
1NT。对手的1H或1S占去很多空间，重开叫比1C或1D之后更难叫的准确。你应该尽可能力争以重开叫参与叫牌，但是一旦决定重开叫，没有必要过分进取。

(10)
1S pass pass ?
S Q87 H A2 D AKJ107 C AJ7
2NT。加倍之后叫2NT也是可能的处理方法，但你不希望同伴叫红桃又再叫红桃，或者可能叫过红桃又叫草花。你也不想在加倍和同伴应叫2H后被迫叫3D。2NT不仅恰当的描述了你的牌，而且此后如果同伴坚持叫红桃，你的感觉不会那么差。
