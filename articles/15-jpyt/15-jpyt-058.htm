干扰性2NT


干扰性2NT约定叫之所以能够使用，是基于如下坚实的基本原则：

在敌方作出一个花色叫牌（开叫或者争叫），而紧接着后又有一个简单加叫之后，无论我方是希望参与竞争性叫牌，或者是希望作平衡叫牌，看来，对我们来说，2NT都不可能是一个很好的定约。

因此，我们就有可能把2NT作为一个空闲叫品，即在敌方已叫和加叫一门高花之后，把它处理成技术性或干扰性的叫牌。

为什么要限定敌方已叫和加叫花色仅为高花呢？

如果敌方已叫和加叫一门低花之后，我们完全可以在2阶水平叫出我们的花色（除非敌方所叫花色是方块，而我们希望叫出草花）。通常，我们是没有必要被推进到3阶水平上去叫牌的。但是，在敌方2阶水平的加叫之后，除了敌方所叫花色是H，我们可以叫出2S的情况之外，无论如何我们都要被迫进到3阶水平上去叫牌。

为了使我们更好的理解和使用干扰性2NT，我们首先来讨论、限定一下，在同伴作出加倍以后，你使用2NT的含义：

－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－－

在敌方已经作出了2阶水平的高花加叫之后，同伴作出了加倍（技术性加倍、惩罚性加倍、应叫性加倍、否定性加倍、平衡性加倍、合作性加倍等等），你叫出的2NT叫牌是约定性和逼叫性的，作出干扰性2NT叫牌的目的，是开始寻求、定位最好的和最有效的将牌花色。

东家要持有什么样一手牌才能叫出干扰性2NT呢？

只要东家感到这一手牌不知道该作什么定约（花色的而非无将的）为好时，他都可以叫出干扰性2NT。他可以持有都将可能成为将牌的三色套牌，也可以持有某种类型的两色套牌，甚至是一手极端平均型的牌！

是否在以上叫牌过程之后，东家还可以在3阶水平上叫出自己的花色来呢？

当然可以！前面已经解释过，我们已经限定了东家在不能确定将牌花色选择时，才叫干扰性2NT。因此，在同伴对敌方2阶水平的高花加叫作出加倍之后，东家可以选定自己希望的长套花色作为将牌。

下面再给出一些示例，说明东家可能持有的在3阶水平上叫出自己长套花色的一手牌。

在同伴对敌方2阶高花加叫加倍之后，东家的这种3阶水平花色“自动”答叫，通常是基于持有5张套；但是，如果是持有4张强套，也可以不保证5张套－－我们在上面第一例中已遇到了。在东家持如下一手牌是，除非他冒险Pass，否则他总是应该叫出3C来的：

这种情况下，甚至连4张C好套也未保证。

作出干扰性2NT叫牌，是否可能具有“特殊无将争叫”的结构呢？

确实可能。但是，两者之间也有重大的区别。作出特殊无将争叫的一家是基于持有两套牌，而主动使用2NT争叫的。而作出干扰性2NT叫牌的一家，是被同伴对敌方2阶高花加叫的加倍所驱使，被动作出干扰性2NT叫牌的，并且他几乎不太可能持有两套牌。

使用干扰性2NT这种约定叫方式有一个明显的长处，就是可以使同伴在敌方1阶高花叫牌之后，一开始就避免作出不利的将牌花色猜测，而能够更多的使用技术性加倍。而在此后的敌方2阶高花加叫之后，再使用否定性加倍，或应叫性加倍，甚至特殊无将争叫。

干扰性2NT约定叫，仅仅是试图继续发扬光大我们叫牌理论中的“不要随便去猜测叫牌”的优良传统而已。

对干扰性2NT的答叫（Responding to a Scrambling 2NT）

如果敌方2阶加叫花色是H，事情就相当简单－－作出干扰性2NT的一家是在向同伴提供低花选择，而且同伴必须在两门低花中作一选择（通常，两套低花长度几乎相等）。因此，同伴所需要作的事情就是：选择手上较长的低花。

例如，在下面的叫牌过程之后：
  RHO   你   LHO   PD
  1H    －   2H    －
  －    X    －    2NT
  －    ？

你持有：
  S J643 H 82 D K75 C AQ94
叫：3C

  S AJ7 H 9 D J7432 C K642
叫：3D

  S J743 H A D QJ109 C J643
叫：3D
我们看到，在你的两低花套长度相等时，你可以选择较强的低花套作为将牌－－“最便宜叫牌原则（Up the Line）”在这种情况下不适用。

然而，当敌方2阶加叫高花时S时，“最便宜叫牌原则”就成为一个重要因素了。在这种情况下，作出干扰性2NT的一家可能持有三套牌，或两套牌，神智极端平均型牌。因此，同伴通常采用的策略是，应当答叫出最低级别花色的至少4张套来。

例如，在下面的叫牌过程之后：
  RHO   你   LHO   PD
  1S    －   2S    －
  －    X    －    2NT
  －    ？

你持有：
  S 6 H AQ92 D 10865 C QJ83
叫3C。

你持有：
  S 8 H AQ10 D 97643 C K853
也需要叫3C（最便宜叫牌原则！）这手牌很容易受到叫3D的诱惑，但同伴可能持下面的牌叫的2NT：
  S A743 H 8643 D A C 10762
他可以选择3C或3H。

下面的例子，给出了作出加倍的一家和作出干扰性2NT叫牌的一家两手牌。

（1）
南   西   北   东
1S   X    2S   －
－   X    －   2NT
－   3D

  S 9        S 7632
  H AJ62     H 854
  D A8654    D 972
  C AK5      C QJ10

如果不使用干扰性2NT方式，东便没有可能在同伴第二次加倍之后作出合适的叫牌。只能猜测性的叫3C吧，因为这是他最强的，也是最低级别的花色套。而现在，西却可能寻求到较好的3D定约。

（2）
南   西   北   东
1S   －   2S   －
－   X    －   2NT
－   3C   －   3D
－   3H

  S 87        S 953
  H AJ105     H 8643
  D K94       D AQ63
  C Q643      C A10

东的叫牌显示了红色两套，西当然选择3H。很多牌手在持东这手牌时，在西作出平衡性加倍之后，喜欢简单叫出3H，希望西持有4张H。但是，如果西在仅持有3张H和他这一手牌的点力时，他仍然可以用加倍作出平衡叫牌，你们也可能寻求到很好的D4－4配合机会。而现在，当东作出干扰性2NT叫牌后，双方正确的找到了H配合。

我们新设计的这个约定叫，在处理敌方作出了2阶高花叫牌，同伴又作出允许罚放为惩罚性的加倍之后的叫牌，同样也是很方便灵活的。

你作为西家，持有下面两例中的一手牌，在下面两个叫牌过程之后，难道还有必干扰性2NT更有效的技术性处理措施么？

（5）、（6）这两手牌西家罚放是相当危险的，敌方至少8张将牌打2阶被加倍的定约，对你方不是什么好事情。

我甚至于允许，在同伴未对敌方某家的2阶高花叫牌（或敌方牌手间的加叫）作出加倍的某些叫牌过程，也可以使用干扰性2NT约定叫。例如，在下面的几个例子里，你可以把西家的2NT叫品处理成干扰性2NT。

（7）

     －   －   2H
－   －   2S   －
－   2NT
西持有：

（8）
南   西   北   东
          1H   -
2H   －   －   2S
－   2NT
西持有：
S Q H 73 D A9754 C K8743
既然东未在第一轮对敌方牌手的1H开叫作1S争叫，看来作S定约并不理想。因此，西作出干扰性2NT叫牌，就是要求同伴从两低花套中选择一门较长花色。

（9）
南   西   北   东
1S   －   2S   －
－   2NT
西持有：
S 874 H KQJ6 D 9 C AJ743
由于西作平衡位置，所以西的2NT必然是干扰性的，显示持两套牌，但是，并不是显示两套低花，而是H和一套低花，如果西作出平衡性加倍，那么在东持如下一手牌时，就会导致灾难：
  S K5 H 74 D Q87432 C K106
而现在，东就可以在持如上一手牌时选择3C。

（10）

2H   －   －   2S
－   2NT

西持有：
S 8 H J97 D AJ106 C 85432

西在此时采用干扰性2NT比叫3C灵活得多。西在此时也绝对不需要把它作为自然叫，因为他只要持一手稍微“像样的”平均型牌，完全可以作出合作性加倍。

（11）
南   西   北   东
               1D
1S   X*   2S   －
－   2NT
X*：否定性加倍
西持有：
S 8 H AQ74 D K103 C 98654

没有理由让敌方舒舒服服的作2S定约，而且西需要持有另外的S好牌才敢于作第二次负加倍（万一同伴选择Pass，你要承受的起！）。因此，为什么不在3D或3C定约试一试运气呢？西的干扰性2NT叫牌为东提供了这一机会。

下面我们这个有关2NT叫牌的最后一个例子，可以把它归结于我们的干扰性2NT约定叫一类，在专家牌手的圈子里，它已是被作为特殊无将争叫类型处理的。

（12）
南   西   北   东
1S   2C   2S   －
－   2NT
西持有：
S 9 H AKJ6 D 85 C AQ8742
或：
S 9 H 85 D AKJ6 C AQ8742

如果同伴准备选择红色套，他可以直接叫出3D或3H。否则，就叫出西的C。

总之，干扰性2NT只要求很少的记忆。但是，它却对我们在选择将牌花色时避免不必要的猜测大有帮助。我们仅仅失去了把2NT作为自然叫的机会，前面我们已说过，在这种情况下，付出的这个代价极小，也很值得。它是把2NT叫品处理为更好的援助性叫牌约定叫的有一个例子。
