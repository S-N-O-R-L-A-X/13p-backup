加倍之后的叫牌

假设叫牌进程是：

西   北   东   南
          1D   X
－   1S   －   ？

你拿下面的牌会如何处理？
S AQ6    S KJ9    S AK
H AKJ7   H AKQ5   H AQJ9
D 86     D 876    D 973
C AK75   C AKJ    C AKJ6

因为缺少方块档张，你不能再叫无将（即使无将是你方最好的定约，也应让持Kxx或Qxx的同伴坐庄）；没有4张黑桃支持你也不能加叫同伴的黑桃到3、4阶。多数牌手在没有更好叫品的情况下，会扣叫2D，这也是我推荐的。但接下来的问题就复杂了，北拿下面的牌应该叫什么呢？

    S 8743
    H 864
    D Q5
    C 9872

他应该用重叫黑桃的方式示弱么？拿下面这手牌，他应该跳叫3S么？

    S K8742
    H 87
    D 876
    C Q94

扣叫人保证再叫么？这个扣叫逼叫到哪里？问题很多。

下面的处理方法不是我发明的，但我推荐使用它：扣叫保证至少有好的20点，暗示一手不适合叫无将和强加叫的牌。扣叫者的同伴叫最便宜的叫品（本例中为2H）表示5点以下牌力（在对方花色上的J和无护张的Q不应计入有效点力，持好牌型时要向上调整）。在这个完全是约定性的最便宜叫品之后，扣叫者必须再叫但不再逼叫；扣叫者的同伴除最便宜叫品之外的叫牌都是进局逼叫。

  本节之初的叫牌发展到：
西   北   东   南
          1D   X
－   1S   －   2D*
－   2H*  －   ？

我们给出的南家的3手牌都应再叫3H。是的，这个定约可能有点高了，但总比对着同伴一手白皮叫到4H要好。北家可以再叫，他会拿这手牌叫4H：

    S 8654
    H Q93
    D 87
    C Q543

或者拿这手牌叫3S：

    S 987542
    H 75
    D Q8
    C J54

在下面这个进程中，你们也可以停在2阶上：

西   北   东   南
          1C   X
－   1S   －   2C*
－   2D*  －   2H
－   ？

北拿这手牌将Pass 2H：
    S J754
    H J82
    D 9762
    C 84

持下面的牌则再叫2S，然后南可以Pass：
    S 108754
    H 82
    D Q762
    C 95

如果叫牌进程是：

西   北   东   南
          1H   X
－   1S   －   2H*
－   ？

北拿这手牌将再叫3S：
    S Q10742
    H A8
    D 762
    C 1084

或拿这手牌再叫3D：
    S Q842
    H 754
    D K1054
    C 92

显然，即使使用这套处理方法，你还是会遇到一些困难，但有约定总比没约定好。至今为止我还没有发现比这套方法更好的处理方法。
