作者：一叶知秋

第五回  和稀泥裁判受累   择利弊加倍无将

　
 　　几通战鼓之后，等级赛进入中盘，各组的情况还是馄饨一碗 ---- 看不清，令预测家和评论家纷纷跌破眼镜。究其原因，乃是战况激烈，水平接近；强队起伏，弱旅奋蹄。牌评家的一致看法还是看好那些印象中的强队，其实桥牌之妙，在于强弱之势常常易位，只要兢兢业业，弱队翻身只在一念之中。
　　点点将前三轮的资料打包，发回西牛贺洲。并附去一封信给观音菩萨：

观音菩萨：
　　您好！
　　在网上搞桥牌比赛，真累。
　　人间的桥牌比赛好复杂呀，我觉得跟神仙间的其他游戏很不一样。首先是这些牌手都很计较胜负，为了一个毫无意义的问题，就可以纠缠多时，甚至当场翻脸，将往日的情谊一笔勾销，令人寒心。
　　其次是有些队很不在乎比赛的机会。平时一直叫唤着让小五赶趟儿组织比赛，临了却弃权或在成绩不佳之后退出，给比赛的组织者增添了很多麻烦，这也是导致小组中各队成绩和名次波动很大的原因之一。
　　人间的INTERNNET，看来也有很大的缺陷，经常搞得牌手出牌室了就回不来落座，为此引起的纠纷（每一轮都有几次）。

　　当然，最累的是还是那些裁判和组织者们。

　　当牌手不熟悉规则时，他们要声嘶力竭解释；
　　当牌手拿家伙打架时，他们要身挡其间劝说；
　　当牌手拍屁股走人了，他们要打扫牌室卫生；
　　当牌手都进入梦乡了，他们要写评论发成绩。

　　而他们都是无偿、自愿的。我想，他们都是热心的人，桥牌离不开他们。
　　他们的名字也很好玩的呀：什么VIP、ESCAPEOD、赶趟儿、艾哈布、……更有意思的是【和稀泥】，多么形象哦，我建议他作为桥牌界的形象代表出席宇宙大会，风头一定能盖过足球裁判光头科里纳和中国猪球的形象代表米驴。裁判要是都学会了和稀泥，球场上、牌桌上应该就不会有打架斗殴的事情了吧！观音菩萨能否大发慈悲将这些热心裁判全部载入仙册呢？

　　很多人坐南家拿到类似的牌，都会遇到一个难题：是争叫1NT还是作技术性加倍（因为有四张黑桃）？

    说实话这里并没有标准的模式可以指导读者必须要加倍或者必须要争叫1NT。但是点点觉得还是可以先比较两个叫品的优劣：

（1）加倍：好处是可以表明四张黑桃（低限时加倍应该保证有另外一个高花4张哦）；缺点是无论是同伴在二阶应叫一个花色后（对方未叫）还是对方抬高一个水平后（同伴未叫），你都无合适再叫叫品。 
[N]   [E]   [S] [W]  
1H        X       /
    　　 2C        /        ？ 
        同伴做出应叫后，你确实很难再叫。你再叫2NT呢还是别的，都是严重歪曲持牌情况的。

[N]   [E]   [S] [W]  
1H        X       2H
    　　  /        / ？ 

（2）争叫1NT：好处是牌点和牌型极为清楚（16-18点，均型或者半均型敌花有止）；缺点是没有直接表明出持有4张黑桃，同伴不晓得要不要去找也许是非常合适的4-4高花配合。另外，直接争叫1NT的风险是被人家砍上一刀（遇到下家很强牌时）。

1H       1NT       /
    　　  ？

            H:J10
            D:K94

        如果北家是均型牌，在这样的进程下，即使有四张黑桃也不敢贸然叫出2S来（因为不了解同伴是否有另外一个高套）。


    加倍还是争叫（1NT）的选择，真是鱼与熊掌之不可兼得噢。加倍灵活，争叫清楚。

　　点点在此推荐自己的看法，用一个简单的方法来鉴别。假设叫牌过程（你坐在南家）:
　　    [N]      [E]      [S]      [W]  
             1H      X/1NT     2H
         /        /        ？ 
　　就是说：你在加倍或者争叫1NT之前，先考虑清楚，如果后续的叫牌进程是“下家加叫到二阶，而同伴没有叫牌”，你将会怎么办？是否再

次加倍？

                 1H        X       2H 
         / /        ?

               1H       1NT      2H 
         /        /        ？ 
　　在的情况下，同伴（北家）并不清楚南家的牌点情况，在8点以下的情况下，他一般不会选择在三阶出套。
　　在的情况下，同伴（北家）已经清楚了南家的牌点和大概的牌情，唯一不清楚的只是南家黑桃的张数为四张。一般情况下，他不会主动叫四张套黑桃。
　　综合上述分析，持有南家这手牌，第一轮争叫1NT比较合适。首先争叫1NT是不逼叫的，同伴如果没有力量，完全可以选择PASS，而不是象在同伴加倍后必须出套。其次，如果同伴有点力量，他可以选择2C的斯台曼问叫寻找高花配合。第三，即使西家直接加叫到2H，由于南家的限制性争叫（1NT），北家完全可以采用莱本索尔约定叫来进行弱牌争叫。而不必担心同伴冒局（有关这一点请大家参考莱本索尔约定叫）。

    还有一点考虑是：将这一类牌（16-18点带四张黑桃的牌）划分出去后，第一次作技术性加倍的牌将更加明确，也可以减少自己后续叫牌的烦恼。

    一个可能出现误区的地方是： 
     [N]      [E]      [S]      [W]   
                 1H        X       2H 
         / /        X 
[N]     [E]      [S]      [W]   
               1H       1NT      2H 
         /        /        X 
    这两个加倍都不应该是惩罚性加倍，而是技术性的。因为凭南家一人之力，是不可能惩罚对手的。例如南的持牌是：
            S:A52
            H:AK104
            D:AQ10
            C:764
    这样的牌算是争叫1NT最强的牌了吧，有谁能保证可以打宕对手已经有了配合的2H定约？
　　中南家的第二次加倍更多的是从牌型上选择的（更像是4-1-4-4），中的第二次加倍则确定黑桃为四张, 好比人家直接阻击2H后的加倍。
