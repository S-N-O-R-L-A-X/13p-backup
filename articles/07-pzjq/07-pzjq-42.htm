新版“点点巅峰奇遇记”

作者：一叶知秋

第二回  小卒仔评说聚峰  寂寞人独打十副


　　话说点点来到亚轩最大的桥牌俱乐部 ---- 聚峰俱乐部，门口高挂着一副对联 ---- 聚而不散，峰而无极。入得门来，但见松柏林立，翠青缀地；左侧一溜楠木桌椅摆开，坐者凝神，观者屏息，一派练兵景象。东侧俱乐部的荣誉展台上，金杯熠熠，137名会员靓照依序排列，端的是声名赫赫、人气旺盛。

　　为了跟亚轩最大俱乐部的地位相称，聚峰俱乐部主对于首次等级赛非常重视，一共派出8支队伍参赛，他们是：聚峰-桥灵队、聚峰-阳光队、聚峰-解放者队、聚峰-欢欣鼓舞队、聚峰-海洋队、聚峰-战友队、聚峰-神出鬼没队、聚峰-高山流水队。在接受采访时，聚峰俱乐部主管老兵新传（小卒仔）对俱乐部的几支队伍寄予了厚望。他认为：桥灵、解放者、神出鬼没、战友和高山流水队在本次比赛中都非常有希望进入甲级队。

　　聚峰--桥灵队是聚峰两届俱乐部内部联赛的冠军，全部采用精确制，队伍比较整齐，他们的特点就是发挥稳定，起伏较小，本次参赛的选手都是老搭档，俱乐部主管对他们进入甲级赛甚至取得更好的成绩寄予厚望。聚峰--解放者是新组的队，全部由南京牌手组成，采用中国梅花制，是第二届俱乐部内部联赛的亚军。神出鬼没队主要由中国游戏中心桥牌站的高手组成，实战经验丰富，进攻性很强。聚峰--高山流水队牌手整齐，寂寞孤独和运筹无量都是著名牌手，但是这个组对手较强，要出线需要一番努力。而聚峰--战友队的出线希望则很大，因同组对手不算太强。俱乐部内的另三支参赛队实力也不弱，都有机会出线。

　　告别了小卒仔，点点心中对首轮比赛的评论对象已经有了谱。

　　在万众瞩目和期待中，第一届亚轩等级赛终于拉开了帷幕。几十路诸侯摆开架势，各自催开人马奋力厮杀，都希望能旗开得胜。

　　点点到得赛场，举目望去，如来佛祖派遣来的西土神仙均无踪影，大概一个个都隐去了真实身份，混迹于泯泯众人之中了。朝秩序册上搜寻一遍，发现聚峰--高山流水这个队名颇带仙风，想起小卒仔的介绍，于是悄悄来到该桌旁观并兼顾其他桌上发生的情况。今晚聚峰-高山流水队与千军王牌队对阵，大家落座之后，这个聚峰-寂寞孤独坐在西家，一声不吭，将13张牌如同圣旨一般握着，其神态倒与他的名字颇为相符。说来也奇，16副牌中，这个寂寞孤独倒是坐了十把的庄！

　　如果是四明手摊开打，相信这副牌的南北都可以轻松打成4H。闭室南北未能作成3H的关键在于红心第一轮没有深飞将牌。

　　有意思的是开闭室的西家采取的开叫叫品是不一样的：开始寂寞孤独开叫1S，而闭室西家是开叫2S（阻击叫）。初看之下，似乎定约的不同是因为开室东西方使用精确，闭室东西方使用自然所致。

    可以说，这副牌的庄位是寂寞孤独利用开叫1S抢来的。

    其实，现在国际牌坛有一个比较流行的“一阶开叫法则”，称之为【20法则】，我借这副牌介绍给中游牌友：就是说如果开叫人手上的两个最长花色的张数之和加上总的大牌点一旦超过20的话，就可以开叫一阶花色而不做阻击叫的选择。

    按照这个理论，西家开叫1S就是正常的了，因为西家的大牌点为10，最长的两个套(S+C)的张数为10，综合两者，达到了20这个一阶开叫门槛，因此可以直接开叫1S而不用阻击叫2S了。

    这一种开叫法则其实可以与叫牌体系无关了。例如：

                S:8
                H:AQ8652
                D:K10842
                C:9

    这手牌可以开叫1H了。牌点9＋两套长度11＝20。

    需要说明的是：两个长套中的第二长套，要求最少是四张以上。否则还是选择阻击叫，例如：

                S:82
                H:AQJ8652
                D:K82
                C:9

    虽然（H+D）张数为10，牌点为10，我还是推荐阻击叫3H而不是开叫1H。这是避免与高阶阻击叫混淆的办法之一。

    南北方在这副满贯叫牌中，虽然最后叫到的6H满贯定约宕了，但是他们的几个约定叫倒是值得推荐给各位牌友的，尤其是其中涉及的控制力问叫。

　　1NT! =  这是一个特约（所以才加上！），1C开叫人在询问应叫人的全手牌控制，这是谭申禄先生在《精确叫牌法新探》一书中所介绍的。是组合将牌问叫前的一种过渡叫品，它不仅能利用直接将牌问叫前的部分空间（例如1C-1S-2S就浪费了5个叫牌时空），而且能够让主叫人在很低的叫牌序列上就知道联手是否有满贯，真的是一个不可多得的好约定，归纳起来就是：

　　“开叫1C，同伴积极应叫1H/1S后，开叫人再叫1NT为全手牌的控制力问叫。应叫人加级答叫”：

          2C（+1级）  = 0-2控制（A=2控，K=1控）
          2D（+2级）  = 3控制
          2H（+1级）  = 4控制
          2S（+1级）  = 5控制
          2NT（+1级） = 6控制（这副牌的南家正是回答2NT，是表示6个控制）
          ..... 其他类推

    打精确体系的牌友都知道【将牌问叫】和【控制问叫】，而对【控制力问叫】与【支持张问叫】可能有点陌生。其实【控制力问叫】与【将牌问叫】是不矛盾的并可以并存在一个叫牌进程中。当应叫人答叫全手牌的控制之后，开叫人可以继续作延迟的将牌问叫和后续的控制问叫，只不过这个时候叫牌序列可能稍有不同，例如：

北         东         南         西
－－－－－－－－－－－－－－－－－－－－
1C         /          1H         /
1NT!       /          2NT!（6控）/
3H（*）= 延迟的将牌问叫

具体应用到这副牌：
北         东         南            西
－－－－－－－－－－－－－－－－－－－－
1C         /          1H            /
1NT!       /          2NT!（6控）   /
3H（*）    /          3NT（5张1大） /
4D（控制问叫）/       5C（A，但是无Q/K）
5H

　　当应叫人回答为6个控制时，北家非常简单就知道同伴为3A（因为6控必是3A、2A+2K、1A+4K的组合，现在北家自己有3K，同伴只能是3A了）。再通过延迟性的将牌问叫（3H）得知，将牌上没有Q，是Axxxx，除非有HJ10之类，将牌是有失墩的。通过4D的控制问叫知道方片上没有Q的，虽然联手的控制已经达到11个（外边只缺一个K），但是红心不坚固，方片缺KQ，所以这样算来满贯的机会真的是要减低很多，不上满贯或不选择红心作为将牌就是开叫人需要认真考虑的了。

每一副牌都可以是一个故事，我想第二回应该收尾了。

    不过在这副牌中，由于南家是双张方片,因此如果他一直没有垫掉D2，假如SA在南家，他可以扑上SA，并回攻方片，饿死庄家的SK，定约还是要下的。

    点点检查其他各桌这副牌的结果，大部分是东家做6NT，只有一桌是西家坐庄并且达成超一。

    西家做6NT，而且北家首攻方片的话，只要打牌的顺序不出错，就可以超顿！总之还是要祭起习惯的武器：紧逼。

    首先要检查紧逼的四大要素是否满足：BLUE。

         B = 要求有一个防守人看守S+D花色（可能满足）
         L = 现有12个赢墩，只差一个（满足）
         U = 由于你的两个威胁张SK和D9是分开两个手的，因此对于任何一个防守人都具备了U的条件（满足）
         E = 进手张条件。由于东家有红心顶张，西家有方片顶张作为各自的进手，暂时满足。

    就是说会有一个同时持有DQ和SA的防守人将受到单挤。

    绝对不能先打掉DAK（因为方片是既是紧逼北家的威胁花色，也是联手的桥）；最后要注意兑现赢墩的顺序。要先兑现梅花，不能先兑现红心（这是因为东家在最后阶段已经没有其他进手了，而SK这个威胁张是需要进手的，只有用红心上手才能在最后形成紧逼态势）。

    正确的打牌顺序应该是：

    第一墩D8
    兑现CQ、CJ
    HJ去西家
    兑现梅花长套CA、CK、C10
    用H9到东家HQ、HK、HA，
    在剩下最后三张牌时，北家受挤
