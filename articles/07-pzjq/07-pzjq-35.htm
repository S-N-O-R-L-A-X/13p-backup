第4篇：英雄有泪也轻弹(献给婷婷)

    婷婷--多美的名字。我想你一定是一个乖巧的女孩。那天你来看我和药渣打牌，你说你喜欢桥牌，也喜欢累神的贴子。累神晕了，累神虽是江湖剑客，无行浪子，但一听到女孩的恭维就找不着北了，累神就为你改变一下胡砍的风格，讲一个真实的桥牌老英雄的故事：

    在中国的桥牌圣地上海，绝大多数第一第二代国手都从这里涌现。从王陆大师到唐氏兄弟徐胡二老，这些名震牌坛的前辈，累神以为，他们为中国桥牌所做的贡献将来一定会在中国桥牌馆中看到。累神要写的是一位几乎与以上这些牌手同代但至今仍在第一线顽强战斗的浦东队的顶梁柱周家弘：

   在累神还在玩争上游的时侯，周家弘就已是桥牌国手了。他身高超过2米，站在赛场堪称一景，我们都亲切的叫他长师付。长师付牌气温和我们这些小字辈都喜欢和他开玩笑。长师付的牌风凶猛犹擅长双人赛，是累神觉得难打的敌手之一。长师付自加盟浦东队后，始终以主力牌手的身份征战全国赛场，累神屈指一算，大约有6年了，浦东队每次大赛几乎没出过前6名，成绩之稳定远超过那几只有时连小组都出不了线的牛货们。但奇怪的是，浦东总是与冠军失之交臂。有几次很有希望但总在最后一刻功亏一溃。冠军对长师付和他的战友们似乎很近却又那么遥不可及，6年，说说很快其实足以击垮意志薄弱者的神经，至少累神肯定承受不起。有一次A类俱乐部我队和浦东争第3名，刚输牌的我心灰意冷，长师付却对我笑咪咪的道：卷毛，下次再战吗：我的心头滋味无穷，对我来讲我还年轻，还有很多的下一次，而长师付呢?

   今年俱乐部锦标赛，浦东队一路过关斩将杀进决赛。全队上下众志诚诚，老板也飞马赶到助阵。但对手是强大的中建队，决不白给。决赛一开始，由于过度紧张同伴抽错牌打下了一个铁成的局，长师付一边安慰同伴一边寻找战杌。婷婷，累神是无神论者，但有时人的精神真的会改变命运。创造奇迹，就象上帝的灵光一现：

                           H KQx
                           D 10xxx
                           C x
                   S Qxx      N     S x
                   H Jxx            H xxxx
                   D xxx  W      E  D Kx
                   C Kxxxx    S     C QJxxxx
                           S AJxx
                           H Axx
                           D AQJx
                           C Ax

                 N      E     S      W
                             2NT     /
                3H      /    4D      /
                4H      /    5C      /
                6C      /    7S      //

    长师付开叫20-21点的无将，同伴3H转移S，长师付4D是表示S极配，D特征。同伴4H呼应满贯，5C和6C都是扣叫，最后的7S当然是英雄气概。首功一张H，长师付眼都不眨就将西的SQ飞死。休息时，我问长师付为何这样打。长师付说：这牌6S一定会叫，假如DK飞不到，我只要打对将牌3-1，我仍然可能不输。有勇有谋，婷婷。这就是一个老牌手展示的境界：

    最后，浦东终于获得了胜利，当长师付举起冠军奖杯的时候，在真诚热烈的掌声中，我却看到了长师付眼里流淌的泪花。诚然，一个小小的冠军不足以补偿6年拚杀的付出，但一个职业牌手所必须拥有的坚定不移的信念。百折不挠的责任感以及对桥牌的爱才让我们动容。

      朝霞固然灿烂。
      但我们更感动夕阳的辉煌。
