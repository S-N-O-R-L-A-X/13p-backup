推论其推论


推论中一项特别有用的领域，我们可给予更多的注意，那就是我们不仅能从对方的叫牌中作推论，也应考虑他们从我们的叫牌中推论多少。在两种打法间作选择时，可能要依据我们的叫牌有多明显而定：

北发牌，双方无局：

        －     －     1H
 －     2H     －     3NT
 －     4H     All Pass

首攻：C3

东家以CA吃住第一墩后，并没找到方块回攻而回C5。现在你可以第三圈梅花抛一张方块－正常的打法是马上飞王牌，希望如果飞牌失败的话，东家仍没找到致命的方块回攻；然后你就舒服了，你的黑桃、草花赢墩可以解决桌上两墩方块。

防家对你的牌知道多少？他们不会知道你有如此强壮的黑桃套，可以利用这一点么？

你应该在飞红心之前，先提第三轮草花，并垫一张明手的黑桃，现在东家肯定很有机会回攻错误的花色。

 西     北     东     南

 －     1H     1S     2NT
 －     3C     －     3D
 －     3NT    all Pass

首攻：S3

美国的庄家保罗－索罗威（Paul Soloway）让东家的SK吃住第一墩，吃住下圈黑桃后，他考虑两个打法。

他可以方块下桌拉红心，如果东家扣住，HK吃到后改动方块，3－3分布就可让他有九墩牌。但是索罗威想到他曾叫过两次方块，东家很容易的会上HA再迫出黑桃档张，因他知道他的方块可再上手。

所以第三墩索罗威出HK：如果东家吃了H Axx而西家持H Qxx，可打大三墩（东家可扣住HA不吃来阻止，但索罗威会将注意力转到方块上）。这是正确打法，可惜红心未如所愿，索罗威定约失败。

意大利的叫牌过程不同：

 西     北     东     南
                      1C
 －     1D     1S     1NT
 －     2H     －     2S
 －     3C     －     3NT

首攻：S3

1C和1D是人为叫品，但之后的叫牌都是自然的。北家显示了他破烂的两套，但南家贝拉多纳（Belladonna）并没有言及方块。

前两墩相同，但之后贝拉多纳以DA下桌拉红心。在叫牌资讯不足的情况下，东家罗伯特－哈曼（Robert Hamman）难以读出牌情，他让HK吃到。现在贝拉多纳转而动方块，建立了九墩牌。
