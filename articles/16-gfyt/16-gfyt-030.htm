放弃飞牌的形势


加拿大牌手丹－赫伦认为：当你在做无将定约而只少一个赢墩的话，你会考虑飞牌的可能性，但当你的左手敌方已有一个做好的长套，而且还有一个快速进手张时，不要对飞牌抱有幻想，放弃飞牌不会给你造成损失。

如果你认为这个观点并不太新，也不算发明创造，那么，就把它当作一个忠告吧！不论是新观点，还是善意的忠告，记住它在实战中都会受益的。比如你是南家，主打3NT定约，联手牌是：

S KQ
H AQ4
D KQJ652
C 87

S J43
H 1087
D A93
C A943

西家曾经争叫过2C。现在他首攻CK，接着又出CQ、CJ，你于第三轮用CA停住，明手垫H4，东垫了一张小方块。

在IMP赛和双人赛上，你分别该如何打？

从叫牌过程几乎可以肯定，西持有SA。如果你现在奔吃明手的方块赢张的话，最后出牌在明手，你就无法对红心进行飞牌了。飞不飞呢？

正确的打法应当是放弃飞牌的机会，兑现方块赢张，这样做至少可以再观察一番东、西家手中的牌情。

在你兑现方块时，西被迫垫牌。如果他垫了一张草花，你就可以顶下SA，从而拿到第9墩牌；如果他不垫草花，那他就只有两张边花了，他只能留下一张红心以及SA。因此，如果他原本就有HK的话，此时你就可以将其击落。也就是说，如果飞牌会成功的话，那么这一机会并没有失去。我们来计算一下，庄家在第四轮就飞红心失败了的话，定约将会宕2；如果按照上面所说的打法，即是不能击落HK，定约也只宕1。

最糟糕的打法是在用CA进手后，立即打一轮S，想顶下SA或“偷”到一墩黑桃。

全副牌如下：

        H AQ4
        D KQJ652
        C 87
S A985            S 10762
H J3              H K9652
D 87              D 104
C KQJ106          C 52
        S J43
        H 1087
        D A93
        C A943
