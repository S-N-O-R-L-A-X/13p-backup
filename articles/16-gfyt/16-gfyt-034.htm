首攻王牌的研究


From 将吃－桥牌天地

（一）不适合首攻王牌的情况

首攻王牌不应该是盲目的，不能因为不知道要攻什么，就攻王牌；也不能因为某些叫牌过程看来该攻王牌，便不加思索的首攻王牌。不论任何首攻，都应该详加思考，根据手上牌情，敌方的叫牌过程，尽量作出正确的判断。

在介绍首攻王牌的情况之前，先看一下不适合首攻王牌的情况。以下是一些不适合首攻王牌的情况∶

1.可能使你方的王牌赢磴减少

例如王牌首攻人持了王牌 Kx、Qx、Qxx、Jxxx，很可能会使得原本有的一个王牌赢磴不见了；或是首攻人持了王牌Jx、Jxx而同伴有王牌K或Q时，在庄明手的王牌AK或AQ分家时，原本庄家自己处理王牌将会让防家吃到一磴，但是首攻王牌反而让庄家在王牌没有失磴，例如∶

      例一.           明 KT94
          首攻者 J65          防家 Q8
                      庄 A732

      例二.           明 QT74
          首攻者 J65          防家 K8
                      庄 A932

另外一种状况则是首攻人自以为安全的从1~3张的小王牌中攻出，反而帮庄家找到了王牌Q的位置。当然的，首攻单张王牌大牌(AKQJ之类)也往往都是在帮庄家解决问题，例如∶

      例三.           明 Q965
          首攻者 J            防家 K73
                      庄 AT842

      例四.           明 K965
          首攻者 J            防家 Q73
                      庄 AT842

      例五.           明 K965
          首攻者 A            防家 J73
                      庄 QT842

2.有更重要的事要先处理

2.1 同伴指示首攻 - 请参考赖特纳尔加倍与首攻指示性加倍

2.2 必须及早提吃或建立旁门赢磴

例如同伴曾经叫过牌；或是在敌方叫到小满贯，而首攻者持了王牌A或K，这时候很明显的，必须在王牌A或K被逼出来之前，利用首攻，建立旁门赢磴，所以这时候可以考虑从有Qxx甚或Kxx的牌组牌组引出，或是穿梭明手牌组。

这里要提一下一些盲目首攻王牌的情况，当庄明手看来很像要交互王吃的状况或是开Flannery时，教科书都说要首攻王牌。所谓盲目的状况就是不加思索的攻出王牌，开叫Flannery的敌方可能是两门高花都配合，首攻王牌将使得庄家顺手清光防家王牌，用另外一门高花垫掉低花的失张，所以首攻者应该至少要根据手中牌情考虑一下是否有可能会发生这种状况。在庄明手叫了三门才找到王牌配合时，似乎应该首攻王牌避免被交互王吃，首攻者仍要根据手中牌情来决定首攻。

例如下面的叫牌过程∶

        南  西  北  东
        1S   -  2C   -
        2H   -  4H   -
         -   -   =

(1).西家持了四张小黑桃时，便要考虑，南家的黑桃是否坚强，首攻王牌可能让庄家顺手清光防家王牌，用黑桃垫方块失张，所以说不定应该先提吃方块赢磴；

(2).又或西家持了四或五张小黑桃与单张王牌，则东家有很大机会是短黑桃带王牌，则发展旁门赢磴似乎就显得较重要；

(3).又若西家持了黑桃AQxx，为避免北家王吃黑桃建立南家的第四或五张黑桃，就要考虑首攻王牌的可能；

(4).若西家只有一两张小黑桃，那么明显的东家会有长黑桃，可能还有大牌，看来庄家的黑桃要建立须花一大段时间，则首攻便应该考虑建立旁门赢磴；

(5).若西家是两三张黑桃带A或K，王牌有Axx，便可以考虑首攻小王牌，上手再拔王牌A，引小王牌，使北家没有机会王吃黑桃；

2.3 首攻者需要王吃或让同伴王吃

首攻者自己有短门想要王吃；或是庄家牌组持有四张以上，或是在敌方两门配合後，未被选为王牌的那门持有三四张，同伴看来是单张或缺门有王吃该门的机会时，当然不能首攻王牌。

2.4 攻击庄明手之间的桥引

当叫牌显示庄明手有一门可吃通牌组，而有该牌组的一家桥引很少时，或该牌组建立需要保留某些旁桥引时，可以考虑∶

(1).首攻该门，使敌方在该门打大前先断桥(例如∶AKQxxx对著单张)，或是使敌方在未清光王牌前无法奔吃该门

(2).首攻旁门，攻击旁门可能潜在的桥引

2.5 迫王

当防家在某一门拥有坚强的号码或张数时，而某一位防家在王牌持有大牌与张数时，也可以考虑不断引出该门，逼迫庄明手中有王牌的人王吃，最後可能导至王牌失控或升王。看看下面的例子∶

        南  西  北  东
        1H  1S  2D  2S
        3C   -  3H   -
        4H   -   =

西家持 S KQJT  H A842  D 92  C 743，王牌8有机会吃到，朝迫王型来防御的话，就要首攻黑桃，如果有必要的话，在庄家清王时，王牌A要扣住到到第三圈才吃，然後再引黑桃。

当庄明手有两门配合时，首攻王牌往往是加速防家死亡的防御，庄家可能顺手清光王，顶出王牌或旁门缺的大牌，然後奔吃另外一门，定约可能还会超个几磴；这种时候，防家应该要先把自己该吃的赢磴先提一提，有机会的话，说不定也能制造出迫王或升王的局势。


（二）适合首攻王牌的情况

首攻王牌不应该是盲目的，不能因为不知道要攻什么，就攻王牌；也不能因为某些叫牌过程看来该攻王牌，便不加思索的首攻王牌。不论任何首攻，都应该详加思考，根据手上牌情，敌方的叫牌过程，尽量作出正确的判断。

在介绍不适合首攻王牌的情况之後，我们再来看看适合首攻王牌的情况。以下是Resenkranz所列出的首攻王牌的情况∶

1. 保护大牌的力量
2. 消减敌方王吃的潜力
3. 从加倍抽取线索
4. 心战性首攻王牌
5. 消极性首攻王牌

但因手边的中国桥艺缺了几期，没有办法一一介绍，以下的部分是综合中国桥艺360期与我的一些心得。

一. 避免旁门大牌受伤

常见的情况有∶

(1).庄家有较多的牌力，而首攻者有嵌张

(2).庄明手的王牌有九张以上的好配合，而引旁门很容易检举同伴

(1).庄家有较多的牌力，而首攻者有嵌张∶典型的叫牌过程为
        1S - 2S; 4S
如果首攻者手上AQ或KJ的嵌张，在已知庄家有15点以上时，从这些花色低引或引出大牌都是很冒险的。

(2).庄明手的王牌有九张以上的好配合，而引旁门很容易检举同伴∶如果手上尽是一些单独的Q或J，甚或是全手无点，门门3~4张小牌，便要考虑到首攻旁门检举到同伴大牌的可能性，如果从叫牌来看，有很高的机率检举到同伴，也可考虑首攻王牌，特别是在面对满贯定约时。这是一种消极性的首攻王牌，有些定约就是不能太积极防御，庄家打来打去就是少一磴，而积极防御反而可能帮防庄家生出一磴，尤其是有一些被冻结花色存在时。

二. 保护大牌的力量

常见的状况有∶

(1).同伴开叫过无王或表示过平均牌形

(2).敌方以劣势牌力，作高线牺牲叫，显然凭藉王吃的力量

在这两种情况下，防方显然都有较具优势的大牌力量，而要吃到这些大牌赢磴，就必须要肃清敌方王牌，从而减低敌方藉由王吃甚至是交互王吃来制造赢磴。

看一些例子∶

例一.   东  南  西  北
       1NT  2D   -   -
         X   -   -   =

东家已表明为强平均牌，西家持 S Q2; H 9765; D T963; C K42，持有长王，些许的大牌力量，同伴在王牌上可能也会有一两张大号码，首攻王牌将是不错的选择。

例二.   东  南  西  北
       1NT 2NT!  X  3C   !=两门低花55
         -   -   X   -
         -   =

东家持 S JT97; H AQ63; D AQT; CQT，旁门有许多潜在赢磴，同时红花有太多嵌张，黑桃如果能成为赢磴，同伴要有大牌，加以庄家选择了梅花也就暗示了有王吃方块的可能性，所以首攻王牌。

例三.   东  南  西  北
         -  1S  2C  2H
        3C  3D  4C  4S
        5C   X   -   -
         =

北家持 S KT5; H KQ9874; D K54; C 8，显然的，敌方的点力较少，旁门也没有任何一门可以吃通或垫牌，所以应该首攻王牌，避免被交互王吃。

例四.   东  南  西  北
        1S   X  2S   -
         -   X   - 2NT
        3H   X  3S   -
         -   X   -   -
         =

南家持 S Q; H KJT; D KJ764; C AKQ5，同伴的2NT显示其仍然有5~6点以上与有号码的王牌张数，西家在东家的3H之後选择了黑桃定约，又暗示了西家可能有王吃红心的能力，以及梅花大牌可能被东家王吃，而南家的嵌张又躲在东家後面，不论从那一个角度来看，首攻王牌都是正确的选择。

三. 消减敌方王吃的潜力

常见的状况有∶

(1).开叫Flannery特约

(2).敌方在叫了三门牌组之後才找到配合，首攻王牌以预防交互王吃

(3).由手中的牌预见到短王的一方有王吃的可能

(a).开叫Flannery特约∶在已知一家有五张红心与四张黑桃的情况下，答叫人选择了一门来当王牌，这通常暗示答叫人有王吃另一门的可能性，首攻王牌可以消减答叫人王吃的潜力。

(b).敌方在叫了三门牌组之後才找到配合，首攻王牌以预防交互王吃,典型的叫牌过程有∶

            1S - 2D; 2H - 3H; 4H 或
            1C - 1D; 1S - 3S; 4S

庄明手在王牌外各有一长门，很可能可以靠交互王吃来完成定约，如果首攻者在S或D有四张带A或K，庄家在交互王吃前必须让你上手，则你更有清第二次王的使敌方无法交互王吃的机会。

(c).由手中的牌预见到短王的一方有王吃的可能∶当庄或明手叫过一长门(四或五张)，而你手上有Axxx或Kxxx时，且居於该长门的上家位置时，敌方可能可以王吃该门，而你的同伴虽然亦在该门为短，但却是居於王吃的庄家或明手的上家不易位置，为避免这种被短王牌的一方王吃的状况，你可以考虑首攻王牌。

当然，并不是所有开叫Flannery或叫过三门才找到配合时，都一定要首攻王牌，当敌方有两门好配合时，首攻王牌的结果很可能将是∶庄家顺手清光王，以旁门长门垫掉另一旁门失张，简单超磴。这种时候，防家应该把自己的赢磴先提光，然後再来想办法发展击垮定约的赢磴。
