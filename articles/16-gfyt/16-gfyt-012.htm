将牌骗招


麦克尔-科特尼（Michael Courtney），澳大利亚最好的桥手和桥艺作家之一，把目光投向了将牌套，在本文中，他给防守方找到了一个从庄家那里偷盗一墩牌的骗术。

当庄家准备忍让一墩将牌的时候，防守方设法诱使庄家从明手出掉一墩大将牌，经常是十分重要的，尤其当明手能够做超将吃的时候，更是如此。

看看下面这手牌：
        S Q862
        H AK
        D QJ8
        C AK97
S J93           S A5
H 7543          H Q106
D 752           D AK94
C 642           C Q1085
        S K1074
        H J982
        D 1063
        C J3

西家展示了这一高超技术。对抗南家的4S定约，他首攻方块，同伴DK，DA，继续小方块到明手的DQ。庄家出王牌小到暗手的SK，然后继续王牌。此时，西家摆上SJ（王牌骗招），庄家SQ盖了！东家的第13张方块提升了S9，注定了庄稼的失败命运。英雄的称号当然会授予给东家，是东家最终打出了王牌提升。但西家的SJ实在是功不可没。

下面这手牌来自Ron Klinger的《Kosher Bridge》

        H AKJ104
        D 42
        C A4
S J106           S A
H 873            H Q96
D J108763        D A95
C J              C KQ9863
        S 98743
        H 52
        D KQ
        C 10752
西   北   东   南
          1C   --
--   X    2C   2S
--   4S   AllPass

西家首攻CJ。Klinger指出：如果打牌进程是这样的：CA站住；D2到DA；东家兑现CK；东家送方块到庄家的DK；庄家出S3，西家出S6：此时，Klinger说，庄家必须出S2，摆下东家的单张SA，因为如果王牌是2-2分布的，草花上必有一个王牌升级。

完全正确。但西家这个时候可以使用“王牌骗招”去建立一张王牌赢墩：他第一墩王牌摆上SJ，如果庄家盖上了SQ，东家SA后回出草花，西家的S6将逼出明手的SK，将S10树立成赢墩。


凯尔西在他著名的：《The Tricky Game》中，给出了下面这副牌，作为一个标准骗张的例子。
        S AKQ2
        H A1083
        D Q82
        C A5
S 94             S J1083
H K2             H J96
D AKJ976         D 103
C J102           C 9763
        S 765
        H Q754
        D 54
        C KQ84
防守4H定约。西家曾经开叫过1D。
西家提掉DA，DK后，换攻CJ，庄家CA后，出DQ，想欺骗东家王吃。
东家是Martin Hoffman，没有上当，他垫了一张草花。庄家提HA，再出小红心，Hoffman跟HJ--将牌骗招--庄家没办法拒绝盖上HQ，到西家的HK。第四轮方块升级了H9，定约一下。和前面一样，庄家只有坚持原来的打法放过这墩王牌J，才能把定约带回家。

　

如果同伴没有回出给你王牌升级的牌，请不要责备我。同伴很难判断出你是否打出了一个王牌骗张。毕竟如果你真的已经没有王牌了，那么王牌升级的打法无论如何都是不可能发生的。
