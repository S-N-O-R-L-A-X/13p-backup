第七章  安全打法


严格而言，安全打法是指那种打法，在可以允许失墩的范围内，冒着多失一墩的危险，来保证不额外多丢失一墩超出允许范围不能丢失的牌墩。例如对于下面这种牌：

Q X X X

A 10 9 X X X

如定约者还可以丢一墩，并由他手中出牌，则正确的打法是从A这边出小牌：如果对方有一家是单张K，这种打法显然多丢了一墩，但对任何一方持有KJX时却是可以避免丢失两墩的安全打法。

推而广之，很多称为安全打法的那些打法，事实上并不是去冒多失一墩的危险，而是防备一种特殊的分布。所以在下列情形下：

Q X X X

A J 9 X X X

出Q之所以称为安全打法，是因为它考虑到东家有K10X这种可能性；如果出一张小牌来飞，则东家可保留K10以对付明手的Q。

飞牌的技巧

一般原则是飞牌时应先出小牌，上例正是一个例外。象下面这一类牌中，有些桥手是会犯错误的：

A Q 10 X

K X

9 X X X

J X X

显然出J就会丢失一墩。

下例也是同样的情况：

A J 10 X X X

9 X

如西家有单张大牌，不小心地出9，也会造成多丢一墩的损失。

本章第二例是出小牌飞牌这一条原则的一个例外。下面是一个类似的例子：

A Q 8 X X

J X X X

如果定约者要拿这个套的全部墩数，他最好的机会是出一张小牌用Q飞。但如果许可在这个套中丢失一墩，他就该出J。这是考虑西家有K109X的安全打法。

下面也是同样的打法：

K J 8 X

Q X X X X

如果剩下四张牌全在东家，就毫无办法可想；但如果在西家，则只要南家出Q，就可使西家在这个套中只拿一墩。

拒绝飞牌

有很多安全打法是先拔大牌而不飞牌，请看一个经常遇到的情形：

A Q X X

X X X X

虽然在实战中许多人喜欢先以Q飞，但如果没有其他因素要考虑，应先拔A，这是一种防止右敌持有单张K的安全打法。

当两手持有如下九张牌时，安全打法很重要：

A Q X X X

X X X X

如果想全拿这个套的五墩牌，最好用Q飞。但是定约者如允许丢一墩牌时，他应先拔A，因飞牌可能丢给单张K。

上例是众所周知的，但许多有经验的桥手在下述情况下也会犯错误：

A Q X X X X

J X

不管牌张怎样分布，一墩是必丢的。正确的打法是先拔A。有些桥手会由明手出小牌，希望J拿墩。这时，如果东家持单张K，这样打法没有问题，但如西家持单张K，就要丢两墩。

下面是牌局上最常见的组合，但桥手中十有八九不能处理好：

K J X X

A X X

如果要拿四墩，唯一希望自然是假设西家持有QXX。但定约者经常只需从这个套中拿三墩。这样他必须避免飞牌，先打A和K，然后由手中向J方向出一张小牌，除非东家有Q10XX四张，三墩是稳拿的。这种安全打法的优点是考虑到东家持有QX的可能性。这时如试图飞牌，东家Q会得一墩，西家还要赢得第四墩牌。

深飞（Deep Finesses）

上面所举的那一类安全打法是防止自己的大牌被对方大牌吃掉。下面所举的例子说明如何防止自己的大牌扑空。所有稍有经验的桥手都熟悉下面情况：

K 5 3

A 10 7 4 2

先由明手出K，再出小牌，东如出小，南当然出10，这样在东有QJXX时只丢一墩。

下例也是同样打法：

K 9 5 4

A 10 7 6 2

如果在这个套中允许丢一墩，安全打法是出2，西家如出3，则明手出4，这种打法不管哪一家持有QJXX都能拿到四墩。

在下例中：

K 9

A 10 7 5 3

在这个套中拿到四墩的最好机会是手上出小牌，用9飞，如西家有QX或JX，则防御方只能拿到一墩。

按最佳机会打

有一些情形，有多种打法可供选择。对下面一手牌来说：

K 9 X

A 10 X X

最好的打法是连出A和K，只有在防方之一持有QJXX时，才丢两墩。

众所周知，对于下面一手牌：

A 9 8 X

Q 10 7 X

最好的机会是双飞。先出10，明手跟小牌，如丢给J，再飞K。这样打只有在东家持有KJ时，才会失两墩。

但两家如持有如下九张牌，打法又很不同：

A 9 8 X X

Q 10 X X

如双飞，则东家持KJ或KJX时，就会丢两墩。最好的打法是先出A，只有在西家持有KJX时，才会损失两墩，但如西家持KJ时就不然，所以比双飞好。

在下例中，很少桥手能用正确方法打：

Q J X X

K X X

通常的打法是由手中向明手出小牌，希望J拿墩，然后再由明手向K出小牌。这样只有在对方3-3分布时才能拿三墩。其实还有可能利用额外的机会，即西家只有一张A和一张小牌。正确的打法是先向QJ出小牌，如这一墩拿到，回到手中再从KX出小牌；这样，如西家原来持有AX，防御者就只能拿到一墩；如原来是3-3分布，这样作仍稳得三墩。

避免丢过两墩

有一组安全打法，其目的是使失墩不超过两墩。安全打法几乎总是在头一轮不飞牌。

A Q X

10 X X X X

如定约者可以丢两墩，正确的打法是先拔A，然后由手上出小牌转向QX，这种安全打法对任何4-1分布都适用。当持有AQ五张套时，也可以采用同样打法。

A Q X X X

10 X X

为避免丢失三墩，安全打法是先拔A，然后由明手向10X出小牌。如将前一手牌中的Q与10对调如下，仍可采用同样的安全打法：

A 10 X

Q X X X X

定约者仍然只允许丢两墩而不是三墩时，这时他应先出A，然后由手上向10X出小牌。这是防止任何一种4-1分布极简单的安全打法。

一些特殊的安全打法

还有一些牌的组合，需要用特殊的安全打法，它们不能归入上述各类。

K J X

A 9 X X X

只有一种方法保证对任何4-1分布情况都能拿到四墩。先拔K，然后由手中向JX出小牌，不管哪一方持有Q10XX，这种打法都能成功。

如把J和9对调，则打法稍有不同：

K 9 X

A J X X X

稳拿四墩的打法是先打出A，然后由手上向K9出一小牌。如西家很牌就盖上所打出的牌，如西家告缺，则用K拿，然后向JXX出9。

对下一手牌大多数桥手都知道如何处理：

10 X X

A K 8 X X

定约者先出A，如果西打出Q、J或9，则稳拿四墩的安全打法是向着10X出小牌。

下面一手牌的正确打法不是人所共知的：

A 9 X X X X

J X

只有一个机会可以只丢一墩，即由明手出小牌，希望东家有K10或Q10。

有些安全打法很少具有实际意义。很可能你一生也不会知道其益处。例如：

10 X

A Q X X X X X

为了保证不丢两墩以上，应由手上向明手出小牌。

南家主打4，西家曾对1进行过技术性加倍。防御者连出三墩红心，第三轮南家将吃，如将牌平均分配则定约不难完成。但由于西家的技术性加倍，很可能东家有将牌长套。定约者面临的问题和上例一样，如果他打出将牌AKQ，东家将阻止他兑现草花，定约者将丢一墩方块、两墩红心和一墩黑桃。安全打法是先拔A（防止西是J单张），再出一张小黑桃。这样，如果对方回方块就可用A拿进；如回红心，则可用明手将牌吃张。

如有一张黑桃大牌在明手手里，打来更为简单。比如说，明手是AXX，定约者手上原来是KQXXX，就可以打出KQ，然后连续出草花，直到东家吃张为止。
